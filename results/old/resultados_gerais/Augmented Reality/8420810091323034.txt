2014 - 2014
Desenvolvimento de uma GUI Software para Analise de Imagens Médicas
Descrição: O objetivo do projeto era desenvolver uma ferramenta que possibilita cirurgioes a alinhar manualmente os dados pre-operativos com as visões intra-operativas, simulada por uma modelo do corpo humano. A linguagem de programação utilizada foi JAVA com auxilio da ferramenta VTK. Esse é bom para que os cirurgiões visualize o registro, não dependendo apenas da abstração do caso estudado e ajuda em programas de inicialização automática.. 2013 - 2014
Processamento Cinemático em Tempo Real: Estudo das Estações da Rede GNSS
Descrição: Utilizando o software BNC-Ntrip é possível acessar o servidor que recebe todos os dados coletados pelas estações da rede GNSS. E com o software TrackRT(MIT) em um outro servidor depois de ter baixado as efeméricas ultra rápidas pelo Gamit/Globk executa-se simultaneamente o BNC e o trackRT obtendo assim a análise do movimento das estações..
