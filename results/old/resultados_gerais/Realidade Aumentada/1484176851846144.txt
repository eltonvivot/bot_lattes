2016 - 2017
USO DE RFID PARA GERENCIAMENTO DE MERCADORIAS EM AMBIENTES COMÉRCIAIS
Descrição: O projeto visa elaborar um gerenciamento de compras usando a tecnologia de RFID (Identificação por Rádio Frequência), o qual permitirá pagar as compras de forma mais rápida, apenas passando todas as mercadorias pelo caixa, sem a necessidade de verificar individualmente cada uma para contabilizar seu preço. A tecnologia é composta por uma antena e um transceptor, que realiza a leitura do sinal, através de um pequeno circuito, aonde a informação é guardada e pode ser instalada em uma tag ou cartão. O sistema utilizará arduino, Linguagem Java e MySql, para desenvolver um sistema de hardware e software integrado. O software será para manipulação das informações das mercadorias, como incluir, cadastrar peço, etc. Esse modelo de gerenciamento é muito mais rápido e prático do que o modelo atual usado nos estabelecimentos comerciais, pois elimina o tempo gasto passando produto por produto através do código de barras.. ANTENAS DIRECIONAIS PARA REDES SEM FIO
Descrição: As redes sem fio estão sendo bastantes usadas porém, o grande problema é a segurança das informações, já que elas utilizam sinais de rádio e qualquer pessoa com equipamento adequado pode receber o sinal. Apesar de existirem softwares para diminuir essa fragilidade, eles não resolvem o problema de segurança.
Este projeto tem a finalidade de limitar a área de cobertura das redes sem fio e, para isso, propôs-se desenvolver uma antena para limitar a área de cobertura da rede sem fio.. Situação: Concluído; Natureza: Pesquisa. Integrantes: Johelden Campos Bezerra - Coordenador.
