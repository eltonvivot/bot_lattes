Robocode no ensino de programação de computadores
Descrição: A lógica de programação é uma das mais importantes disciplinas para os alunos que ingressam em um curso da área de informática, pois conceitos adquiridos nessa disciplina estão relacionados com desempenho do aluno em outras disciplinas do curso. A lógica de programação é também uma das disciplinas com grande índice de reprovação e uma das causa da evasão de alunos. Alguns alunos tem dificuldade em adquirir as competências e habilidades relacionadas com a lógica de programação através das metodologias de ensino tradicionais. Para esses, é importante o uso de ferramentas para ajudar o processo de aprendizagem e para estimular o raciocínio crítico e pensamento estratégico. O Robocode é uma dessas ferramentas. Robocode é uma plataforma de software livre desenvolvida pela IBM. Os robôs são programados em uma linguagem de programação que aborda conceitos avançados de programação. Este projeto propõe a utilização do Robocode como meio para exercitar a prática em programação, possibilitando a familiarização dos alunos com a linguagem de programação e potencializando os objetivos e estratégias de aprendizado das disciplinas de seu curso, além de promover a participação interativa através de batalhas robóticas virtuais que os estimulem a construir algoritmos satisfatórios.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (1) . Integrantes: Mônica Sakuray Pais - Coordenador / Jorge Mayck Campos Lima - Integrante.

Aplicação de algoritmos evolutivos no agrupamento de dados para caracterização de cobertural vegetal em imagens de satélite
Descrição: A redução dos recursos naturais estimula uma constante aquisição de informações ambientais espaço temporais com o objetivo de compreender padrões de uso e ocupação e organização do homem no espaço. Para tal, imagens de satélite para fins de mapeamento, planejamento e monitoramento ambiental tem sido utilizadas em conjunto com novas tecnologias e métodos. A classificação de imagens é o processo de extração de informações para reconhecer padrões e objetos homogêneos em produtos de sensoriamento remoto para mapear temas de interesse em áreas na superfície terrestre, através do agrupamento de dados. O objetivo do agrupamento de dados é particionar os dados em grupos de objetos mais similares entre si do que com relação aos objetos dos demais grupos, segundo alguma medida de similaridade ou dissimilaridade. A literatura sobre o problema de agrupamento de dados é extensa, sendo que por várias décadas diversos algoritmos de natureza, características e propósitos distintos têm sido propostos e investigados. Não são claras, no entanto, as reais diferenças existentes entre essas diferentes metodologias em termos de eficácia e eficiência computacional, dentre outros fatores, tanto no âmbito teórico como experimental. Este projeto tem como objetivo aplicar métodos de classificação de imagens baseados na computação evolutiva e comparar os seus resultados.. Situação: Concluído; Natureza: Pesquisa.
