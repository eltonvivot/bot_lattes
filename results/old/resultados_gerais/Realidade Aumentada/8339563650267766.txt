2006 - 2008
Desenvolvimento de um sistema tomográfico com feixe de prótons
Descrição: O objetivo principal do presente projeto é o desenvolvimento de um sistema tomográfico, que pode utilizar um feixe de prótons. O projeto deverá ser desenvolvido baseado na continuação das pesquisas preliminares de nossa equipe.O projeto inclui varias metas básicas, como o estabelecimento de um conjunto padrão de procedimentos para simulações computacionais da pCT que incluirá as simulações de propagação de prótons através de objetos bastante simples usando os códigos GEANT4 e SRIM2003. Os resultados das simulações serão comparados com cálculos teóricos e dados experimentais. Finalmente, um procedimento padrão de simulações usando o código GEANT4 será elaborado. A partir dos resultados simulados, o procedimento será usado como uma tarefa que permite efetuar a modelagem computacional das medidas pCT. Deverá se desenvolver um sistema tomográfico móvel, capaz de obter as imagens utilizando o feixe de prótons. 
Pretende-se avaliar a sensibilidade do método pCT para as pequenas variações de densidade dentro de um objeto. Além disso, pretende-se analisar plásticos padrões para simular os materiais biológicos em radiologia. Serão efetuadas as simulações computacionais para medições com objetos de diversas características 
geométricas e de materiais diferentes. Baseado nisso, a escolha da geometria e do material será feita para cada phantom. O sistema deverá ser testado utilizando o feixe de prótons do ciclotron CV-28 do IEN/CNEN no Rio de Janeiro.. Situação: Concluído; Natureza: Pesquisa.
