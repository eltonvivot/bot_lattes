Geração Automática de Código HDL a partir de Modelos UML usando Orientação à Aspectos para Sistemas Embarcados de Tempo-Real
Descrição: A presente proposta se encontra inserida dentro do contexto do projeto de pesquisa do proponente (pesquisador 1C do CNPq) e da tese de doutorado apoiada tambem pelo CNPq (processo número 141214/2005-8) cujo objetivo é fornecer uma metodologia de projetos baseados em modelos (do inglês, Model-Driven Design ou MDD) que utiliza conceitos do paradigma orientado à aspectos, juntamente com a UML. Além disso, a referida tese já possui um protótipo de uma ferramenta de geração de código chamada GenERTiCA, que utiliza um mecanismo de script configuráveis para gerar automaticamente código fonte para a estrutura, comportamento e tratamento de requisitos não-funcionais de um SETRD. Além de gerar código, GenERTiCA também realiza o entrelaçamento (do inglês, weaving) do aspectos, que foram especificados no modelo UML, no código gerado. Dessa forma linguagens que não foram inicialmente criadas para suportar o conceito de aspectos , como C/C++ ou VHDL (Very high integrated circuits Hardware Description Language), podem ser usadas como linguagens alvo do processo de geração de código a partir da UML. Mais especificamente, o presente projeto visa a utilização da ferramenta GenERTiCA para gerar código VHDL a partir de modelos UML que separam o tratamento dos requisitos não-funcionais do SETRD utilizando aspectos. Dessa forma será necessário descrever os scripts contendo as regras de mapeamento dos elementos presentes no modelo UML para elementos da linguagem VHDL. Além disso, será necessário fornecer a implementação (na forma de scripts utilizados para a geração de código) dos aspectos que aparecem especificados no modelo UML. Como estudos de caso, pretende-se utilizar o modelo do sub-sistema de controle de movimento de uma Veículo Aéreo Não Tripulado (VANT), além do modelo do sub-sistema de controle de movimento de uma cadeira de rodas ?inteligente?..
