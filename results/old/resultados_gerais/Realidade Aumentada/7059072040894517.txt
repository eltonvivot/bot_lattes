Decomposição de Benders na Resolução de Problemas de Programação Linear Inteira sobre Multisemigrupos b-Complementarios
Descrição: Este projeto visa implementar a decomposição de Benders para um problema de multisemigrupos. Para se ter uma base de comparação, o problema será testado com os algoritmos convencionais de programação linear sem particionamento. Também pretende-se pesquisar outros algoritmos de particionamento de problemas de programação matemática e compará-lo com a decomposição de Benders para o problema acima mencionado. Estas comparações envolvem análise teórica de complexidade de tempo e espaço dos algoritmos envolvidos e validação com simulações usando entradas aleatórias.. Situação: Em andamento; Natureza: Pesquisa. Integrantes: Yuri Tavares dos Passos - Integrante / Eleazar Gerardo Madriz Lozada - Coordenador / Lucas Carvalho Souza - Integrante.
2015 - 2016
Comparação de algoritmos de controle de tráfego para enxames de robôs com alvos comuns
Descrição: Enxames de robôs são sistemas compostos por uma grande quantidade de robôs, relativamente simples, dispostos no mesmo espaço e interagindo entre si para alcançar um objetivo comum. Nestes sistemas, a execução de uma dada tarefa é processada de modo descentralizado, ou seja, cada robô deve processar as informações obtidas localmente por seus sensores e eventualmente comunicar o resultado aos outros robôs no enxame. Em geral, os robôs possuem um limitado poder de processamento e de comunicação. Devido a esta limitação, os algoritmos desenvolvidos devem ser robustos e escaláveis, garantindo que a inclusão de mais robôs aumentem a eficiência destes sistemas.
Dentre os problemas enfrentados em enxames de robôs, têm-se o controle de tráfego. Neste problema, os robôs devem se coordenar para evitar congestionamentos, especificamente quando os robôs devem se movimentar para o mesmo local simultaneamente. Neste trabalho, pretende-se desenvolver e pesquisar algoritmos distribuídos para o controle de tráfego em enxames de robôs quando os mesmos possuem alvos em comum. Visa-se comparar os algoritmos quanto ao tempo de chegada dos robôs
ao alvo, tempo de saída do alvo e diminuição de congestionamentos.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (1)  / Mestrado acadêmico: (1) . Integrantes: Yuri Tavares dos Passos - Coordenador / Andersoney dos Santos Rodrigues - Integrante.
2014 - 2015
Implementação de uma biblioteca aberta e eficiente para problemas de otimização em matrizes esparsas
Descrição: Matrizes esparsas são matrizes contendo uma grande quantidade de zeros. Este conceito é útil em várias áreas: teoria dos grafos, processamento de sinais, combinatória, soluções de equações diferenciais parciais. Em particular, na Visão Computacional, existem algoritmos no estado da arte que reconhecem objetos numa cena usando técnicas de computação numérica a partir de matrizes esparsas. Deseja-se
implementar tais algoritmos de modo rápido o suficiente para viabilizar sua execução em dispositivos dotados de câmeras, como robôs ou computadores pessoais. Em vista disso, o objetivo deste trabalho é implementar uma biblioteca livre com código aberto na linguagem C para calcular problemas de estimação com matrizes esparsas em tempo aceitável para aplicações de Visão Computacional.. Situação: Desativado; Natureza: Pesquisa. Alunos envolvidos: Mestrado acadêmico: (1) . Integrantes: Yuri Tavares dos Passos - Coordenador.
