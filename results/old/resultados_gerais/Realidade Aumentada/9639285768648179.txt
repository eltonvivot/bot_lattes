WEB - Construção de uma Arquitetura para Desenvolvimento de Aplicações Comerciais em Plataforma Web
Descrição: O projeto proposto tem como objetivo produzir um framework para o desenvolvimento de aplicações comerciais em ambiente Internet, permitindo maior produtividade no desenvolvimento de sistemas corporativos de pequeno, médio e grande porte, total portabilidade, e grande facilidade de integração a novas tecnologias. Construir geradores de código para todas as camadas, a partir da definição dos dados de um sistema. 2004 - 2006
COOVIS - Visualização de Informações Utilizando Visões Coordenadas
Descrição: O projeto proposto tem como objetivo a criação de ferramenta computacional para visualização de grande quantidade de dados, permitindo ao usuário entender melhor o relacionamento entre eles e com isso tornar-se mais apto a confirmar uma decisão, ou tomar uma decisão sobre as informações geradas pelo uso da ferramenta. O projeto terá como premissas os seguintes pontos: x A ferramenta proporcionará uma interface bidimensional intuitiva e amigável aos usuários, permitindo ao mesmo uma maior produtividade quando da execução de suas tarefas; x A ferramenta permitirá ao usuário interagir com mais de uma técnica de visualização, desta maneira um relacionamento não claro em uma técnica pode ser descoberto em outra; x A interação entre as técnicas será sincronizada, permitindo, por exemplo, que a mesma seleção nos dados seja visualizada entre as diferentes técnicas simultaneamente; x As técnicas utilizadas serão mapa em árvore, diagramas de dispersão e coordenadas paralelas; x A ferramenta será desenvolvida utilizando-se a linguagem Java, que permite a portabilidade para diversas plataformas; x A ferramenta permitirá diversas fontes de dados de entrada, desde arquivos textos pré-formatados e estruturas XML a banco de dados relacionais. A área de visualização de informação é um tópico de estudo atual e que se relaciona a diversas áreas, tais como: interação humano-computador, banco de dados, mineração de dados, engenharia de software, programação, etc. A concepção de um ambiente de visualização de dados que contemple tais características funcionais e operacionais certamente se apresenta com uma inovação considerável. Os principais resultados esperados para o projeto são o desenvolvimento e disponibilização nos mercados regional e nacional da ferramenta de visualização proposta e a obtenção de know how na metodologia e aspectos de implementação envolvidos na concepção do projeto..
