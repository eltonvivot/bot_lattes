Middleware 5G - Middleware para Internet das Coisas com uso de 5G
Descrição: Middleware é uma plataforma de software que faz o gerenciamento de dados dos dispositivos e provê interoperabilidade entre os mesmos em ambientes de Internet das Coisas (IoT). Redes de transmissão 5G estão se popularizando entre diversos dispositivos presentes em ambientes IoT. Dentre os objetivos deste projeto está o desenvolvimento de um Middleware para IoT capaz de receber dados de dispositivos 5G e realizar o processamento dos mesmos. É importante o desenvolvimento com foco na alta disponibilidade provida pela tecnologia 5G, além da baixa latência, obrigando a plataforma Middleware a possuir excelente tempo de resposta para requisições.. Blockchain - Uso de Blockchain na Internet das Coisas
Descrição: A Internet das Coisas (IoT) visa a conexão de dispositivos físicos do nosso dia-a-dia através da internet. Estes dispositivos poderão ser acessados remotamente pelo usuário para sensoriamento e atuação. Um dos grandes desafios no desenvolvimento de soluções para IoT é a questão da segurança de dados. Por trafegar dados sensíveis, muitas vezes ligados à informações pessoais de usuários, existe a necessidade latente do desenvolvimento de soluções IoT que considerem abordagens seguras. A tecnologia de Blockchain surgiu como uma das abordagens mais seguras para transmissão e gerenciamento de dados computacionais. Porém, ainda é incipiente o uso de Blockchain juntamente com soluções IoT. Um dos motivos é o alto custo computacional que a tecnologia Blockchain tradicional carrega. Dentre os objetivos deste projeto está o desenvolvimento de soluções inteligentes de Blockchain para uso em ambientes IoT. Uma das alternativas é a substituição do método PoW do Blockchain por métodos mais leves como, PoS, entre outros algoritmos de consenso.. Situação: Em andamento; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2) . Integrantes: Marcos Roberto dos Santos - Integrante / Amilton Martins - Coordenador / Everton Matos - Integrante.

Soluções Edge Computing - Arquitetura para aplicação de soluções Edge Computing
Descrição: Sistemas computacionais tradicionais utilizavam uma camada de nuvem (Cloud Computing) para armazenamento e processamento de dados. Porém, com o avanço da tecnologia, é cada vez mais comum que dispositivos finais possuam um alto poder de processamento computacional, como pode ser visto em smartphones modernos, placas de desenvolvimento como Raspberry, entre outros. A partir disso, surgiu a área de Edge Computing, que visa a distribuição de processamento entre a camada Edge, correspondendo a borda da rede - dispositivo final -, e a camada Cloud, correspondendo a servidores tradicionais. Deste modo, evita-se a transmissão desnecessária de dados, diminuindo a latência e perda de disponibilidade em sistemas computacionais. Porém, ainda existe a necessidade do desenvolvimento de arquiteturas padrão para comunicação e gerenciamento de dados em Edge Computing. Dentre os objetivos deste projeto está o desenvolvimento de arquitetura computacional, utilizando técnicas conhecidas no desenvolvimento de soluções para Internet das Coisas (IoT) como: web services REST, protocolos de comunicação DTLS, LPWA e WebSocket..
