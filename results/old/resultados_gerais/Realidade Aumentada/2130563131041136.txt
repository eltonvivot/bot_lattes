2020 - Atual
Uma plataforma adaptável de jogos para o desenvolvimento de habilidades de crianças com deficiência
Descrição: Esse projeto apresenta a criação de uma plataforma adaptável de jogos desenvolvida especialmente para auxiliar os profissionais que trabalham com crianças com deficiência. Os jogos serão flexíveis, de modo que o especialista poderá configurar o jogo para atender melhor as características específicas de cada criança, por exemplo, com a interação mais devagar ou mais rápida, com nível de dificuldade mais fácil até o mais difícil, com cores básicas ou cores contrastadas, entre outras funcionalidades que podem ser diferentes para cada jogo.  A plataforma estará disponível por meio do navegador de internet e os interessados poderão fazer o download dos jogos na versão desktop e para dispositivos móveis. Esse projeto apresenta uma proposta para uma plataforma adaptável de jogos criados para auxiliar profissionais especializados em crianças com deficiência a desenvolver habilidades específicas. Os jogos devem ser flexíveis para que os especialistas possam trabalhar habilidades que escolherem de acordo com as necessidades atuais, como concentração, atenção, memória, entre outras.. 2018 - 2020
Descoberta de Diretrizes para Jogos Acessíveis e Adaptáveis
Descrição: A Tecnologia Educacional é uma área de conhecimento que propõe a utilização de tecnologias como parte do processo pedagógico de construção do conhecimento. Neste contexto, jogos digitais podem ser integrados no processo educativo, o que permite que crianças experimentem momentos lúdicos como parte do processo de aprendizagem. No entanto, boa parte dos softwares educativos presentes no mercado não possuem opções de customização voltadas para crianças com deficiência ou mobilidade reduzida. Tendo isso em vista, foi desenvolvido uma plataforma de jogos acessíveis e adaptáveis com destaque para o jogo da memória Cuca Fresca e para o jogo AdaptaQuebra-Cabeça. Este projeto será expandido, com o desenvolvimento de novos jogos adaptáveis, bem como a formulação de um conjunto de diretrizes para jogos acessíveis.. Situação: Concluído; Natureza: Pesquisa. 2017 - 2020
Primeiros passos da comunicação por meio de computador para crianças com deficiência
Descrição: Essa proposta de projeto de pesquisa tem como público alvo crianças com deficiências que precisam aprender a interagir como dispositivos eletrônicos (computadores e smartphones). Foi proposto o desenvolvimento de uma ferramenta que ajudará nos primeiros passos da comunicação através da tecnologia visando a iniciação dos usuários no uso das ferramentas voltadas para a área de Tecnologia Assistiva. A ferramenta desenvolvida deve ser um jogo da memória totalmente flexível, em número de cartas, figuras nas cartas e mecanismo de interação. Quanto ao tipo de interação, inicialmente, a proposta é para computadores desktop e dispositivos móveis através de interação convencional. Entretanto, pretendemos realizar pesquisas para incluir novos tipos de interação, como rastreamento de olhos e cabeça, possibilitando, que mesmo as crianças com deficiência que não possuem o movimento das mãos, possam utilizar a ferramenta.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2)  / Mestrado acadêmico: (1) . 2015 - 2018
Aperfeiçoamento do modelo PoliFacets de design da metacomunicação de documentos ativos
Descrição: Atualmente, o PoliFacets é um modelo de design da metacomunicação de
documentos ativos para apoiar o ensino e aprendizado de programação. Por enquanto, a
pesquisa se concretiza em um modelo de design para documentação ativa que fomenta a
reflexão sobre programas, onde o principal objetivo é ajudar os designers a criar
representações que auxiliem os aprendizes na resolução de problemas, apoiando a
reflexão em/sobre a ação. Mas a questão de pesquisa que buscamos responder neste
projeto é: '(Se e) Como o escopo de aplicação do modelo PoliFacets pode ser ampliado?'
e além disso, o que é necessário para que isto ocorra com sucesso? A pesquisa deste projeto
se propõe a responder essas perguntas.. Situação: Concluído; Natureza: Pesquisa.
