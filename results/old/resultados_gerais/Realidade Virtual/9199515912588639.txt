SHoP: A self-adaptive Homogeneous Platform
Descrição: The main goal of this project is to develop an adaptive homogeneous MPSoC platform. We have created a NoC-based MPSoC prototyping built of 16 Spartan3-1000 FPGAs in a mesh-based topology. 2002 - 2003
Hardware/Software Co-design: Um Estudo de Caso Usando Criptografia de Chave Pública
Descrição: Algoritmos de criptografia estão evoluindo para suprir necessidades de segurança. A aritmética modular é parte integrante destes algoritmos, especialmente nos sistemas criptográficos de chave pública. Para obter melhor desempenho do sistema é desejável implementar operadores de aritmética modular em hardware. Muitos algoritmos criptográficos requerem a implementação para operandos com mais de 1024 bits. Adicionalmente, o algoritmo deve ser flexível o bastante para suportar protocolos independentes, uma característica da maioria dos sistemas de criptografia modernos. A reconfiguração dinâmica é uma questão crítica para possibilitar o balanceamento entre os algoritmos criptográficos requeridos para protocolos independentes. Outra questão importante, dada a complexidade de um sistema criptografico, é o projeto integrado hardware/software do sistema (co-design). O objetivo geral do projeto consiste em utilizar o algoritmo RSA como objeto do estudo de caso em co-design com a finalidade de comparar os resultados de um particionamendo equilibrado entre hardware e software com implementações realizadas sem a utilização dessa técnica. Neste momento está sendo realizada uma revisão do estado da arte a respeito de implementações do algoritmo RSA. Em paralelo está o algoritmo está sendo implementado utilizando-se uma linguagem de programação de alto nível (C). Em seguida, a partir dessa implementação será realizada a identificação de quais partes do algoritmo consomem mais processamento (análise de complexidade) para que se possa então implementar tais módulos em hardware. A tendência dos sistemas digitais modernos é possuir partes programáveis e configuráveis, consistindo de componentes de software e hardware (SoC). Pretende-se pois pesquisar e empregar esses sistemas utilizando-se técnicas de co-design, e eventualmente propor e desenvolver ferramenta(s) de apoio ao co-design para sistemas criptográficos.
