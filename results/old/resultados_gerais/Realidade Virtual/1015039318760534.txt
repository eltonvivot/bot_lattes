2020 - Atual
ESTUDO DO EFEITO DA REDUÇÃO DOS DADOS NO DESEMPENHO DAS REDES NEURAIS CONVOLUCIONAIS APLICADAS À SEGMENTAÇÃO DE FÁCIES SÍSMICAS
Descrição: O objetivo deste projeto é avaliar a capacidade de segmentação de fácies sísmicas, realizadas a partir da representação dos dados de entrada por uma fração deles, utilizando a redução progressiva dos dados. Isto é feito a partir da avaliação do desempenho das redes neurais na tarefa de segmentação dos dados sísmicos. Os dados de trabalho são os dados sísmicos do Bloco F3, do setor holandês do Mar do Norte, e o respectivo cubo classificado em função das fácies sísmicas. Os dados de entrada são as seções na direção inline do Bloco F3. Os métodos de redução de dados usados são a separação randômica e separação uniformemente espaçada de dados.. Situação: Em andamento; Natureza: Pesquisa. Alunos envolvidos: Graduação: (1)  / Doutorado: (4) . Integrantes: Lucas Moreira Dias - Integrante / Gerson Gomes Cunha - Integrante / Maria Célia Santos Lopes - Coordenador / Bruno de Souza Silva - Integrante / Carlos Henrique dos Santos Barbosa - Integrante.
2020 - Atual
ANÁLISE DO USO DE FILTROS DE AUMENTO DE DADOS APLICADOS EM DATASETS DE TREINAMENTO DE REDES NEURAIS CONVOLUCIONAIS PARA CLASSIFICAÇÃO DE FÁCIES SÍSMICAS
Descrição: Aumento de Dados (Data Augmentation) é uma técnica que visa a geração de uma maior variabilidade de dados, com consequente aumento da representatividade da amostra no treinamento de um modelo de rede. Dessa forma a proposta desse estudo é a avaliação dos melhores filtros de aumento de dados para classificação e segmentação de fácies sísmicas, através análise quantitativa e qualitativa das métricas das redes geradas e suas predições.. Situação: Em andamento; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2)  / Doutorado: (4) . TÉCNICAS AVANÇADAS PARA VISUALIZAÇÃO INTERATIVA DE DADOS DE SAÍDA DE REDES NEURAIS CONVOLUCIONAIS NA SEGMENTAÇÃO MULTICLASSE DE FÁCIES SÍSMICAS
Descrição: A proposta deste projeto é o desenvolvimento de uma ferramenta para visualização interativa de imagens e dados de segmentação de fácies sísmicas, preditos através de redes neurais convolucionais. Com essa ferramenta é possível a análise qualitativa da performance da rede treinada, as incertezas associadas aos dados preditos e principalmente a visualização, com uma interação dinâmica, e consequente estudo de fácies sísmicas classificadas pelas redes convolucionais.. 2013 - 2014
APLICAÇÃO DE CONTROLE E AUTOMAÇÃO NA SEGURANÇA DE CRIANÇAS NO INTERIOR DE AUTOMÓVEIS
Descrição: O projeto pretende elaborar um estudo para o desenvolvimento de um protótipo de equipamento capaz de identificar o risco iminente e avisar aos pais quando ocorrer um possível esquecimento da criança no interior do automóvel, evitando assim o possível óbito ou prejuízo a saúde da criança. Estudos comprovam que alguns tipos de equipamentos ( celular, plataforma Arduino, chip de celular, Sensor e Cadeira de carro para criança) poderão ser utilizados para detectar o esquecimento de uma criança no interior de um automóvel. Dessa forma, serão consideradas algumas vantagens e custos benefícios na elaboração do protótipo. Será visto: ?...porque as crianças chegam ao óbito?. Novas discussões serão tratadas a respeito das perspectivas futuras para este tipo de aplicação.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2)  / Doutorado: (1) . Integrantes: Lucas Moreira Dias - Integrante / Robson da Cunha Santos - Coordenador / Carlos Henrique de Azevedo Pires Soares Coutinho - Integrante.
