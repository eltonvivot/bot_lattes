Representação visual multinível para auxiliar a exploração de conjuntos de dados
Descrição: Técnicas de projeção multidimensional têm sido largamente utilizadas para auxiliar na exploração e organização de conjuntos de dados. Na visualização, essas técnicas podem representar no espaço 2D ou 3D as relações de similaridade existentes nos espaços de características que descrevem os conjuntos de dados. Um dos problemas que impacta a exploração da representação gráfica gerada por técnicas de projeção é a sobreposição dos marcadores utilizados para representar as instâncias dos dados. A sobreposição pode ser resultante da própria redução de dimensionalidade, da alta similaridade entre as instâncias, e do espaço limitado de tela utilizado para visualizar o conjunto de dados. Na literatura, algumas técnicas de remoção de sobreposição têm sido empregadas para minimizar o problema da sobreposição, entretanto, conforme o conjunto de dados aumenta, também é necessário um maior espaço de tela para visualizar todo o conjunto de dados, exigindo do usuário mais interação para navegar sobre o conjunto projetado. Uma alternativa apresentada por algumas técnicas é realizar uma exploração hierárquica para reduzir o espaço necessário para criar a representação visual ou para sumarizar o conjunto de dados. Este projeto tem o objetivo de auxiliar a exploração de conjuntos de dados por meio da representação multinível do resultado obtido de técnicas de projeção multidimensional. Dessa maneira, o usuário obterá uma visão geral do conjunto de dados sob análise e explorará partes de interesse conforme navega nos diferentes níveis da representação gráfica gerada. Em cada um dos níveis, as técnicas de remoção de sobreposição poderão ser empregadas para evitar a sobreposição dos marcadores. Adicionalmente, representativos de cada nível serão apresentados para fornecer um sumário dos grupos apresentados para o usuário.. Situação: Em andamento; Natureza: Pesquisa. 2014 - 2017
Um Modelo de Coordenação Extensível para Múltiplas Visões Coordenadas e Abordagens para Auxiliar o Processo de Análise Visual
Descrição: Este projeto apresenta duas propostas de trabalho. A primeira tem por objetivo adaptar o modelo de coordenação desenvolvido durante o doutorado do pesquisador principal. Diferentes técnicas de visualização de informação podem ser utilizadas simultaneamente para explorar um ou mais conjuntos de dados. Para isso, técnicas de coordenação podem ser empregadas para facilitar a exploração e evitar a sobrecarga que o usuário sofre na troca de contexto entre as diferentes técnicas de visualização. No doutorado do pesquisador principal foi proposto um modelo com foco na arquitetura de software de uma ferramenta denominada Projection Explorer (PEx), o que dificulta sua extensão para outros sistemas. Como resultado deste projeto, pretendemos disponibilizar um modelo de coordenação extensível, que possa ser facilmente utilizado por qualquer sistema que necessite realizar a coordenação entre múltiplas visões. Também, na linha principal relacionada à coordenação, pretendemos criar uma arquitetura de software que permita a coordenação remota entre múltiplas visões, possibilitando a colaboração entre diferentes pesquisadores. Por fim, adaptaremos uma das técnicas de coordenação para exploração de coleções de documentos para gerar mapeamentos semânticos com o propósito de destacar mais documentos de interesse. Para isso, uma ontologia será empregada durante a etapa de criação do mapeamento. A segunda proposta está relacionada à melhoria da análise visual de conjuntos de dados, na qual pretendemos melhorar a análise de conjuntos de dados por de abordagens híbridas. Nessa abordagem uniremos duas técnicas de visualização, em uma única visão: uma para destacar o relacionamento entre as instâncias (e.g., projeções multidimensionais) e outra para destacar o relacionamento entre os atributos das instâncias (e.g., coordenadas paralelas).. Um modelo de coordenação extensível
Descrição: Diferentes técnicas de visualização de informação podem ser utilizadas simultaneamente para explorar um ou mais conjuntos de dados. Para isso, técnicas de coordenação podem ser empregadas para facilitar a exploração e evitar a sobrecarga que o usuário sofre na troca de contexto entre as diferentes técnicas de visualização. Geralmente, os sistemas de visualização utilizam um modelo de coordenação para gerenciar os mecanismos de coordenação de um sistema de visualização específico, dificultando a sua extensão por outros sistemas que necessitem coordenar múltiplas visões. Este projeto tem por objetivo adaptar o modelo de coordenação desenvolvido durante o doutorado do orientador. Esse modelo foi desenvolvido com foco na arquitetura de software de uma ferramenta denominada Projection Explorer (PEx), o que dificulta sua extensão para outros sistemas. Como resultado deste projeto, pretendemos disponibilizar um modelo de coordenação extensível, que possa ser facilmente utilizado por qualquer sistema que necessite realizar a coordenação entre múltiplas visões.. 2012 - 2013
Modelos para Aprimoramento do Processo Exploratório Apoiado por Múltiplas Visões Coordenadas
Descrição: Atualmente, diversas áreas de aplicação necessitam de mecanismos mais efetivos para analisar dados provenientes de naturezas distintas. Tipicamente, esses dados são abstratos, não estruturados e possuem uma natureza multidimensional (e.g., coleções de documentos). Dados que não possuem uma natureza multidimensional podem ser representados como tal por meio da aplicação de algoritmos extratores de características (e.g., coleções de imagens). Assim, técnicas de visualização de informação projetadas para interpretar dados multidimensionais podem ser aproveitadas para analisar dados não estruturados. Este trabalho empregará técnicas de visualização de informação para construir mapas de similaridade a partir de dados multidimensionais como uma forma de representação desses dados. Também empregará novas técnicas de coordenação desenvolvidas pelo candidato em seu doutorado. Essas técnicas são apoiadas por um modelo de coordenação que estende um outro previamente proposto na literatura. Este trabalho visa dar continuidade a esse modelo, gerando os últimos resultados para que ele possa ser publicado em uma revista internacional. Além disso, ele também tem o objetivo de construir um modelo para armazenar os mapeamentos gerados pelas técnicas de coordenação durante o processo exploratório. Desse modo, esses mapeamentos podem ser recuperados sempre que o usuário desejar, isto é, o processo exploratório não é perdido com o tempo e também pode ser disseminado para colaboradores. Os resultados preliminares desse modelo de armazenamento serão submetidos a um congresso internacional.. Situação: Concluído; Natureza: Pesquisa. 2011 - 2012
Visualizações multidimensionais para exploração de coleções de imagens e dados volumétricos temporais
Projeto certificado pelo(a) coordenador(a) Rosane Minghim em 28/03/2014.
Descrição: Atualmente, diversas áreas de pesquisa necessitam de mecanismos mais eficientes para analisar a grande quantidade de dados provenientes de diversas fontes geradoras de dados. Tipicamente, esses dados são abstratos, não estruturados e possuem uma natureza multidimensional (por exemplo, conjuntos de documentos). Conjuntos que não possuem uma natureza multidimensional (e.g., coleções de imagens) podem ser representados por um modelo multidimensional resultante da aplicação de algoritmos que extraiam características dos dados originais. Assim, técnicas de visualização de informação projetadas para manipular dados multidimensionais abstratos podem ser aproveitadas para manipular dados que não são originalmente multidimensionais. Além desse tipo de aplicação das técnicas de visualização, uma outra tendência é utilizá-las para apoiar a análise de conjuntos de dados que possuem uma representação visual inerente aos dados. Por exemplo, no caso de dados volumétricos, o espaço de características pode ser explorado por meio de uma visualização de informação e o resultado da exploração pode ser refletido no espaço do objeto (modelo 3D).Seguindo essas tendências, o presente trabalho tem o objetivo de utilizar técnicas de visualização multidimensional para auxiliar na exploração de coleções de imagens, melhorando a interação do usuário sobre o resultado das técnicas de projeção multidimensional e técnicas de posicionamento de pontos. Também temos o objetivo de utilizar técnicas de projeção multidimensional para auxiliar na exploração de dados volumétricos, apoiando a identificação de estruturas e padrões em dados volumétricos de uma maneira geral e também em conjuntos que variam no tempo, por exemplo, conjuntos de dados volumétricos provenientes de diferentes iterações de uma simulação. É previsto que parte do desenvolvimento desse trabalho de exploração de dados volumétricos será desenvolvido na Jacobs University em Bremem na Alemanha, sob a supervisão do Prof. Dr. Lars Linsen.. VolVis: Explorando Grandes Conjuntos de Dados Volumétricos Multi-Dimensionais Variantes no Tempo Usando Projeções
Projeto certificado pelo(a) coordenador(a) Fernando Vieira Paulovich em 26/05/2014.
Descrição: Projeções multi-dimensionais são frequentemente empregadas para a criação de representações gráficas de conjuntos de dados abstratos que reflitam algum padrão estrutural existente entre as instâncias de dados multi-dimensionais. Dessa forma, as habilidade visuais humanas podem ser empregadas no processo exploratório dos dados, apoiando atividades de identificação e extração de padrões e anomalias, acelerando a interpretação da informação subjacente. Este projeto propõe a definição de uma abordagem baseada em projeções multi-dimensionais para análise de conjuntos de dados volumétricos, que podem apresentar a característica de serem variante no tempo, suportando tarefas que envolvam a descoberta e seleção de estruturas de interesse a serem visualizadas. Como resultado espera-se a definição de um arcabouço para a análise de dados volumétricos da área médica, como as fibras neuronais, e para a análise de conjuntos variantes no tempo, como os resultantes de simulações de fenômenos naturais, aplicações carentes de ferramentas principalmente quando conjuntos de dados de grande porte são considerados.. 2007 - 2010
Múltiplas visões coordenadas para exploração de mapas de documentos
Projeto certificado pelo(a) coordenador(a) Rosane Minghim em 28/03/2014.
Descrição: O presente trabalho prevê a extensão de um arcabouço mineração visual de conjuntos de documentos, desenvolvido nos último 3 anos, dentro de um projeto de visualização de informação mais amplo. A abordagem adotada até o momento prevê, além de outras funcionalidades que apóiam a interpretação de conjuntos de documentos, a criação de visualizações que mapeiam conjuntos de documentos no plano através de projeções ou de abordagens de posicionamento no plano baseadas em árvores de similaridades. Um sistema existe para apoiar todas as fases da criação desses mapas de documentos. Entretanto, é praticamente consenso nas áreas de visualização e de analítica visual (visual analytics) que múltiplas visões somente não permitem ao analista um entendimento completo dos dados. Uma abordagem de coordenação entre as múltipas visões é necessária. No caso de mapeamentos visuais de texto existe muito pouco desenvolvimento de ferramentas de coordenação, em particular em abordagens que envolvem aspectos de mineração (por exemplo, extração de tópicos), como é o nosso caso. Este projeto propõe o desenvolvimento de uma metodologia para coordenação entre múltiplas visualizações para mapas de documentos, sua implementação no arcabouço anteriormente desenvolvido para mineração visual de textos, chamado Projection Explorer (PEx), e sua avaliação com relação a conjuntos de dados diversificados..
