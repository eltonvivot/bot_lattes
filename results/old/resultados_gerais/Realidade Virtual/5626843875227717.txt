2014 - Atual
Plataforma de desenvolvimento para Androic MIT App Inventor
Descrição: A plataforma App Inventor permite que pessoas com conhecimentos básicos de programação criem aplicativos de software para o sistema operacional Android OS que podem ser executados em bilhões de dispositivos móveis(ou não): telefones celulares, tablets, TVs etc. 
App Inventor é uma plataforma web de código aberto desenvolvida pela Google, e agora mantido pelo Instituto de Tecnologia de Massachusetts (MIT).
O App Inventor, a través de uma interface gráfica amigavel, permite aos usuários arrastar e soltar objetos visuais para criar aplicativos. Dada a sua simplicidade, é utilizado em educação desde crianças até universitarios (na ultima versão foi adicionada uma interface com o LEGO Mindstorms (não abordada neste livro)) O desenvolvimento da lógica do aplicativo ( que usa a metáfora de peças de um quebra-cabeças) foi baseada em teorias de aprendizagem construtivistas.
Alguns comentários sobre o App Inventor chamá-lo limitado ao compará-lo com outras plataformas de desenvolvimento Isso é errado. O App Inventor permite o acesso a todas as funções (atividades) disponibilizadas pelo Android OS. O App Inventor tem se convertido em uma ferramenta poderosa e pronta para produzir aplicativos úteis, comercializáveis e complexos. 
Através de aplicativos simples se abordam os principais módulos de interface e blocos de controle primeiro de forma isolada e depois de forma integrada em aplicativos mais complexos. Se estudarão as funções mais utilizadas na programação de dispositivos móveis: 
?	telefonia, 
?	mensageira (SMS/torpedos), 
?	localização (GPS), 
?	mapas (Google Maps API) 
?	bancos de dados (Google Fusion Tables API). 
O projeto se focaliza na versão AI2 do App Inventor disponibilizada em dezembro de 2013.. Situação: Em andamento; Natureza: Pesquisa.
