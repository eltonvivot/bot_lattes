2005 - 2007
MoUbi - Composição Dinâmica de Software para Computação Móvel e Ubíqua
Descrição: O presente projeto de pesquisa e desenvolvimento tem como objetivo o desenvolvimento de uma infra-estrutura para composiçã dinâmica de software, no domínio de computação móvel e ubíqua, baseada em agentes de software. Tal estrutura, a ser implementada como arcabouços na linguagem Java, deve estar de acordo com a especificação de componentes Compor CMS, assim como padrões da indústria, tais como XML, OSGi e UPnP. (CTINFO / Edital CTINFO/MCT/CNPq nº 11/2005 - PD&I-TI). Situação: Concluído; Natureza: Pesquisa. 2005 - 2007
MobileServices - Arcabouço Baseado em Componentes para a Disponibilização de Serviços para Dispositivos Móveis
Descrição: O objetivo do presente projeto é construir um arcabouço de software baseado em componentes para a disponibilização e acesso a serviços para dispositivos móveis. Espera-se com a infra-estrutura baseada em componentes, prover um mecanismo flexível para a disponibilização de serviços, independente da plataforma ou comunicação utilizada pelos dispositivos móveis, sejam eles provedores ou usuários dos serviços. (Apoio a Projetos de Pesquisa / Edital CNPq 19/2004 - Universal). Situação: Concluído; Natureza: Pesquisa. 2002 - 2003
cVivus - Desenvolvimento de Sistemas Evolucionários para Tratamento de Problemas de Alta Complexidade
Descrição: Este projeto visa o desenvolvimento de algoritmos de alto desempenho, baseados na concepção de algoritmos genéticos, para solução de problemas de alinhamento, sequenciamento e classificação de cadeias de DNA. Uma importante aplicação do produto desse projeto será de apoio técnico ao Projeto Genoma-Cana do Programa FAPEAL-FAPESP. Como decorrência natural e relevante da execução deste Projeto, teremos a capacitação de recursos humanos, de alta qualificação, para o Estado Alagoas.. Situação: Concluído; Natureza: Pesquisa.
