Gestão e Operação de Fábrica de Software
Projeto certificado pela empresa Estratégia Tecnologia da Informação em 23/10/2016.
Descrição: Produtos de software são essenciais para todas as atividades da sociedade contemporânea. No entanto, a produção sistemática de software, com economia, eficácia e qualidade, ainda constitui um grande desafio para a Engenharia de Software. O conceito de Fábrica de Software tem sido utilizado como uma abordagem para enfrentar esse desafio. O objetivo do presente projeto de pesquisa é investigar soluções para promover o aumento da qualidade e da produtividade nos processos de Engenharia de Software realizados no contexto de fábricas de software. Em particular, o projeto busca identificar soluções para problemas ainda não resolvidos em diversos tipos de atividades técnicas e gerenciais de engenharia de software aplicadas em fábricas de software. O projeto utilizará como base para coleta de dados e como contexto de validação experimental de resultados a Fábrica de Software do Instituto de Informática da UFG. Com isto deseja-se identificar os principais problemas e as práticas atuais para a produção de software em fábricas de software em um ambiente acadêmico e controlado, porém similar aos ambientes que são tipicamente utilizados nas fábricas de software utilizadas pela indústria de software. A partir dessa identificação, o projeto analisará a adequação de modelos e normas de melhoria de processos de software para resolver os problemas específicos do contexto de fábricas de software..
