2012 - 2015
Socialização Imersiva: um Ambiente para a Apoiar a Habilitação Social de Pessoas com Deficiências Múltiplas, com foco em TID
Descrição: Este projeto visa o desenvolvimento de software assistivo para auxiliar no tratamento terapêutico no aspecto da socialização de pessoas com transtornos invasivos do desenvolvimento (TID). O software produzido será desenvolvido na plataforma de realidade virtual OpenSim constando de cenários adequados a situações sociais vivenciadas na vida diária de pessoas com esses transtornos. Para isso, serão utilizadas abordagens terapêuticas em conjunto com técnicas de computação, especialmente os scripts colaborativos.. Situação: Concluído; Natureza: Pesquisa. 2012 - 2015
Sistema Colaborativo de Imersão Musical para Crianças com Autismo
Descrição: Este projeto visa gerar um sistema colaborativo imersivo para a prática de atividades de musicalização para crianças autistas, chamado de Music Spectrum. Para isso, o ambiente será desenvolvido utilizando a arquitetura da plataforma do OpenSimulator (OpenSim), plataforma de código livre, multiusuário e que permite a criação e edição de aplicações e modelagem em 3D. O Music Spectrum se propõe a auxiliar no processo de desenvolvimento de habilidades de interação social das crianças com TEA, visto que autismo é um complexo conjunto de distúrbios sociais, marcado pelo prejuízo na interação social, comunicação e padrões limitados ou estereotipados de comportamentos.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2)  / Mestrado acadêmico: (2)  / Doutorado: (2) .
