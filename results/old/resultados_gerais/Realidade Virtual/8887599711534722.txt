2012 - Atual
U-Store Infra estruturas de armazenamento de dados em computação nas nuvens
Descrição: Com o aumento da velocidade de conectividade e evolução dos sistemas WEB, começam a surgir sistemas de Internet mais complexos e poderosos, que mais comumente são chamados de Computação nas Nuvens. Computação nas Nuvens é uma plataforma de suporte a sistemas de software que provê aos seus usuários: gerenciamento, uso sobre demanda, adequação às necessidades, racionalização do uso dos recursos e automação dos processos relacionados a criação de infra-estruturas de suporte. Neste contexto, o objetivo deste projeto é associar os conceitos que norteiam a computação nas nuvens e utilizar a plataforma P2P como mecanismo eficiente na implementação de um sistema de armazenamento e retenção de dados na nuvem, com uma interface WEB e tendo como objetivo principal fornecer acesso a esta plataforma como serviço. Este sistema irá utilizar a capacidade de armazenamento ociosa, disponível nos discos de qualquer maquina, sendo ela servidor, Desktop de usuário, celulares, pen drivers, entre outros, para o armazenamento destes dados. Desta forma, se dispensa a existência de sistemas centrais uma vez que caracteriza cada Peer como parte da composição final de armazenamento, diferenciando-se desta forma das demais propostas existentes. Ainda como resultado deste projeto será disponibilizado para download em um portal de acesso público e gratuito a ferramenta implementada para os usuários interessados e, assim, começar a criação da plataforma de dispositivos, ou comunidade, que darão suporte aesta plataforma de armazenamento de dados em nuvem acessível como serviço e aderente aos conceitos de computação nas nuvens..
