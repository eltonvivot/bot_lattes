Acesso a Recursos Informacionais para Deficientes Visuais (PROBIC 2008/2830)
Descrição: Na atualidade o acesso à informação e comunicação por meio do uso da informática tornou-se indispensável no cotidiano das pessoas. O uso de ferramentas como computadores e a internet ajudam na obtenção de informações e conhecimento fornecendo um maior e mais fácil acesso a vários tipos de conteúdos. Com os constantes avanços que ocorrem na área tecnológica, estes meios de informação acabam exercendo uma grande influência nas diversas atividades do cotidiano da população.  Para poder ter acesso a estes recursos os indivíduos acabam tendo de procurar adequar-se a esta nova realidade, onde eles buscam atualizar-se constantemente. Neste cenário, os portadores de deficiências têm uma dificuldade adicional considerando que, para este grupo, o avanço tecnológico não se dá na mesma proporção.  Em particular, estão os deficientes visuais que, devido às suas limitações visuais, carecem de ferramentas que os auxiliem no seu acesso aos meios de informação. Este trabalho pretende identificar as necessidades e dificuldades dos deficientes visuais em relação ao acesso aos meios de comunicação e informação, servindo, assim, de auxílio ao desenvolvimento de metodologias e ferramentas para os deficientes visuais.. Uma Ferramenta Didática de Apoio ao Ensino da Linguagem SQL (PROBIC 2005/273)
Descrição: O objetivo do projeto proposto é o desenvolvimento de uma ferramenta didática que possa apresentar ao usuário, alunos de banco de dados em potencial, os passos seguidos pelo SGBD para a obtenção do resultado de uma consulta SQL. Nessa ferramenta, conectada a um SGBD relacional, o usuário descreve uma consulta SQL e lhes são apresentados os resultados intermediários do processamento de cada uma das cláusulas da consulta, até que o resultado final seja atingido. A apresentação desses passos facilita a compreensão por parte do aluno e torna o ensino da linguagem SQL menos subjetivo.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (1) . Uma Ferramenta Didática para Projeto de Banco de Dados Utilizando o Modelo Entidade-Relacionamento (PROBIC 2005/271)
Descrição: O objetivo do projeto proposto é a implementação de uma ferramenta CASE didática, onde alunos possam construir esquemas conceituais de bancos de dados utilizando a mesma notação gráfica do modelo ER. A partir desse esquema conceitual, a ferramenta pode gerar, de acordo com regras de mapeamento entre o modelo ER e o modelo relacional, os comandos SQL necessários à criação do banco de dados em algum SGBD relacional. Tal ferramenta é de grande auxílio ao processo de ensino/aprendizagem de projeto de banco de dados, unificando a notação gráfica do modelo ER abordada na literatura e a utilizada nas ferramentas CASE de apoio.. Desenvolvimento de um Sistema de Banco de Dados para o Programa de Erradicação do Trabalho Infantil (PETI) (PROBIC 2004/191)
Descrição: Anualmente, novas crianças e adolescentes são incluídos no PETI, programa do Governo Federal, que tem como objetivo retirar crianças e adolescentes de 07 a 14 anos de idade do trabalho considerado perigoso, penoso, insalubre ou degradante, ou seja, que coloque em risco sua saúde e segurança. Tal inclusão é feita a partir da constatação de que a família possua uma renda familiar per capita de até meio salário mínimo. Com isso, o PETI gera um considerável volume de informação dessas crianças e adolescentes, que, por falta de maiores recursos, é armazenado e gerenciado de forma manual. O projeto propõe o desenvolvimento de um sistema de banco de dados informatizado para o armazenamento e gerenciamento mais eficiente da informação relativa às crianças e adolescentes assistidos pelo PETI. O banco de dados informatizado possibilita a melhoria das condições de manipulação dessa informação, visando uma maior eficiência no manuseio da informação sobre cada uma das crianças e adolescentes assistidos pelo programa. Assim, torna-se possível o acesso mais rápido à informação, por armazená-la de forma mais organizada e automatizada.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (1) .
