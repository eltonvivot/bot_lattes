2016 - 2017
Sistema web para classificação climática das regiões brasileiras
Descrição: Através da classificação climática determina-se o tipo de clima de uma região e conhecer as características climáticas de uma região é fundamental na hora da tomada de decisão no que diz respeito ao plantio de determinada cultura. Antes de se efetuar uma classificação climática necessitam-se dos dados de excesso de água no solo (EXC), deficiência de água no solo (DEF) e evapotranspiração potencial (ETP), sendo estes obtidos através do cálculo do balanço hídrico, que nada mais é do que contabilizar as entradas e saídas de água no solo. Esse processo, na maioria das vezes, é manual e lento devido a grande quantidade de informações que precisam ser analisadas. Assim, o objetivo desse projeto é analisar as informações envolvidas nesse processo e realizar uma classificação climática para todas as regiões brasileiras, disponibilizando ao usuário um panorama atualizado das mesmas. Essa classificação climática será gerada a partir de um sistema que será desenvolvido para a plataforma web. Para ter acesso as informações o usuário só necessitará acesso à internet, seja pelo computador ou dispositivo móvel tal como celular ou tablet.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2) . 2015 - 2016
Estudo e Desenvolvimento de software Meteorológico para auxílio na agricultura
Descrição: Determinar a melhor época para o cultivo de determinados tipos de sementes e grãos pode ser um diferencial para se obter uma colheita bem-sucedida se levando em consideração o mínimo de desperdício possível. Assim, é necessário definir o tipo de cultura que será trabalhado e também qual a melhor época do ano para o plantio da mesma. Para se determinar isso é necessário se obter a Evapotranspiração Potencial (ETP) que se refere à quantidade de água que seria utilizada por uma extensa superfície vegetada com gramado, cobrindo totalmente a superfície do solo e sem restrição hídrica, indicando os períodos onde há excesso ou escassez de água no solo.
O objetivo desse trabalho é estudar alguns dos métodos disponíveis para cálculo da ETP e metodologia para cálculo do balanço hídrico, além de desenvolver um software com versões para Desktop e também para dispositivos móveis que utilizam o sistema operacional Android para automatização dos cálculos envolvidos. No caso específico do desenvolvimento para Android, será necessário um estudo das tecnologias utilizadas no desenvolvimento para dispositivos que utilizam esse sistema operacional. Com isso, o usuário poderá identificar qual é o mês ideal para o plantio de suas culturas, analisando os períodos de escassez e excesso de água no solo.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2) . 2003 - 2005
MetaMídiaWEB - Modelo de Metadados na Indexação e Recuperação de Objetos Multimídia na WEB
Descrição: O tratamento de objetos multimídia tem sido problemático devido à complexidade de suas estruturas, à análise de grande conjunto de dados armazenados e ao baixo desempenho de recuperação quando a pesquisa em conteúdo é efetuada. Na indexação alguns problemas relacionados à gestão de metadados devem ser atacados, tais como: (i) metadados tomam uma variedade de formas; (ii) novos conjuntos de metadados são criados; (iii) distintos usuários podem propor diferentes tipos de metadados; (iv) adoção de diferentes vocabulários de metadados significa aumento de buscas usando vocabulários que não são familiares. Uma arquitetura de informação deve ser flexível para permitir inclusão de metadados à medida que novas necessidades apareçam. Tendo em vista o exposto, o objetivo deste projeto é apresentar um modelo de metadados na indexação e recuperação de objetos multimídia para internet. A metodologia a ser utilizada envolve: definir requisitos para metadados que devem estar disponíveis aos usuários, desenvolver a arquitetura de metadados e implementar o modelo proposto. Dentre os resultados esperados destacam-se: (i) integração de dados de diferentes bases disponíveis através da Internet, através do uso de uma base de metadados descritivos; (ii) indexação de objetos multimídia, usando um modelo de metadados; (iii) compartilhamento de metadados entre objetos, através de mecanismo de herança; e (iv) utilização da composição de objetos para reduzir os resultados de pesquisas na Internet. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (10) .
