2012 - 2014
Physical Unclonable Functions for SoC Devices
Descrição: We propose the study of Physical Unclonable Functions as an alternative to computationally hard problems commonly used as basis for cryptographic protocols. PUFs are functions uniquely determined by the very physical properties of a given electronic device, meaning that they can?t be cloned since the exact physical structures of a device can?t be recreated. In this project, we propose to develop specific protocols for user authentication and key exchange based on PUFs. Such protocols are of great importance for e-banking, e-commerce and secure transactions applications, which are rapidly expanding markets in Brazil.. Situação: Concluído; Natureza: Pesquisa. 2012 - 2013
Desenvolvimento e demonstração de Tecnologias de Smart Grid para Integração em Sistemas de Rede e Telecomunicações
Descrição: Atualmente no Brasil a Resolução Normativa Nº 469 da ANEEL, de 13 de Dezembro de 2011, que modificou os Procedimentos de Distribuição ? PRODIST, institui uma série de indicadores da qualidade do serviço de distribuição de energia elétrica. Tais indicadores podem ser obtidos a partir de um sistema de smart grid, orientando as operadoras de forma a melhora o serviço quanto provendo dados necessários para fiscalização e regulação do mercado. O projeto proposto tem como objetivo instalar uma pequena infraestrutura piloto de smart grid para demonstração e estudo de viabilidade de tecnologias atuais. Além disso, o projeto visa a desenvolver um protótipo de equipamento de medição de SG capaz de coletar dados sobre consumo de energia e disponibilidade do serviço em domicílios e instalações de consumidores. Através deste objetivos, espera-se obter um melhor entendimento sobre as tecnologias de SG comercialmente disponíveis, bem como dar um primeiro passo em direção ao desenvolvimento de tecnologia nacional para este tipo de sistema. A primeira fase deste projeto consiste na análise de tecnologias de smart grid comercialmente disponíveis, de forma a se instalar uma infraestrutura piloto e se desenvolver um protótipo de sistema para interação de usuários com o sistema de SG, a partir do qual usuários podem verificar seu consumo e comprar energia. A presente proposta também visa a estudar opções de tecnologias disponíveis para smart grids e desenvolver um protótipo nacional de um equipamento para medição de consumo e indicadores para consumidores finais que se integre a sistemas SG. Este equipamento de medição deve coletar dados sobre consumo e disponibilidade do serviço de distribuição de energia e enviá-lo para uma central agregadora de dados a partir de uma rede de comunicação específica. Para este propósito serão avaliadas diversas tecnologias de rede e de medição, de forma a determinar a combinação de tecnologia com melhor relação custo/benefício para o cenário brasileir. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (4)  / Mestrado acadêmico: (1)  / Doutorado: (1) . Integrantes: Bernardo Machado David - Integrante / Anderson Clayton Alves do Nascimento - Coordenador / Marcelo Dias Holtz - Integrante / Anésio de Leles Ferreira Filho - Integrante.
