2020 - 2020
Oportunidades e Desafios no Gerenciamento de Dados com a Tecnologia Blockchain
Descrição: CÓDIGO: UFCVIRTUAL-07/2020. A tecnologia Blockchain fornece uma série características como: irrefutabilidade, auditabilidade, distribuição de dados, replicação de dados, imutabilidade, disponibilidade, segurança e criptografia. Este projeto de pesquisa leva em consideração, um premissa, que aplicações possuem requisitos de dados diferentes. Assim, aplicações que possuem requisitos de dados que estão associados a essas características supracitadas da tecnologia Blockchain podem se beneficiar desta tecnologia. Este projeto tem como objetivo geral identificar oportunidades e desafios de pesquisa no gerenciamento de dados com a tecnologia Blockchain diante dos requisitos de dados das aplicações. Situação: Concluído; Natureza: Pesquisa. Integrantes: Leonardo Oliveira Moreira - Coordenador.
2017 - 2019
Uma Plataforma para o Gerenciamento de Serviços Persistentes de Dados em Ambientes Distribuídos
Descrição: Este projeto tem como objetivo desenvolver uma plataforma para gerenciar serviços no contexto da persistência de dados que atendam aos requisitos em ambientes de computação distribuída. Inicialmente, este ensaio realiza um estudo exploratório a fim de pesquisar as plataformas que gerenciam e manipulam persistência de dados no âmbito de infraestruturas de distribuição. Diante disso, são elencados os requisitos desejáveis que são necessários para existirem na plataforma proposta. Além disso, este projeto realiza um estudo, mais aprofundado, das técnicas existentes e utilizadas para realizar as características de serviços de persistência de dados em ambientes distribuídos com qualidade de serviço, para um entendimento teórico e técnico da viabilidade e aplicabilidade destas técnicas. Diante dos resultados do estudo, são concebidas tecnologias ou frameworks a serem utilizados para auxiliar no processo de desenvolvimento do objetivo geral deste trabalho. Com isso, é desenvolvida, conforme os requisitos elencados, a plataforma de gerenciamento de serviços persistentes de dados no âmbito distribuído. Por fim, aspectos de avaliação de eficácia e eficiência da plataforma são analisados e discutidos em aplicações e ambientes reais. Situação: Concluído; Natureza: Pesquisa.
