1999 - 2000
MUSEUVIRTUAL: Ferramenta de Autoria para Criação de Museus em Realidade Virtual para apoio à Aprendizagem Colaborativa via Internet
Descrição: The  MuseuVirtual  project focuses on the development of authoring tools for the collaborative building of non-immersive museums in virtual reality. A number of prototypes have been already developed. The rationale behind museuVirtual is based on the observation that virtual museums found on the Internet normally have very limited opportunities for user interactivity. We examined different kinds of interactivity and proposed an authoring tool that allows a teacher or student to create virtual presentations where dynamic interactive objects and chatterbots may be programmed. This proposal is strongly based on the concepts of actor-based programming and constructivist theory, which have been applied in the last five years to teach basic programming skills to beginners. The museuVirtual project is developing a tool that not only allows the students to be the organizers and curators of their own museums, but also to work collaboratively, by building such museums in a virtual distributed environment. The research topics approached in this project were: collaborative authoring of VR environments, ergonomic design, VR servers, chatterbots in VR environments, actor-based language and design patterns to program behavior of VR dynamic objects.

1998 - 2000
Processos e Técnicas de Engenharia de Software para Desenvolvimento de Aplicações de Realidade Virtual
Descrição: Como qualquer sistema de software, a construção de ambientes virtuais requer um processo de desenvolvimento adequado às suas peculiaridades. No entanto, como a área de RV é muito recente, não existem ainda, na literatura, trabalhos suficientes sobre como desenvolver tais aplicações de software. A necessidade de se ter um processo sistemático, que dirija e facilite a criação de ambientes e aplicações de RV deu origem a uma série de estudos, dos quais resultou o presente projeto. O projeto enfoca o desenvolvimento de aplicações e ambientes de RV, compreendendo: (a) Definição de um processo iterativo de desenvolvimento de software, adaptado para a criação de sistemas de RV. O processo proposto está sendo testado através de estudos de casos e implementações; (b) Desenvolvimento de uma ferramenta de simulação, com recursos de RV, para apoiar a validação de sistemas de tempo real. A ferramenta está sendo implementada e testada através de exemplos típicos de sistemas de controle de processos.

RESE - Remote Education System using a Coupled CAVE / Workbench/ Monitor
Descrição: Virtual environments build a very important and interesting basis for educational computer systems because they enable the teaching person to ilustrate the educational content in three dimensions with stereo viewing. Using network technology it is further possible to bring geographically dispersed groups in such a virtual environment, together with the discussed content. Thus, it is possible for a teacher to give lessons from a remote place using a virtual environment while the students can participate being somewhere else without loosing the joint viewing. The aim of the project is to develop such a system for education using three types of virtual environments like a CAVE, a responsive workbench and monitors coupled over a network. The educational contents will be delivered by the medical/anatomical field and the agricultural field. The RESE Project has been developed in cooperation with the German National Research Center for Information Technology - Institute for Media Communication -  Department of Virtual Environments - Fraunhofer Institute of Sankt Augustin, Germany.
