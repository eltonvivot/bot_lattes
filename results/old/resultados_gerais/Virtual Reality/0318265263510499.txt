2008 - Atual
Prisma Web (FAPESPA 1009/2008) - Ferramenta Para Visualização De Grandes Bases De Dados Em Plataforma Web
Descrição: Dentre as várias técnicas utilizadas no processo KDD, a visualização de informação e a exploração visual de dados têm importante papel, pois permitem que o usuário formule rapidamente hipóteses sobre uma grande quantidade de dados complexos e explore-os em inúmeros cenários dinâmicos. O desenvolvimento de ferramentas de visualização de informação em plataforma Web traz como desafios nas áreas de otimização de algoritmos, usabilidade, trabalho coordenado, segurança, escalabilidade e muitas outras. O projeto proposto tem como objetivo criar uma aplicação em ambiente web para visualização de grande quantidade de informação utilizando múltiplas visões coordenadas. Para atingir o objetivo, será necessária a construção de uma arquitetura para a camada servidora da ferramenta, de modo a dar flexibilidade a novas técnicas de visualização. A ferramenta será baseada em plataforma Java. Para suporte a grandes quantidades de dados, serão desenvolvidos algoritmos para manipulação dinâmica e interativa com base na seleção de amostras significativas...
