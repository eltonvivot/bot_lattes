2013 - 2014
Auto-ajuste de Sistemas Gerenciadores de Banco de Dados, utilizando técnicas de computação evolucionária
Descrição: Fatores econômicos tem levado ao crescimento das infraestrutura e fornecimento de processamento computacional como um serviço, também conhecido como Computação em nuvem ou Cloud Computing, no qual o consumidor do serviço tem algumas garantias de qualidade de serviço (QoS), como disponibilidade e desempenho, que são expressas através de um acordo de nível de serviço (SLA), que especificam também níveis de qualidade e penalidades em caso de falha.  Sistemas de gerenciamento de banco de dados (SGBDs) para computação em nuvem devem estar preparados, para prover a elasticidade, isto é, a capacidade de adicionar ou diminuir recursos, bem como atender as métricas definidas em uma SLA. Neste contexto, uma característica chave é o monitoramento constante do desempenho do SGBD. No ajuste de desempenho de SGBDs, um dos aspectos principais é o o problema de Selecção de Índice (ISP), frequentemente realizada por Administradores de Sistemas de Banco de Dados (DBA), para acelerar o desempenho de comandos submetidos. Neste trabalho, algoritmos genéticos (AG) são explorados com o objetivo de demonstrar que oferecem uma solução eficiente e adequada para o problema de Seleção de Índice, desenvolvendo uma ferramenta que proporcione o auto-ajuste de índices em um SGBD, exigindo o mínimo de intervenção de um DBA.. Situação: Concluído; Natureza: Pesquisa. Integrantes: Wendel Goes Pedrozo - Coordenador / Adriano Serckumecka - Integrante.
2012 - 2013
Exploração de fontes de dados RDF na Web Semântica
Descrição: Propõe-se neste trabalho estudar um modelo de exploração dos dados da Web Semântica, que estarão semi-estruturados e distribuídos na internet.
Tal modelo de exploração permitirá a um usuário explorar a Web Semântica mesmo tendo conhecimento superficial sobre determinado domínio, dando ao usuário uma interface para consulta a fonte de dados RDF, com funcionalidades de busca e navegação, além de possibilitar a manipulação de subconjuntos dessa base, como a realização de operações de união, diferença e interseção.. Situação: Em andamento; Natureza: Pesquisa. Integrantes: Wendel Goes Pedrozo - Coordenador.
