2021 - Atual
Implementação de modelo de otimização e inteligência artificial para o projeto integrado de sistemas offshore
Descrição: Dentre os maiores custos associados ao projeto de sistemas oceânicos, destaca-se a obtenção de uma solução inicial de projeto, bem como a otimização dos parâmetros envolvidos; visando a melhoria de desempenho do sistema. Para mitigar essa problemática, apresenta-se nessa proposta o desenvolvimento de modelos de otimização aplicados ao projeto de sistemas offshore. Dentro do contexto geral, inclui-se 4 frentes principais: desenvolvimento de modelos para a otimização de linhas em lazywave de risers rígidos, desenvolvimento de modelos para a otimização de sistemas flutuantes, desenvolvimento de modelos de otimização para o projeto de sistemas de ancoragem em configuração spread mooring e o desenvolvimento integrado de sistemas de produção de petróleo. O sistema a ser desenvolvido usará como base o ambiente Synapse, de autoria do coordenador deste projeto, que vem sendo desenvolvido a muitos anos e destina-se ao projeto de sistemas complexos de engenharia. O sistema Synapse permite um elevado grau de customização e contém, além de uma interface gráfica, diversos algoritmos associados a otimização multidisciplinar e inteligência artificial. Assim, propõem-se a customização do sistema Synapse implementando diversas funcionalidades desenvolvidas especialmente para o projeto integrado, otimizado e automatizado de sistemas de produção offshore, tal como descrito nas etapas e atividades propostas. Como procedimento de verificação, diversos casos conhecidos serão analisados, e as respostas obtidas serão confrontadas com as respostas disponíveis. Por fim, novos casos serão desenvolvidos mostrando todo o potencial da ferramenta implementada.. Situação: Em andamento; Natureza: Pesquisa. 2019 - 2022
Desenvolvimento de técnicas e ferramentas para o projeto otimizado de sistemas veiculares. Fase 1: Resistência ao avanço
Descrição: Inserido no âmbito das atividades do Laboratório de Otimização e Projeto Integrado (LoopIn ), Laboratório de Simulação Naval (LaSin ), esta proposta propõe o desenvolvimento de modelos computacionais para o projeto otimizado de sistemas veiculares. Os novos avanços científicos realizados na pesquisa de novos materiais e técnicas de fabricação permitem que projetistas repensem os atuais projetos de sistemas veiculares, visando soluções mais leves e eficientes. No atual cenário de competição global, qualquer diferencial que represente uma vantagem estratégica é uma importante meta a ser perseguida por pesquisadores, projetistas e armadores. Neste contexto, propõe-se o desenvolvimento de uma metodologia de projeto de sistemas veiculares, tendo como foco o desenvolvimento de técnicas e ferramentas para o projeto otimizado de embarcações, aeronaves, veículos, entre outros. Dessa forma, busca-se uma metodologia que incorpore uma interface eficiente e um algoritmo de otimização capaz de realizar mudanças no projeto, direcionando para a solução otimizada. Nesta primeira fase os esforços se concentrarão no desenvolvimento de uma metodologia para a análise da resistência ao avanço de embarcações. Tal metodologia deve estar integrada ao ambiente de projeto Synapse e deve ser robusta e eficiente, para contemplar análises multifásicas considerando o movimento do corpo; tal como ocorre na análise de embarcações de alta velocidade. Inicialmente propõem-se a utilização da plataforma OpenFOAM para a análise numérica da resistência ao avanço e espera-se como resultado, uma integração robusta, eficiente e transparente ao usuário entre o sistema Synapse e a plataforma OpenFOAM.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (7) . Integrantes: Thiago Pontin Tancredi - Coordenador / Ernane Silva - Integrante / Filipe Dutra da Silva - Integrante.

2011 - 2014
Pacote Nacional de Projeto e Maquinário de Embarcações de Apoio Marítimo à Produção de Petróleo no Mar
Projeto certificado pelo(a) coordenador(a) Helio Mitio Morishita em 27/02/2014.
Descrição: Atualmente está havendo uma retomada da indústria de construção naval no país devido em grande parte à indústria do petróleo em alto mar. No entanto, no caso dos barcos de apoio marítimo, a maioria das embarcações que são atualmente fabricadas no Brasil são baseados em pacotes de projeto e maquinário importados. Para reverter esta situação, e por entender que o aprimoramento da capacitação da engenharia naval brasileira é importante para o país, este projeto tem por objetivo o desenvolvimento de projetos de embarcações de apoio à plataformas de petróleo instaladas na região do pré-sal, com elevado padrão de desempenho, empregando sistemas com viabilidade de fabricação no país..
