2017 - 2018
Jogos Digitais: construindo um sistema de busca para o enfrentamento do estresse ocupacional
Descrição: A escolha de quais características dos jogos digitais a serem armazenadas em uma base de dados é uma tarefa importante no sentido de oferecer, a um sistema de busca, meios para escolher as melhores opções de jogos para o enfrentamento do estresse ocupacional.Este projeto visa a construção de uma base de dados com jogos, no contexto do estresse ocupacional. Dessa forma, o ato de jogar pode se tornar um importante aliado para os profissionais afetados pelo estresse e também para as empresas no sentido de diminuir o montante de recursos financeiros gastos para o tratamento das consequências causadas pelo estresse.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (1) . Integrantes: Renato de Aquino Lopes - Coordenador / Danilo Gonçalves Vieira - Integrante.
