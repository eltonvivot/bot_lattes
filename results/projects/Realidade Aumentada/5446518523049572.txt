2003 - Atual
Investigação de Técnicas de Mineração de Dados Multi-Relacional e em Hierarquias de Generalização
Descrição: Esse projeto tem por objetivo o desenvolvimento de técnicas para a mineração de dados envolvendo múltiplas tabelas de um banco de dados relacional. Para tanto, há necessidade de adaptações/extensões dos algoritmos existentes de mineração de dados, pois estes processam apenas uma tabela. Dependendo do tipo de relacionamento entre as tabelas, a junção de várias tabelas em uma única pode levar a perda de informações, daí a necessidade de realizar a mineração nas relações, mantendo-as separadas.
Uma outra investigação nesse projeto refere-se à mineração de dados em hierarquias de generalização. O relacionamento entre classe/subclasse da hierarquia pode conduzir à descoberta de regras dependentes de classe, mostrando diferenças interessantes entre o comportamento do conjunto de dados como um todo e o comportamento de cada subclasse da hierarquia.. Desenvolvimento de Software Baseado em Componentes Multimída
Descrição: Esse projeto propõe pesquisar técnicas, métodos e ferramentas., objetivando a definição de estratégias que auxiliem o desenvolvimento de software baseado em componentes multimídia. Aplicações de um domínio, cujos componentes foram construídos, poderão ser desenvolvidas fazendo reutilização desses componentes armazenados em banco de dados. As estratégias a serem investigadas serão responsáveis, numa primeira etapa, pela realização do processo de desenvolvimento de componentes multimídia e, numa segunda etapa, pelo processo de desenvolvimento das aplicações multimídia que reutilizam os componentes. Nessas etapas estão envolvidas atividades que vão desde a definição do domínio do problema e modelagem dos componentes, até a implementação, recuperação e reutilização desses componentes pelas aplicações do domínio.. IMiMD - Indexing and Data Mining in Multimedia Databases
Descrição: Esse projeto tem por objetivo o desenvolvimento de técnicas, algoritmos e ferramentas de software para suportar o tratamento de informações multimidia e dados com estrutura interna complexa. O projeto é centrado em utilizar o conteúdo dos dados para possibilitar a indexação e recuperação baseada em seu conteúdo, bem como a descoberta de padrões e aglomeramentos (clusters) nos dados. Para atingir tais objetivos, esse projeto integra as áreas de Banco de Dados, Inteligência Artificial e Visualização de Dados. Uma segunda grande frente de pesquisa desse projeto é data mining e reconhecimento de padrões em grandes conjuntos de dados. Essa frente de trabalho utiliza três diferentes bases conceituais e tecnologias: tratamento de visualização de grandes conjuntos de dados, tratamento semântico e utilização de técnicas de Inteligência Artificial e Redes Neurais.

1997 - 2002
AMMO - Authoring and Manipulation of Multimedia Objects
Descrição: O objetivo desse projeto é prover um ambiente para suporte à autoria, armazenamento e manipulação de aplicações multimídia. Foi desenvolvida a arquitetura de um servidor de objetos multimídia, cujos módulos de armazenamento e recuperação de objetos multimídia foram inicialmente implementados utilizando o sistema gerenciador de banco de dados orientado a objetos O2. A estrutura do banco de dados multimídia, nessa implementação, seguiu o padrão MHEG-5 para a representação da estrutura de classes do banco de dados multimídia mantido no ambiente. Posteriormente uma nova versão dessesmódulos e do banco de dados foi desenvolvida, seguindo o padrão SMIL e utilizando o SGBDOO Jasmine. Pesquisas mais recentes no ambiente têm utilizado o SGBD pos relacional Caché. 
Duas ferramentas de autoria multimídia, uma baseando-se no padrão MHEG-5 e a outra no padrão SMIL, foram desenvolvidas como parte desse projeto. 
Pesquisas no contexto do projeto exploraram vários aspectos do banco de dados multimídia, tais como: busca por conteúdo nebulosa em diferentes tipos de mídia e processo de extração de conhecimento em base de dados multimídia..
