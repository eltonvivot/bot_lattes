2012 - 2013
Projeto DroidMensor
Descrição: Trata-se de um projeto de iniciação científica para ensino médio - PIBIC-EM. Este projeto visa desenvolver aplicativos em Sistema Android voltados para serviços de Levantamentos Topográficos e para a criação de bancos de dados para Sistemas de Infromações Geográficas, podendo ser utilizados em smartphones, celulares.. Situação: Concluído; Natureza: Pesquisa. Integrantes: Silvio Henrique Dellesposte Andolfato - Coordenador / Wellygton W. de Azevedo - Integrante / Kauê de Moraes Vestena - Integrante / Bruno Marcelo Walter Szura - Integrante / Josiane de Araujo - Integrante.Financiador(es): Conselho Nacional de Desenvolvimento Científico e Tecnológico - Bolsa.
