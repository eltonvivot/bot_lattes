2011 - Atual
Desenvolvimento de arquitetura de baixo custo para monitoramento de condições do tempo em áreas agrícolas por meio de miniestações agrometeorológicas e rede sem fio
Descrição: Este projeto propõe o desenvolvimento de uma rede de estações de monitoramento de dados agrimeteorológicas de baixo custo, que são capazes de se comunicar através de redes sem fio e transmitir as informações de monitoramento para bases de dados remotas. A partir  desses dados de monitoramento pretende-se melhorar o  planejamento estratégico das propriedades agrícolas e ampliar-se o uso das técnicas de agricultura de precisão no campo. Inicialmente, os dados que serão monitorados pelas miniestações serão os dados de temperatura do ar, umidade do solo e precipitação (chuva) os
quais são extremamente importantes para o planejamento e execução das práticas agrícolas.. Situação: Em andamento; Natureza: Pesquisa. Alunos envolvidos: Graduação: (6) .
