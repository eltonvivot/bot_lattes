2020 - 2021
Desenvolvimento de uma Ferramenta para Automatizar a Geração de Horários de Aula
Descrição: O gerenciamento de horários é um dos problemas mais recorrentes em qualquer instituição de ensino. Apesar das diversas ferramentas existentes no mercado, muitas das vezes com preços elevados, as necessidades de uma instituição nem sempre são atendidas, muitas das vezes devido ao fato de ser um problema complexo e a sua modelagem ser complexa, gerando assim uma grade de horários que acarretará em uma eventual necessidade de realizar mudanças manuais, para só depois poder realizar a geração do horário. Com isso, o estudo das técnicas atuais e com o poder computacional crescente, a criação de um software personalizado é uma das soluções para este problema. Este software vem ao ponto de se evitar a criação de uma grade de horários de forma manual, facilitar a entrada e saída de dados (de forma personalizada) e a criação de regras para a geração desta grade, economizando-se assim tempo e recursos para tal atividade. Após um levantamento de informação sobre heurística, gerenciamento de projetos e a real necessidade dos usuários serão formulados um protótipo e criação do software, tomando como piloto o Campus V do CEFET-MG. Espera-se como resultado desse projeto que o aluno possa desenvolver seus conhecimentos adquiridos no curso, ampliando sua visão de aplicação do curso e a criação de uma nova ferramenta para geração e controle de horários das aulas, de maneira simplificada e otimizada, que atenda às especificidades do CEFET-MG.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2) . Integrantes: Emerson de Sousa Costa - Coordenador / Tiago Alves de Oliviera - Integrante.

2019 - 2020
Desenvolvimento De Uma Aplicação Web Para Gerência E Controle De Reservas De Recursos
Descrição: A gerência de projetos é um tema de pesquisa muito importante e aplicado em diversas áreas, principalmente no desenvolvimento de produtos e gerência de empresas pública ou privada. Uma das vertentes da área gerência de projetos é a gerência de recursos que trata de assuntos que inclui os processos para identificar, adquirir e gerenciar os recursos necessários para a conclusão bem-sucedida de um projeto ou gestão. Uma das formas de gerenciar os recursos nas instituições de ensino é fazer isto da maneira tradicional (manualmente utilizando-se de papeis) ou através de sistemas (softwares) para realizar essa alocação de serviços. Sendo assim esse projeto visa a especificação, o desenvolvimento e a entrega de uma Aplicação Web para realizar a gerência dos recursos disponíveis em uma unidade do CEFET-MG, sendo essa aplicação de acesso público para a comunidade interna e externa, disponibilizando dados históricos para melhorias de gerência de alocação, fornecendo um ambiente restrito para as reservas e com isso tornando-se um ambiente único de reservas e visualização das mesmas. Ressalta-se também que o sistema contribuirá para a economia de impressão das reservas utilizadas de forma manual. Além disso, este projeto propiciará o desenvolvimento de dois discentes nas áreas de gerência de projetos, programação para web e criação de aplicativos, a partir do estudo e aplicação de conceitos e teorias desenvolvidas nos cursos de graduação da área de Computação do CEFET-MG.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2) . Integrantes: Emerson de Sousa Costa - Coordenador / Tiago Alves de Oliviera - Integrante.

Modelagem de protótipos para sistemas reais
Descrição: A modelagem matemática está presente em muitas aplicações, pois permite o estudo de diversas situações, sem que estes precisem ser desenvolvidos em situações reais. Esse fato faz com que os custos com análises de situações reais sejam reduzidos a custos de análise dos modelos matemáticos obtidos. Neste projeto é proposto um conjunto de atividades para que o bolsista obtenha modelos de baixa ordem para protótipos desenvolvidos no Laboratório de Sinais e Sistemas (LSS) do CEFET-MG ? campus Divinópolis.  A obtenção de modelos de baixa ordem é fundamental para a aplicação de técnicas de sintonia de controladores, sobretudo em processos industriais.  Para a construção dos modelos matemáticos, o bolsista deverá estudar métodos determinísticos clássicos para construção de modelos de primeira e segunda ordem, com atraso, a partir de respostas do sistema a uma entrada em degrau. O bolsista ficará responsável pelo planejamento e realização de experimentos para a obtenção de dados, tanto para a construção dos modelos quanto para a validação dos modelos obtidos. Espera-se que o bolsista desperte o interesse pela área de engenharia e também que os conceitos estudados na disciplina de matemática sejam aperfeiçoados. Esses experimentos serão realizados utilizando linguagem python com aplicativos e interfaces desenvolvidas em projetos de iniciação científica de alunos da Engenharia Mecatrônica, disponíveis no Laboratório de Sinais e Sistemas (LSS) do CEFET-MG ? campus Divinópolis.. 2018 - 2019
Modelagem Matemática e Engenharia
Descrição: Diversos problemas de engenharia são regidos por equações diferenciais ou sistemas de equações diferenciais que são matematicamente equivalentes. A solução analítica destes problemas é difícil ou impossível de ser obtida, requerendo em geral que seja realizada sua modelagem matemática com o objetivo de se encontrar uma solução aproximada, que pode ser obtida por diversos métodos numéricos. Há muitas maneiras pelas quais dispositivos e comportamentos podem ser descritos. Podemos usar palavras, desenhos ou esboços, modelos físicos, programas de computador ou fórmulas matemáticas. Em outras palavras, a atividade de modelagem pode ser feita em vários idiomas, muitas vezes simultaneamente. Nesse intuito, esse trabalho visa fazer um estudo de diversos problemas de engenharia, modelá-los a partir de equações diferenciais e obter a solução numérica dos mesmos utilizando algoritmos implementados pelo bolsista. A aplicação da teoria estudada, principalmente o tópico de equações diferenciais, com a parte de simulação numérica auxiliará no despertar do bolsista para que ele se interesse por essas áreas do conhecimento. O acompanhamento do desenvolvimento do projeto será feito a partir de reuniões quinzenais entre o orientador e o aluno bolsista a fim de manter um contato direto para que ele não se sinta desestimulado. Espera-se com isso que o aluno bolsista tenha uma forte base matemática e também um ótimo conhecimento de programação.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (1) . Integrantes: Emerson de Sousa Costa - Coordenador.
