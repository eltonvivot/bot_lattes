1982 - 1984
Sistema de Automação de Documentos com Utilização de Dispositivos Gerais (COMPO)
Descrição: Este projeto consistiu no desenvolvimento de um sistema de programas que permitissem a fácil produção de documentos em que o elemento Desenho era de primordial importância. Isto envolveu o estudo e implementação de um núcleo gráfico, que permitisse manipular os dispositivos gráficos, bem como na elaboração de primitivas gráficas como retas, polígonos, pontos e arcos de círculo. Para automatizar mais a geração dos desenhos foi desenvolvida uma linguagem para a geração dos desenhos (LGE - Linguagem de Geração de Elementos Gráficos).
Tratava-se quanto à implementação, de um sistema do tipo "prancheta eletrônica", que abreviadamente foi denominado COMPO, e foi provavelmente o primeiro Editor Gráfico desenvolvido no país. Este sistema baseou-se em um computador SISCO MB8000, um traçador gráfico HP e um "tablet" de funções, desenvolvido pela FDTE (Fundação para o Desenvolvimento Tecnológico da Engenharia). O sistema COMPO permitia a edição de desenhos interativamente, utilizando uma série de funções gráficas e gabaritos gerados através do próprio COMPO ou da linguagem LGE. Os desenhos gerados podem ser utilizados como documentação ou como entrada para aplicações de CAD/CAM, como por exemplo um projeto de circuito impresso, a partir do diagrama lógico do circuito.. Descrição: A linguagem LGE - Linguagem para Geração de Elementos Gráficos - permite a geração de desenhos através do uso de uma linguagem de alto nível.
