2016 - 2017
Melhorias na estruturação e obtenção de informações climáticas de uma estação meteorológica de baixo custo para produtores rurais através da aplicação de filtros no banco de dados meteorológico
Descrição: As informações são fundamentais para o gerenciamento efetivo e para aumentar competitividade de
qualquer atividade econômica. No segmento ligado à produção agrícola percebe-se o incremento de uso de tecnologias para gestão e aprimoramento de processo de produção. Neste sentido, o presente projeto de pesquisa tem como objetivo geral o aprimoramento, estruturação e disponibilização de informações históricas obtidas de uma estação meteorológica de baixo custo desenvolvida pelo Grupo de Desenvolvimento em Mecânica - GDMEC do IFRS Campus Erechim. Para isso, serão disponibilizadas funcionalidades de filtros de dados no banco de dados meteorológico, utilizando como base as ferramentas do Google Chart em conjunto com as linguagens de programação PHP e Javascript. Dessa maneira espera-se otimizar o funcionamento da estação meteorológica com a disponibilização de informações gerenciais relevantes para o produtor rural e que podem ser obtidas a partir da base de dados já existente.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (1) . Integrantes: Ernani Gottardo - Coordenador / Alisson Dalsasso Corrêa de Souza - Integrante / Vinícius Michelin - Integrante.
