ADA
Descrição: Este projeto de pesquisa e desenvolvimento tem o título de ADA que é um acrônimo para Análise de Dados Aplicados. É também o nome da primeira pessoa a criar um algoritmo de computador, a Ada Lovelace. A Análise de Dados ou Ciência de Dados é um termo utilizado hoje para descrever conjuntos de métodos, técnicas e ferramentas para lidar com dados. Pode-se afirmar que esta área executa as tarefas de adquirir, selecionar, pré-processar, transformar, minerar, interpretar, visualizar e avaliar dados e suas implicações de uso. Esses dados podem vir de grandes volumes de dados como todo o volume de transações comerciais de uma grande corporação ou dados extraídos de uma única imagem de Raios X, por exemplo. Este projeto irá utilizar bases de dados existentes de saúde, agropecuária, meio ambiente, comércio, entre outras, ou criar bases de dados via coleta por sensores diversos como os utilizados em Cidades Inteligentes ou extraídos de análise de sinais como imagens, vídeos e áudios. O objetivo é coletar e armazenar, e analisar dados de fontes diversas para visualização e tomada de decisão, principalmente, com apoio da Inteligência Artificial.. 2017 - 2020
Metodologia Computacional para Correlacionar Exames Médicos de Diagnóstico por Imagem
Descrição: Segundo o INCA, câncer é o nome dado a um conjunto de mais de cem doenças que têm em comum o crescimento desordenado de células, que invadem tecidos e órgãos podendo levar ao óbito. O prognóstico positivo da maioria dessas doenças está relacionado ao diagnóstico precoce. Um dos cânceres com melhor prognóstico é o câncer de mama desde que tratado nas fases iniciais da doença. Existem diversos exames, a maioria por imagem, para detecção desse tipo de câncer, mas nenhum tem resultado unânime, vista que cada equipamento usa um tipo de sensor e captura dados diferentes dos pacientes como por exemplo mamografia, ultrassom, ressonância magnética e termografia. A Termografia ainda não é amplamente utilizada em centros de saúde por não existir um padrão de análise do exame. A mamografia é o exame considerado pelos médicos como "padrão ouro". No Sistema Único de Saúde o laudo das mamografias pode ser realizado, como é feito no Hospital Universitário Antonio Pedro da Universidade Federal Fluminense (Niterói, RJ), pelo programa SISMAMA que foi desenvolvido pelo governo federal, mas não tem recebido suporte de sistema. Além disso, ele utiliza uma classificação ultrapassada. Existe uma sistematização internacional para a avaliação mamária, o Breast Imaging Reporting and Data System (BIRADs). O SISMAMA utiliza o BIRADs versão 2 e atualmente está disponível a versão 6. A maioria dos exames por imagem é analisada visualmente pelos médicos, mas há sistemas computacionais que permitem melhorar o resultado final para uma tomada de decisão mais precisa. Desta forma, com este projeto pretende-se desenvolver um sistema de laudo de mamografia utilizando o BIRADs mais atual. Além disso combinar exames de mama, inicialmente mamografia e termografia para propor uma metodologia de diagnóstico mais acurada.. Situação: Concluído; Natureza: Pesquisa. 2010 - 2013
Processamento e análise de imagens aplicadas à mastologia
Descrição: A proposta desta pesquisa busca auxiliar no diagnóstico por imagens médicas; procurando integrar as principais etapas do processamento de imagens aos métodos de extração de características e conhecimento, visando: a interpretação das imagens; a identificação de tecidos, estruturas anatômicas e funcionais, no sentido de conduzir pesquisas direcionadas ao problema de auxiliar nos diagnósticos precoces e na classificação de patologias (benignas/malignas) da mama. O projeto vai desde a aquisição das imagens e sua organização em bancos de dados, passando pela suas análises através de métodos numéricos, até um último estágio, onde técnicas de aprendizado de máquina devem ser utilizadas de forma a ser feita uma extração de conhecimento das Imagens de Mamas.
