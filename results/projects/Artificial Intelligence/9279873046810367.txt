2017 - Atual
Análise e comparação de desempenho de container em Linux com maquinas virtuais para construção de nuvens computacionais
Descrição: A computação em nuvem fornece a ?virtualização de recursos e serviços de TI? usando a Internet como meio para fornecer ou compartilhar esses recursos. Por meio dela, é possível ter um balanceamento de cargas de serviços em tempo real, e o usuário paga somente o quanto ele usar. Nesse contexto, .tem-se os modelos de virtualização baseados em containers, que diferenciam-se do modelo tradicional por utilizar o conceito de contêiner em cima do um único Sistema Operacional que é carregado uma única vez, ao contrario do tradicional que para cada maquina se tem um sistema operacional. Esse conceito reduz de forma representativa o tempo de deploy de alguma infraestrutura ou até mesmo aplicação, pois não há necessidade de ajustes de ambiente para o funcionamento do serviço, visto que o ambiente é sempre o mesmo.. Situação: Em andamento; Natureza: Pesquisa. Integrantes: Allan Gonçalves de Oliveira - Coordenador.
2017 - Atual
ESTUDO E DESENVOLVIMENTO DE FERRAMENTAS PARA MANIPULAÇÃO DE DADOS AMBIENTAIS
Descrição: O meio ambiente sofre constantes mudanças, sejam elas naturais ou por ações antropogênicas. Dessa forma, faz-se necessário o entendimento da magnitude das intervenções humanas e o quão prejudicial isso pode ser para a natureza e para o próprio homem. Há diversas evidências que apontam para o home como maior causador das mudanças que são possíveis de se observar nos últimos anos e, com isso, surge a necessidade de se mensurar e controlar essas ações. Assim, esse projeto visa o desenvolvimento de técnicas computacionais para auxiliar o processo de gerenciamento ambiental utilizando dados de meteorologia e de bioacústica do Pantanal Mato-Grossense.. 2015 - Atual
MiMi: Plataforma computacional para mineração de dados micrometeorológicos
Descrição: A evolução da capacidade de aquisição de dados em equipamentos eletrônicos nos últimos anos impõem desafios ao gerenciamento e análise dos dados coletados em diversas áreas do conhecimento. Pesquisas envolvendo dados micrometeorológicos têm se beneficiado desses avanços e também vivenciado as dificuldades em relação às analise dos dados. Diante disso, o uso de técnicas computacionais para auxiliar o processo de análise dos dados é uma necessidade para usuários desse tipo de dado. Assim, o objetivo desse trabalho é desenvolver uma plataforma computacional para auxiliar o processo de mineração de dados micrometeorológicos. A plataforma desenvolvida tem o intuito de apresentar uma arquitetura de software genérica e flexível. Apresenta-se o conceito de tratar as séries temporais como um novo tipo de dado dentro da plataforma desenvolvida e os algoritmos que manipulam esse novo tipo de dado foram considerados operadores do mesmo. Com a definição de operando e operadores é possível definir a execução de expressões de domínio que representam um fluxo de processamento específico para cada atividade de mineração de dados em um domínio de séries temporais. Para isso, será definido um gerenciador de domínio que executa vários operadores em uma ordem pré-definida. A plataforma MiMi será validada com a execução de atividades de mineração de dados utilizando bases de dados micrometeorológicos do estado de Mato Grosso, Brasil.. Situação: Em andamento; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2) .
