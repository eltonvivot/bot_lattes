RPDBCS, Reconhecimento de Padrões de Defeitos em Sistemas de Bombeio Centrífugo Submerso
Descrição: Este projeto visa utilizar técnicas de inteligência computacional para a identificação de padrões de defeitos em sistemas de bombeio centrífugo submerso (BCS) durante a fase de teste e aceitação desse sistema. O projeto desenvolverá uma metodologia computacional para analisar dados característicos de vibrações de uma bomba centrífuga submersa, coletados em campo durante a fase de teste e aceitação do equipamento, e inferirá automaticamente se esse equipamento se encontra em condições adequadas para ser colocado em operação.. Situação: Em andamento; Natureza: Pesquisa. Integrantes: Thomas Walter Rauber - Integrante / Flávio Miguel Varejão - Coordenador / Fábio Fabris - Integrante / Francisco de Assis Boldt - Integrante.

2003 - 2005
Reconhecimento de Padrões por Equilíbrio de Newton
Descrição: Resumo. Este projeto propõe uma metodologia para casar modelos com dados em imagens digitais. Os modelos      podem ter sofrido transformações de corpo rígido. Os modelos são iterativamente adaptados aos dados da imagem, usando translações e rotações, buscando um estado de equilíbrio de Newton, assim aplicando conceitos da mecânica de Newton a processamento de imagens. As técnicas propostas serão aplicadas em duas e três dimensões. Testes preliminares sugerem uma potencialidade excelente do método de resolver um problema genérico de pesquisa na área de processamento de imagens.
Palavras-Chave. Reconhecimento de Padrões, Processamento de Imagens, Casamento de Padrões baseado em analogias físicas, Corpo Rígido, Equilíbrio de Newton. Situação: Desativado; Natureza: Pesquisa. 2001 - 2002
Método Quantitativo para a Análise da Microvascularidade Retiniana usando Processamento de Imagens
Descrição: Resumo
O objetivo do projeto é a realização de um sistema de processamento de informação visual da retina humana a fim de detectar patologias que se desenvolvem ao longo do tempo. A peça chave do sistema é um software semi-automático capaz de analisar imagens do fundo do olho humano. Deve fornecer medição quantitativas da espessura dos vasos sangüíneos a fim de permitir comparações temporais.
Palavras-Chave: Processamento de Imagens Medicinais, Reconhecimento de Padrões, Retinopatia, Método quatitativo, Microvascularidade retiniana. Situação: Concluído; Natureza: Pesquisa.
