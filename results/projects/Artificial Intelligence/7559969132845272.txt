2016 - 2018
APLICATIVO ANDROID PARA GERENCIAMENTO DE REDES DE SENSORES SEM FIO
Descrição: Esse projeto lida com o desenvolvimento de um aplicativo Android capaz de
gerenciar uma rede de sensores sem fio (RSSF) de forma segura e eficaz. Com objetivo
de se conseguir criar uma plataforma adaptativa para diversas redes, não precisando
realizar diversos ajustes para se adequar a realidade da rede. Todo o desenvolvimento
do sistema é constituído por várias etapas, assim como esse projeto. Inicialmente há um
estudo de caso, ou seja, como deveria ser feito um gerenciamento de um rede de sensores
(usabilidade, segurança e armazenamento), logo o software é modelado e seu banco de
dados em seguida. Depois temos todo o processo de desenvolvimento bruto através da
linguagem de programação de toda a plataforma mobile. Finalmente esse projeto é
finalizado com testes e escrita de artigos. Todas essas etapas fazem parte de como será
gerido e desenvolvido o projeto. Situação: Concluído; Natureza: Pesquisa. Integrantes: Carlos Maurício Jaborandy de Mattos Dourado Júnior - Coordenador.

Estudo e Aplicações de Gerenciamento das Redes de Sensored sem fio com Sistema android de modo seguro
Descrição: Esse projeto visa o estudo de conceitos aplicados a conectividades de sistemas Android com redes de sensores sem fio, tais como bluetooth, wifi, wifi p2p e nfc. Essas trabalhando em conjunto na coleta, envio e controle das informações relacionadas a rede, para fornecer maior segurança, facilidades de gerenciamento e usabilidade. Com objetivo de planejar e estruturar todo uma plataforma Android configurável que trabalhe com qualquer rede de sensores sem fio. Todo o projeto será realizado através de etapas, inicialmente tendo todo o estudo de caso sobre redes de sensores sem fio, diagramação UML, programação, banco de dados e padrões de engenharia de software. Ao final do projeto o aluno será capaz de analisar e ser capaz de planejar, estruturar e fornecer todo uma documentação de como ser desenvolvido uma plataforma mobile que gerencia redes de sensores sem fio.. Sistema Web Para Monitoramento E Controle De Sistemas Embarcados Autônomos
Descrição: Atualmente a tecnologia está presente constantemente na nossa vida, como smartphones, carros, televisão, geladeira, entre outras. Então seria necessária uma plataforma geral para cada usuário, pois se cada sistema tiver sua plataforma o usuário teria que lidar normalmente com várias e não seria viável. Assim nosso trabalho se baseia na criação de programas para o controle dos sistemas automatizados e essa será acessada através da Internet. 2005 - 2007
Comunicação Gestual à Distância com Humanóides Virtuais: Aplicações na Educação
Descrição: Os gestos corporais constituem um meio de comunicação humana de grande importância. Este projeto é focalizado no problema da comunicação gestual em cenários de interação em que esta modalidade de expressão possui algum tipo de impedimento ou restrição. Os ambientes de trabalho colaborativo à distância e os de tele-ensinamento são exemplos de situações nas quais essas restrições apresentam-se como conseqüências naturais do afastamento geográfico entre os participantes. Alguns ambientes de colaboração à distância baseados em técnicas de transmissão de vídeo viabilizam a comunicação gestual, mas com difícil integração visual dos diversos componentes do ambiente, não oferecendo uma interface coesa e fazendo freqüentemente uso de múltiplas janelas (uma para cada usuário participante e uma para cada aplicação compartilhada). Sob uma diferente abordagem, ambientes baseados em técnicas de realidade virtual permitem que cada usuário seja representado dentro de um espaço 3D por objetos humanóides, permitindo aos participantes perceberem-se mutuamente enquanto interagem com os elementos existentes no espaço virtual comum. A integração de expressões e gestos corporais aos avatares permite o aumento da qualidade da comunicação nestes ambientes, contribuindo de maneira significativa para a redução dos problemas perceptivos causados pela distância física entre os usuários. Para realizar a comunicação gestual, os humanóides virtuais são animados, freqüentemente através das rotações de suas articulações. Tais animações, predefinidas ou calculadas em tempo real, são geralmente associadas a eventos ocorrentes nos ambientes de colaboração (um clique sobre um botão, um deslocamento no espaço virtual, uma colisão entre objetos, etc.). Para uma comunicação gestual mais completa, técnicas de aquisição de gestos humanos reais e restituição sobre atores virtuais podem ser empregadas. A aquisição de parâmetros necessários às animações pode ser realizada por intermédio de dispositivos.
Nesse projeto trabalhei na parte da comunicação multimídia e desenvolvimento de recursos com o uso do multicast.. Situação: Concluído; Natureza: Pesquisa.
