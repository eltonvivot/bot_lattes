Abordagens Heurísticas e Exatas para Problemas de Roteamento e Alocação
Descrição: O projeto propõe o estudo e resolução de problemas de otimização na área de roteamento de veículos e escalonamento de profissisonais. Decidiu-se estudar estes dois grupos de problemas pois eles têm diversas abordagens em comum, visto que ambos podem ser vistos como problemas de sequenciamento, um grupo
mais geral de problemas de otimização. Além disso, o grupo de otimização está tendo demanda de aplicações práticas para resolução de variantes destes problemas, o que motivou a convergir esforços para esta classe de problemas. Projetos anteriores do grupo tratava da resolução de quatro problemas de otimização completamente diferentes um do outro. A experiência obtida nos últimos anos foi muito importante, e poder convergir para problemas que contêm semelhanças possivelmente vai permitir uma concentração maior em resolver tais problemas.
O objetivo principal deste projeto é reproduzir e aperfeiçoar o estado-da-arte na resolução de variantes de problemas de roteamento e alocação. Duas ordens de grandezas serão almejadas: uma para resolução exata, e outra heurística. Cada problema é inicialmente contextualizado, modelado matematicamente, e então inicia-se o processo de resolução exata ou heurística.. Situação: Em andamento; Natureza: Pesquisa. Programação Dinâmica Eficiente com Algoritmos Cache Oblivious (Processo CNPq  551457/2009-0 - Edital MCT/CNPq nº 70/2008 - Mestrado/Doutorado)
Descrição: O presente projeto está caracterizado como pesquisa científica, e visa o desenvolvimento de pesquisa dentro da área de complexidade de algoritmos, uma área fundamental da ciência da computação. Um problema pode ser resolvido por diversos algoritmos, e dependendo do problema e dos dados, algumas técnicas são mais indicadas. Recentemente técnicas foram criadas para o projeto de algoritmos aplicados a dados massivos (grandes quantidades de dados). Estes algoritmos são classificados em três categorias: algoritmos de stream de dados, algoritmos de memória secundária e algoritmos cache-eficientes. Os algoritmos cache eficientes, objeto de estudo deste projeto, objetivam fazer melhor uso da memória cache. Ou seja, o algoritmo deve ser projetado de forma a minimizar o número de acessos à memória cache. Para tanto, estes algoritmos exploram a localidade dos dados na cache. Fazendo uso do modelo de algoritmos cache-eficientes, este projeto propõe desenvolver um algoritmo cache eficiente para resolução do problema da mochila ilimitado. Este é um problema NP-Completo. A escolha deste problema é devido ao fato que o mesmo é resolvido como subproblema num método de geração de colunas para resolução do problema de empacotamento unidimensional, também NP-Completo.. Situação: Em andamento; Natureza: Pesquisa. Integrantes: Luciana Salete Buriol - Coordenador / Marcus Ritt - Integrante / Felix Rodrigues - Integrante.Financiador(es): Conselho Nacional de Desenvolvimento Científico e Tecnológico - Bolsa.

2007 - 2009
Recuperação de Informações Multilíngües
Descrição: O objetivo deste projeto de pesquisa é contribuir para o desenvolvimento da Recuperação de Informações Multilíngües envolvendo a língua Portuguesa. A motivação vem da crescente necessidade que experienciamos atualmente de explorar documentos em outros idiomas. Com a disseminação da Internet, a distância física entre o usuário e a informação foi removida. Contudo, a barreira da linguagem ainda precisa ser ultrapassada. A pesquisa nesta área tem se desenvolvido rapidamente desde o final dos anos 90. Apesar dos recentes avanços, ainda existem muitos aspectos não explorados, em especial sobre a utilização da língua portuguesa. Espera-se como resultado deste projeto gerar um sistema de recuperação de informações que aceite consultas na língua portuguesa e busque documentos em inglês. Além disso, algumas questões atuais relativas a estes sistemas serão investigadas: (i) desenvolvimento de algoritmos de stemming para português; (ii) extração de técnicas de mapeamento de conceitos entre idiomas por meio da utilização de corpus comparável e paralelo; (iii) estudo do processo de realimentação de relevantes em um contexto de sistemas multilíngües e (iv) desenvolvimento de técnicas para identificação de termos compostos.. ALGOWEB: Algoritmos para Geração, Análise e Cálculo de Propriedades do GrafoWeb
Descrição: Um grafoweb é originado da estrutura de links das páginas web: cada página é um nó, e cada hyperlink entre páginas é um arco direcionado neste grafo. Este grafo apresenta duas características principais que o identificam como uma rede complexa: é um grafo de grande dimensão (bilhões de nós) e não possui estrutura específica de expansão. Como objetivos específicos, pode-se citar: - propor um modelo para geração sintética de grafos web, levando em consideração a evolução temporal deste - criação de bases atuais de dados temporal com dados da Wikipédia, e extração do grafoweb das páginas web do Brasil - algoritmos de data stream para o cálculo de propriedades do grafo - algoritmos de memória secundária para o cálculo de propriedades do grafo - algoritmo de classificação (\it ranking) levando em consideração dados temporais. Finalmente, outro objetivo do projeto é determinar problemas de dimensões variadas. Situação: Em andamento; Natureza: Pesquisa. Integrantes: Luciana Salete Buriol - Coordenador / Viviane Moreira Orengo - Integrante / Marcus Ritt - Integrante.
