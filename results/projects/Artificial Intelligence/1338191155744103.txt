Towards Self-Driving In-Band Network Telemetry Orchestration in Programmable Data Plane (FAPERGS - ARD 2019)
Descrição: In-band network telemetry is an emerging network monitoring paradigm. By collecting low-level telemetry items in real time, in-band telemetry is able to substantially enhance network-wide visibility - allowing, for example, timely detection problems such as micro-burst. Recent studies have focused on textit{(i)} developing mechanisms to increase network-wide visibility; and textit{(ii)} to design new monitoring solutions. However, little has been done to coordinate the process of collecting telemetry items in this new paradigm. This is particularly challenging for two main reasons. First, depending on which network telemetry items are collected, it might degrade network-wide visibility in terms of consistency and freshness. Second, depending on how network telemetry is collected, it might impact network performance. In this project, we investigate and model the In-band Network Telemetry Planning Problem - the problem of planning the telemetry collection process. The main idea consists of dynamic exploring the interplay between monitoring application performance metrics (textit{e.g.}, accuracy and freshness) and the in-band telemetry data acquisition. Subsequently, we intend to make the proposed operational solutions in a network structure with programmability support.. Situação: Em andamento; Natureza: Pesquisa. 2019 - Atual
Exploração de Tiles na Codificação de vídeo 3D no Padrão 3D-HEVC
Descrição: Este projeto investiga o uso de tiles para acelerar a codificação no padrão 3D-High Efficiency Video Coding (3D-HEVC). A ideia do projeto é analisar o impacto na eficiência de codificação e com vários níveis de aceleração em diferentes particionamentos de tiles. A partir dessa análise, serão propostas soluções para minimizar as perdas de codificação, maximizar os ganhos de aceleração de forma adaptativa, controlar a complexidade da codificação de diferentes regiões do quadro e controlar o aumento de temperatura em regiões de um chip que esteja executando um codificador 3D. Cabe salientar que exceto por trabalhos do coordenador do projeto, não existe outros trabalhos publicados na literatura explorando tiles para a codificação de vídeos 3D no padrão 3D-HEVC.. Situação: Em andamento; Natureza: Pesquisa. Integrantes: Fábio Diniz Rossi - Integrante / Rumenigue Hohemberger - Integrante / Marcos Paulo Konzen - Integrante / Gustavo Freitas Sanchez - Coordenador.
2018 - Atual
Método Automático, Transparente e Dinâmico para melhorar a eficiência energética de aplicações paralelas
Descrição: Os processadores de propósito geral (encontrados nos computadores de mesa) estão tendo seu aumento de desempenho limitado pela energia térmica do projeto (também chamado de TDP - Thermal Design Power). E, finalmente, a maioria dos sistemas embarcados são dispositivos móveis e mantidos ligados através de uma bateria. Portanto, o grande desafio tem sido não apenas aumentar o poder de processamento destes sistemas e o desempenho das aplicações paralelas, mas também fazer isso com o mínimo impacto no consumo de energia. O uso eficiente de recursos, desta maneira, é de extrema importância. Se uma aplicação executa com poucas threads, recursos de hardware podem ficar ociosos (memórias, núcleos e barramento de comunicação). Em contrapartida, esses mesmos recursos podem saturar devido ao alto número de threads que precisam ser executadas em paralelo. Em ambas as situações, a execução será ineficiente, o que pode resultar em lentidão ou consumo excessivo de energia. Desta maneira, nem sempre executar o maior número de threads possível em um sistema pode ser a melhor solução, mesmo que este seja o pensamento comum. Considerando este cenário, escolher o número correto de threads a ser usado no desenvolvimento de aplicações paralelas pode fornecer melhorias significativas no desempenho e economia de energia.. Situação: Em andamento; Natureza: Pesquisa. Integrantes: Fábio Diniz Rossi - Integrante / LORENZON, ARTHUR FRANCISCO - Integrante / Antonio Carlos Schneider Beck Filho - Coordenador.

Uma Abordagem Automática, Dinâmica e Transparente para Otimizar o Desempenho e Consumo de Energia de Aplicações Paralelas em Sistemas de Propósito Geral e Embarcados
Descrição: Este projeto propõe a implementação de uma abordagem automática e transparente para melhorar aplicações paralelas implementadas com OpenMP considerando diferentes métricas: desempenho, energia e energy-delay product (EDP). Ele fará isso pelo ajuste do número de threads, a frequência do processador (através do DVFS), e o número de núcleos ativos (através do power gating), sem a necessidade de qualquer modificação no código fonte, recompilação de código ou interação do usuário final.. Situação: Em andamento; Natureza: Pesquisa. Integrantes: Fábio Diniz Rossi - Integrante / LORENZON, ARTHUR FRANCISCO - Coordenador / Marcelo Caggiani Luizelli - Integrante.

2017 - 2018
Alocação de recursos em plataformas embarcadas
Descrição: Embora a Internet das Coisas esteja mais e mais presente em nosso dia a dia devido à baixa no preço dos componentes eletrônicos nos últimos anos, tal tecnologia ainda precisa de amadurecimento. Dispositivos embarcados ainda sofrem com desafios historicamente estabelecidos, tais como consumo de energia. Os sistemas de comunicação de dados, em sua maioria, são adaptados de outras realidades para esse novo cenário. Dispositivos de borda devem mudar seu paradigma cliente-servidor para suportar orientação a serviços. Portanto, vários desafios ainda se apresentam como oportunidades de pesquisa. O objetivo desse projeto consiste em implementar soluções para ambientes de IoT com foco na otimização de dispositivos embarcados de borda, através do uso de alocação de recursos e virtualização. Para tanto, serão realizados experimentos visando entender os overheads impostos pelas aplicações sobre tais dispositivos, e propor soluções que enderecem tais limitações.. Escalabilidade em Máquinas Virtuais
Descrição: Este projeto investiga a escalabilidade de máquinas virtuais (Xen) em ambientes com diversos processadores (UMA e NUMA). A estratégia inicial é executar um conjunto de benchmarks para fazer a avaliação de diferentes configurações, por exemplo, diferente números de máquinas virtuais, diferente número de processadores, ... Para analisar os resultados dos benchmarks, é importante organizar os dados de uma maneira que possam ser utilizados algoritmos de datamining. assim um modelo analítico dos resultados dos benchmarks está sendo elaborado. Outro aspecto diz respeito ao escalonamento de máquinas virtuais pelo Xen. Escalonamento dinâmico está sendo investigado como uma possível solução para o Xen..
