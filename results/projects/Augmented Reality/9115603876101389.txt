2006 - 2009
Desenvolvimento de um equipamento automático de pequeno porte para diagnóstico utilizando o método ELISA
Descrição: O presente projeto trata do desenvolvimento de uma solução para automação laboratorial, especificamente ensaios imunológicos para diagnóstico laboratorial. O projeto foca principalmente o equipamento automatizado para realização dos diagnósticos imunológicos, não deixando de lado a arquitetura de software, essencial em qualquer boa plataforma de automação. O resultado buscado é a implementação de uma solução em automação para laboratórios de análises clínicas - imunologia (usuários de técnicas imunoenzimáticas) de forma a simplificar a rotina laboratorial, ter controle total sobre os parâmetros dos ensaios, permitir que pequenos laboratórios possam realizar ensaios de maior precisão e maior valor agregado em sua própria instalação, reduzindo também o tempo de entrega dos resultados. Ao final do projeto é esperado o desenvolvimento de uma plataforma robótica (12 a 15 eixos) aliada a um sensor colorimétrico (espectrofotômetro) que realizará de forma automatizada todas as etapas para realização do ensaio, bem como uma interface que permita a inserção do equipamento na rotina laboratorial (análises clínicas).
O projeto foi aprovado junto a FAPEMIG (101675/05) com auxílo do CNPQ (processo 554826/05-4 publicado no DOU de 01/12/05, página 12, seção 3, contando ainda com o apoio tecnológico, no que tange especificamente a área de imunoensaios, da Fundação Ezequiel Dias - FUNED, através de convênio 002/06.. 2006 - 2009
Desenvolvimento de um equipamento automático para diagnóstico utilizando bioquímica clínica
Descrição: O presente projeto trata do desenvolvimento de uma solução laboratorial para análises clínicas, especificamente ensaios bioquímicos. O projeto foca a especificação, implementação e testes de um equipamento para diagnóticos. Será composto de um sistema robótico e software de controle. Deverão ser especificados e validados os processos de fabricação necessários para a fabricação do mesmo, conforme edital MCT/FINEP/SEBRAE/Ação Transversal Cooperação ICT´s MPE´s..
