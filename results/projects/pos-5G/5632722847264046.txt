MAS-Fog
Descrição: The cloud computing model plays a central role providing on-demand processing and storage to online services. The model works very well for digital services, but for some applications low latency communication and sensitive response time are crucial. In this case the centralization of computing and storage capabilities promoted by the cloud model could lead to worse quality of service (QoS). To face some of these issues the fog computing model seeks to spread the cloud capabilities to the network edge. In this context, this project applies a trust and reputation (T&R) model based on the node selection approach to the fog computing paradigm. A multiagent system for coordinating devices that work as fog nodes and end devices is applied to consume services from the fog.. Situação: Concluído; Natureza: Pesquisa. BRA2Cloud
Descrição: Cloud computing has been used as the primary IT solution alternative to provide scalable, reliable and pay-per-use resources and services on-demand, including trusted servers and transient servers.The implementation of fault-tolerant approaches that ensure the reliability and the  services  continuity  becomes  a  major  requirement  for  users  that  want  to effectively explore cheaper transient servers.  This project investigate and propose a multi-agent framework to provide a transparent and resilient environment on transient servers by using multiple fault-tolerant strategies called BRA2Cloud (Brand new Resilient Agent-based Architecture for Cloud computing). The proposed framework uses a reasoning process to predict failure events helping to adjust fault tolerance plans and  parameters. The  framework  includes  elements  to  recognize  the  transient server environment, being useful to identify risks to the current server?s availability, achieving better levels of reliability reducing the total cost and execution time.. LCMAP
Descrição: Multi-agent planning is the agents? coordination process which is a computacionally hard problem. Many works focus on the planning strategy considering the computational process, agents? distribution roles, information privacy, and the resources coupling level. But domain-independent models that explore the balance between coordination process and privacy leading to an efficient execution solution are missing. This project investigates the development of an efficient and domain-independent multi-agent planning model called LCMAP (Lightweight Coordination Multi-Agent Planning) to balance coordination process and privacy through three independent phases: verification, transformation and validation. LCMAP is compared to the state-of-the-art models to evaluate time efficiency and plan length during the problem solving process using loosely and tightly coupled domains with specific evaluation metrics inherited from planning competions. Regarding real world-problems, LCMAP is evaluated as a solution for teamwork management presenting suitability for most of the phases: potential recognition, team formation, planning and team action.. MAS-Cloud
Descrição: Dynamic provisioning of computational resources in cloud platforms is a challenging problem. In this scenario, multiagent system technology offers improvements using intelligent agents to dynamically choose the best computational load distribution reducing execution time and cloud services cost. This project presents MAS-Cloud, a multiagent system to dynamically monitor, predict and provide computational resources in cloud platforms. Using deductive reasoning agents to cooperatively act in a three layer architecture, it is provided transparent horizontal elasticity of virtual machines in public
cloud platforms (i.e., Google, Amazon EC2). MAS-Cloud was evaluated with an agent-based simulator called MASE-BDI that is a nondeterministic and CPU intensive application providing a challenging validation case. A multiple linear regression model was
used presenting good predictions with reduced mean absolute percent error (MAPE) of 3.59% CPU usage and 5.28% of execution time in the Google platform, and similar results in the Amazon EC2 platform.. Descrição: As técnicas tradicionais de recomendação não consideram as preferências do usuário de acordo com o contexto. 2010 - 2012
AGMI: An AGent-MIning tool  to Identify Cartel Formation
Descrição: The main focus of this research project is the problem of extracting useful information from the Brazilian federal procurement process databases used by government auditors in the process of corruption detection and prevention to identify cartel formation among applicants. Extracting useful information to enhance cartel detection is a complex problem from many perspectives due to the large volume of data used to correlate information and the dynamic and diversified strategies companies use to hide their fraudulent operations. To attack the problem of data volume, we have used two data mining model functions, clustering and association rules, and a multi-agent approach to address the dynamic strategies of companies that are involved in cartel formation. To integrate both solutions, we have developed AGMI, an agent-mining tool that was validated using real data from the Brazilian Office of the Comptroller General, an institution of government auditing, where several measures are currently used to prevent and fight corruption. Our approach resulted in explicit knowledge discovery because AGMI presented many association rules that provided a 90% correct identification of cartel formation, according to expert assessment. According to auditing specialists, the extracted knowledge could help in the detection, prevention and monitoring of cartels that act in public procurement processes..
