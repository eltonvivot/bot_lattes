2016 - 2019
Simulação em Realidade Virtual com Retorno Tátil para Treinamento de Procedimentos Endodônticos
Descrição: Este projeto visa criar um simulador de realidade virtual e uma metodologia de ensino de endodontia para treinamento de alunos de odontologia que permita simular de forma visual e tátil os procedimentos de um tratamento endodôntico (tratamento de canal). As simulações vão desde a aplicação da anestesia até o selamento do dente, passando pelas etapas de Acesso à Cavidade  Pulpar,  utilizando  instrumentos  rotatórios,  e  Preparo  dos  Canais  Radiculares,  utilizando  limas  endodônticas manuais. O simulador também irá apresentar, em tempo real, os resultados do tratamento que é simulado, permitindo que o aluno identifique possíveis falhas que estejam acontecendo durante a aplicação da técnica escolhida para o tratamento. Outra ferramenta que será desenvolvida é um sistema de visualização pós simulação, que permitirá rever os procedimentos realizados e comparar com outras simulações. Para a interação com o Ambiente Virtual o aluno utilizará o instrumental endodôntico similar ao real acoplado a um dispositivo háptico, que fornecerá a sensação de toque para o aluno. Para aumentar o realismo das simulações serão utilizados modelos gráficos tridimensionais de alta fidelidade e o retorno de força deve reproduzir os efeitos físicos do contato do instrumental com os dentes do paciente virtual. Pretende-se avaliar as técnicas existentes para a simulação de retorno tátil para corpos rígidos e verificar quais são adequados para procedimentos de odontologia, como desgaste de dentes e inspeção por contato. A partir desta avaliação, pretende-se criar ou adaptar as técnicas para uso específico em procedimentos de endodontia. A fim de validar a simulação serão realizados testes  de usabilidade com professores de Endodontia e alunos destadisciplina em cursos de graduação e pós-graduação. Espera-se que com a simulação os alunos possam repetir o treinamento para melhorar suas habilidades e serem colocados em situações que raramente poderiam ser reproduzidas para todos os estudantes em um ambiente real. Ao final doprojeto será
disponibilizado um simulador equipado com um dispositivo háptico e espera-se que seja feita a transferência da tecnologia desenvolvida para ser colocado no mercado a um custo acessível às universidades brasileiras.. Situação: Desativado; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2) . Integrantes: Tales Nereu Bogoni - Coordenador / Maicon Aparecido Sartin - Integrante / Roberta Scarparo - Integrante / Márcio Sarroglia Pinho - Integrante / Patrícia Maria Poli Kopper Morá - Integrante.
2014 - 2018
Modelagem de dados do cadastro técnico multifinalitário integrando dados bidimensionais e tridimensionais
Descrição: Existe uma preocupação global com relação às necessidades do cadastro, entre elas, o Cadastro 3D e sua relação com o cadastro 2D, já existente. O armazenamento e a recuperação de dados são etapas fundamentais para o cadastro, e com a necessidade de manipular estruturas complexas como a sobreposição de imóveis (públicos ou privados), cabos e dutos subterrâneos, comparece a preocupação com a forma de armazenamento, tanto geométrico como suas relações topológicas. Neste contexto, são utilizados métodos de modelagem de dados para auxiliar o desenvolvimento do banco de dados geográfico, incorporando tanto dados 2D quanto dados 3D. Para a modelagem de dados 2D existem vários modelos na literatura, entre eles está o OMT-G e o GeoFrame. Entre os métodos de modelagem 3D estão a Estrutura Formal de Dados 3D, a Rede de Tetraedros, o Modelo Espacial Simplificado e a Representação por Bordas (B-Rep). Assim, o objetivo desta pesquisa é analisar entre os vários métodos de modelagem de dados cadastrais, qual pode ser usado para combinar os dados cadastrais 2D e 3D. Nesse processo, usando dados advindos das prefeituras da região como Colíder, Nova Santa Elena, Nova Canaã, Itaúba, entre outras, poderá ser aplicado o modelo escolhido. Espera-se ao final desta pesquisa apresentar um método para modelagem de dados tanto 2D quanto 3D usando uma ferramenta para isso. Com esse resultado, pretende-se contribuir tanto com as prefeituras envolvidas como com a academia no desenvolvimento de modelos que possam representar a realidade das prefeituras brasileiras, integrando dados alfanuméricos, dados 2D e 3D.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (1) . Integrantes: Tales Nereu Bogoni - Integrante / Marcelo Leandro Holzschuh - Coordenador / Donizete Ritter - Integrante / Alfredo Zenén Domínguez Gonzáles - Integrante.

Desenvolvimento de um protótipo de simulador de colheitadeira axial de grãos para treinamento de operadores
Descrição: O projeto de pesquisa destina-se a desenvolver um protótipo de simulador de colheitadeira axial de grãos para ser utilizado no treinamento de operadores de colheitadeiras. Com o sistema desenvolvido será realizada a validação do protótipo para averiguar sua utilidade como ferramenta auxiliar no processo de ensino-aprendizagem de operadores de colheitadeiras..
