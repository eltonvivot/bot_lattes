Projeto Para o Controle de Trajetória de Um Manipulador Robótico Didático Utilizando Visão Computacional
Descrição: Este projeto apresenta a implementação de um sistema para movimentar um manipulador robótico de acordo com uma trajetória obtida através de processamento de imagens. A trajetória, traçada aleatoriamente em uma folha de papel pelo usuário, é adquirida por uma camera CCD (digitalizada) e processada por um programa de microcomputador. A análise do sistema "área de trabalho - manipulador - câmera" permite a transformação de cada ponto da trajetória para o espaço das juntas do manipulador robótico. Após o cálculo da trajetória no espaço das juntas do manipulador, o manipulador traça a trajetória.
Para a otimização do traçado é feito um ciclo de captura de cada ponto feito pelo manipulador e comparado com a trajetória capturada originalmente. A diferença é utilizada para o novo traçado otimizado da trajetória.
A trajetória e o movimento do braço do manipulador são feitos em um sistema de duas dimensões, pois a câmera gera informações de uma imagem bidimensional.
Os algoritmos de visão computacional apresentados são baseados em implementações de trabalhos anteriores. A linguagem C foi utilizada pela facilidade de programação, versatilidade, pelo fato de haver algoritmos de tratamento de imagem desenvolvidos e, também, porque o manipulador robótico utilizado possui uma biblioteca de arquivos para esta linguagem que permite a interface para controle do movimento do manipulador..
