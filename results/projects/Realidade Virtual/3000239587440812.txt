StreamMining ? Novas Abordagens para Algoritmos de Aprendizagem em Fluxos de Dados Não Estacionários
Descrição: O tratamento de fluxos de grande volume de dados é atualmente um dos principais desafios na área de mineração de dados. Comparada com a mineração de dados estáticos, o processamento de fluxos impõe novos requisitos computacionais aos algoritmos que necessitam incrementalmente processar os dados de entrada enquanto precisam equilibrar o uso limitado de memória e tempo. Além disso, devido às características não estacionárias do fluxo de dados, os modelos de predição precisam constantemente se adaptarem à ocorrência de mudança e evolução de conceito. Essas características dos dados fazem com que os modelos se tornem obsoletos e incapazes de classificar corretamente novos dados. Isso faz com que os modelos tenham que ser regularmente atualizados. Uma das abordagens mais promissoras para lidar com essa situação é a utilização de conjuntos de classificadores (ensembles). Conjuntos de classificadores proveem mecanismos que combinam classificadores base para predizer com maior acurácia que um único classificador. Outra dificuldade relacionada a classificação de fluxos de dados está relacionada ao fato de que nem sempre os dados recebidos pelo fluxo e utilizados no treinamento dos modelos têm sua classe definida. Nesses casos é necessário utilizar abordagens que aprendizado de máquina semi-supervisionadas capazes de construir modelos a partir de dados parcialmente classificados. Portanto, este projeto tem por objetivo investigar e propor algoritmos capazes de classificar fluxos de dados parcialmente classificados, com presença de mudança e evolução de conceitos, usando conjunto de classificadores. Os resultados serão analisados por meio do uso de ferramentas de análise de algoritmos de classificação de fluxos de dados, tais como o Massive Online Analysis (MOA), utilizando bases de dados sintéticas e reais. Este projeto é subdividido em um conjunto de outros projetos que incluem Trabalhos de Conclusão de Curso, Iniciação Científica e Dissertações de Mestrado a fim de alcançar os resultados pretendidos.. TrajectoryPatternMining - Proposta de um Arcabouço Computacional para a Descoberta de Padrões em Trajetórias de Objetos Móveis usando Tecnologias NoSQL
Descrição: As pesquisas na área de mineração de trajetórias de objetos móveis vêm se desenvolvendo consideravelmente nos últimos anos. Estas pesquisas foram fortemente impulsionadas pela obtenção de dados de posicionamento obtidos por meio de sensores embutidos nos mais diversos tipos de equipamentos, desde centrais de rastreamento de veículos automotivos, passando por smartphones, e chegando aos wearbles (computação vestível). Estes sensores vêm produzindo um enorme volume de informação caracterizando o armazenamento de dados de trajetórias de objetos móveis como um problema de Big Data. Tecnologias tradicionais de armazenamento e processamento de dados não proveem o suporte adequado para as demandas destes grandes volumes de dados. Portanto, este projeto tem por objetivo propor um arcabouço computacional que possibilite a mineração e descoberta de padrões em trajetórias de objetos móveis usando tecnologias NoSQL. As tecnologias NoSQL têm como principal vantagem o armazenamento de dados de forma distribuída e em cluster, o que confere a elas maior capacidade de gerenciamento e grandes volumes de dados. Além disso, como grande parte delas não exige a definição prévia do esquema de dados, isso faz com que elas possam oferecer uma forma menos rigorosa de armazenamento que a oferecida pelos bancos de dados relacionais. Aliada às tecnologias NoSQL, a técnica chamada MapReduce está sendo utilizada no processamento de grandes massas de dados distribuídos. Por meio desta técnica podem ser implementados algoritmos de mineração de dados focados na descoberta de padrões em trajetórias de objetos móveis. Neste projeto, pretende-se explorar as vantagens de tais tecnologias para desenvolver e/ou adaptar algoritmos de mineração de dados utilizando a técnica de programação MapReduce para a identificação de padrões em trajetórias de objetos móveis. O intuito é aplicar os resultados deste trabalho na identificação de padrões de deslocamento de condutores de veículos da cidade de Joinville, cujo objetivo é prover informações para o planejamento viário e urbano da região, e na identificação de padrões de jogadas para jogadores de esportes coletivos, tais como futebol e basquetebol, cujo objetivo é prover subsídios aos técnicos na implementação de táticas e jogadas que resultem em um melhor aproveitamento da equipe.. TrajectMining ? Uma Proposta para Armazenamento e Análise de Grandes Volumes de Trajetórias de Objetos Móveis usando Tecnologias NoSQL
Descrição: As pesquisas na área de processamento e análise de trajetórias de objetos móveis vêm se desenvolvendo consideravelmente nos últimos anos. Estas pesquisas foram fortemente impulsionadas pela obtenção de dados de posicionamento através de dispositivos do tipo smartphone. Estes aparelhos vêm produzindo um enorme volume de informação caracterizando o armazenamento de dados de trajetórias de objetos móveis como um problema de Big Data. Tecnologias tradicionais de armazenamento não proveem o suporte adequado para as demandas destes grandes volumes de dados. Portanto, este projeto tem por objetivo implementar um middleware de armazenamento e análise de grandes volumes de trajetórias de objetos móveis usando tecnologias NoSQL. As tecnologias NoSQL têm como principal vantagem o armazenamento de dados de forma menos rigorosa que nos bancos de dados relacionais. Além disso, utilizam modelos agregados de dados que facilitam o processamento eliminando a utilização de junções. Aliada às tecnologias NoSQL, a técnica chamada MapReduce está sendo utilizada no processamento de grandes massas de dados distribuídos. Esta técnica pode ser utilizada na implementação de operadores OLAP aplicados à Big Data. Neste projeto, pretende-se explorar as vantagens de tais tecnologias para criar repositórios NoSQL e operadores OLAP baseados em MapReduce utilizados na extração de conhecimento de trajetórias de objetos móveis. O intuito é aplicar os resultados deste trabalho na identificação de comportamentos comuns de condutores de veículos da cidade de Joinville, cujo objetivo é prover informações para o planejamento viário e urbano da região, e na identificação de comportamentos comuns a barcos que pescam na Baia da Babitonga, cujo objetivo é prover subsídios para ações tanto na área econômica quanto ambiental das cidades ao entorno.. CicloGeoInfo ? Plataforma para Disponibilização de Informações Geográficas sobre Rotas Cicloviárias
Descrição: As pessoas têm optado cada vez mais por carros e motos como meio de locomoção. Entretanto, as consequências disso são os engarrafamentos e o aumento da poluição. Para diminuir estes problemas, a bicicleta se apresenta como uma alternativa para a substituição dos veículos automotores. No intuito de estimular a população a utilizar a infraestrutura cicloviária existente na cidade de Joinville, foi realizado pela UDESC um projeto para mapeamento por GPS (Global Positioning System) de ciclovias e de trilhas rurais da cidade. As ciclovias e trilhas mapeadas podem ser visualizadas através de um sistema de informações geográficas, desenvolvido na forma de um web mapping, para a apresentação de mapas na Internet. O sistema possui algumas limitações, tais como a visualização apenas em navegadores para computadores pessoais. Entretanto, para atender aos novos dispositivos eletrônicos, tais como smartphones e tablets, faz-se necessário portar o sistema para estes tipos de dispositivos, no intuito de aumentar a utilização do sistema e assim fomentar a utilização da bicicleta. Uma alternativa para a solução de tal problema estaria na utilização de paradigmas de desenvolvimento de sistemas modularizados e padrões abertos de interoperação e comunicação. Para este propósito, a utilização de serviços web se apresenta como uma alternativa promissora. Portanto, o objetivo deste trabalho é projetar e desenvolver um sistema de informações geográficas, na forma de uma plataforma orientada a serviços, para a disponibilização e manipulação de informações geográficas cicloviárias, utilizando o padrão web map service como interface para a interação com os diferentes tipos de dispositivos cliente.. AgroMining ? Mineração de Dados Agrícolas para Agricultura de Precisão
Descrição: O Agronegócio brasileiro a cada dia vem se mecanizando e automatizando para atender cada vez melhor as expectativas de diminuição do preço de produção e aumento da produtividade. Para atender essa demanda, cada vez mais são utilizados sensores ativos instalados em máquinas agrícolas, combinados com informações de GPS e imagens de satélite para melhorar a produção. Entretanto, isso traz um volume tão grande de dados que não podem ser analisados diretamente por agricultores ou mesmos especialistas em agricultura. Portanto, dada a abundante quantidade de dados atualmente fornecidos pelas máquinas agrícolas e a grande heterogeneidade de suas fontes, a aplicação de uma técnica de Knowledge Discovery in Database, para a criação de um data warehouse, e a posterior aplicação de técnicas de data mining, podem ser utilizadas para a criação de um sistema de apoio à decisão para a agricultura de precisão. Assim, este projeto tem com objetivo desenvolver um sistema de apoio à decisão para a agricultura de precisão, disponível via Web, que permite aos usuários terem acesso a dados e a processamentos diversos, que lhes auxiliem na quantificação, no entendimento e no gerenciamento da variabilidade espaço-temporal em campo..
