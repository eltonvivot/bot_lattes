2017 - Atual
Pensamento Computacional no Ensino de Ciências
Descrição: A ciência da computação é uma área estratégia para qualquer nação. O conhecimento dos fundamentos da computação é uma necessidade real na educação formal.  O tema "pensamento computacional" envolve questões sobre como buscar soluções de problema como base na ciência da computação. A questão é tão crítica para o desenvolvimento econômico que se tornou uma prioridade em termos de política pública para educação em diversos países como Canadá, Estados Unidos e Reino Unido.  Os conceitos que envolvem o tema Pensamento Computacional é uma forma para aprender e explorar a computação. O desafio não é tornar todos programadores, mas fazer com que todos compreendam melhor os conceitos que envolvem a computação e assim, consigam explorar e criar novos usos para os computadores. Este trabalho foca na investigação de métodos e ferramentas que possibilitem ou facilitem o desenvolvimento do pensamento computacional nos diferentes estágios do ensino formal, do fundamental até o ensino médio.. 2015 - 2020
Projeto de jogos digitais para o Ensino de Ciências
Descrição: Este trabalho visa dar continuidade do projeto de pesquisa anterior, denominado "Projeto de Jogos Digitais na Educação", porém estreitando o foco da investigação de construção de jogos digitais para o ensino de ciência. Embora um jogo digital seja um software, é importante frisar que a construção desse software não deve considerar apenas elementos da Engenharia de Software, também é necessário considerar os aspectos pedagógicos que o uso desse software irá ou deverá proporcionar. Desta forma esta proposta de pesquisa visa integrar as visões técnica e pedagógica na construção de um instrumento de ensino. A pesquisa irá considerar todo processo de construção desde a especificação dos objetivos de ensino, projeto do software até a implementação de protótipos de jogos digitais que favoreçam o ensino de conceitos científicos, especificamente nas áreas de matemática, química, física e biologia. Este trabalho pretende contribuir com ferramentas, técnicas e/ou metodologias para construção e uso de jogos digitais na educação.. Situação: Concluído; Natureza: Pesquisa. 2006 - 2008
Análise de ferramentas e ambientes virtuais de colaboração para aplicação em ensino
Descrição: O objetivo geral desta pesquisa é realizar uma investigação sobre as tecnologias e ferramentas de software utilizadas para construção e/ou operação de ambientes virtuais de colaboração que utilizam a Internet como meio de comunicação e suas aplicações no contexto do ensino. Os objetivos específicos são: elucidar um conjunto de fatores que caracterizam as ferramentas e ambiente virtuais de colaboração, fornecer um guia para construção de ambientes virtuais de colaboração aplicados ao ensino e áreas adjacentes. Platine e Moodle, MSN Messenger e Java são exemplos de ambientes virtuais, ferramentas e tecnologia, respectivamente, que serão investigados.. Situação: Concluído; Natureza: Pesquisa. Integrantes: Wilson Massashiro Yonezawa - Coordenador.
