Heurísticas de mineração de dados híbridas para  o problema de roteamento de veículos capacitado
Descrição: O problema de roteamento de veículos capacitado (PRVC) é um problema NP-difícil. Para problemas de grande escala, é muito difícil alcançar uma solução ideal com métodos de otimização tradicionais devido à alta complexidade computacional. Neste projeto de pesquisa, novos métodos de solução serão desenvolvidos a partir de hibridizações de heurística da literatura com técnicas de mineração de dados.
Para validar os métodos propostos, serão realizados experimentos computacionais a partir de uma base de dados real, que se apresenta desafiadora para métodos tradicionais da literatura.. Heurística híbrida baseada em GRASP e VNS aplicada ao problema de localização de p-hubs capacitado
Descrição: O problema da localização de p-hubs é um problema NP-difícil, que consiste em encontrar p pontos para estabelecer facilidades e atribuir clientes a esses pontos. Na versão capacitada desse problema, cada hub tem uma capacidade máxima limitando o serviço a ser prestado. Diferentes relaxações do modelo matemático e metaheurísticas como GRASP (Greedy Randomized Adaptive Search Procedures) e VNS (Variable Neighborhood Search) têm sido muito eficazes para resolver problemas de alocação de p-hubs. Neste projeto de pesquisa, é proposto um RL-GRASP-VNS híbrido para o problema de localização de p-hub capacitado, no qual uma relaxação do modelo e o VNS são utilizados, respectivamente, nas fases de construção e busca local do GRASP. Experimentos computacionais serão realizados a partir de uma base real de dados, e comparações serão feitas entre o RL-GRASP-VNS e outros métodos da literatura..
