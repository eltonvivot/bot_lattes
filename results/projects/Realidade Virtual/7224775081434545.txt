Projeto e desenvolvimento de Novas Tecnologias para o Ensino de Física e o uso do Arduino no Ensino de Ciencias
Descrição: Este projeto visa desenvolver recursos didaticos para o Ensino de Física, utilizando-se de novas tecnologias, tais como sistemas de aquisição automática de dados e simulações. Alem disso recursos experimentais são tambem desenvolvidos com o objetivo de permitir a inserção de Fisica Moderna no Ensino Médio.
