METODO DE ANALISE DA CONSISTENCIA GEOMETRICA: DESENVOLVIMENTO DE SOFTWARE DE APLICACAO PARA BASE DE DADOS GEORREFERENCIADOS
Descrição: Desenvolver sistema computacional para análise da consistência geométrica de rodovias de pista simples a partir de dados georreferenciados. A análise da consistência geométrica decorre da avaliação, através de modelos comportamentais, das reações do condutor do veículo mediante características geométricas das vias. As características plani-altimétricas das vias podem ser obtidas através de base de dados georreferenciados com precisão suficiente para análises comportamentais.
A análise da consistência geométrica é um instrumento utilizado em diversos países para avaliar questões relacionadas a seugurança viária de projetos já implantados (obras existentes) como daqueles em fase de estudo (anteprojeto e projeto)..
