2012 - 2013
Produção de uma interface gráfica (software acadêmico) para a simulação de uma coluna de destilação.
Descrição: Neste projeto foi desenvolvido um modelo simulador de uma coluna de destilação em ambiente MATLAB, após a comparação dos resultados do modelo com um modelo já existente, implementou-se uma interface gráfica do usuário (GUI) para a maior facilidade de utilização dos recursos do simulador.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (1) . Integrantes: Felipi Luiz de Assunção Bezerra - Coordenador / Cristiane Holanda Sodré - Integrante.
