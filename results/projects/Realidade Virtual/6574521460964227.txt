2007 - 2009
Máquina de Busca para Imagens disponíveis na Web usando Consultas por Similaridade
Descrição: A utilização cada vez mais disseminada dos recursos da Internet vem promovendo mudanças radicais na forma como as informações estão sendo e serão recuperadas, independentemente da área de conhecimento. Para o aproveitamento desses rescursos, não basta apenas o conhecimento do que é possível utilizar mas, principalmente, como pode-se localizar estes recursos. Neste sentido, é fundamental o papel das chamadas "máquinas de busca". Uma máquina de busca representa uma base de dados contendo os endereços das mais variadas páginas da rede mundial, indexadas por palavras para uma busca posterior. Na prática, estes serviços servem a diferentes propósitos, como por exemplo, procura de endereços/telefones de pessoas, busca de imagens por palavras, mapas, temas da área de saúde, entre outras. No entanto, todas essas máquinas de busca da internet utilizam como parâmentro para a consulta uma ou mais palavras. A internet além de fornecer uma biblioteca rica em textos, também contém uma infinidade de imagens. Essa grande quantidade de imagens deve-se a popularização dos meios de digitalização através do uso de dispositivos como e máquinas fotográficas digitais. Embora exista essa grande quantidade de imagens, não existe na Web uma ferramenta que permita aos usuários usarem uma imagem como parâmetro de sua consulta para localizar as imagens mais semelhantes. O objetivo principal desse projeto é a implementação de um sistema para consultar imagens na Web, onde o usuário introduz uma imagem e a máquina de busca retornará todas as imagens semelhantes. Um ponto inovador deste trabalho é a utilização de máquinas de busca e estruturas de indexação métrica para realizar consultas por similaridade das imagens..
