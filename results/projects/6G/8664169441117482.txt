2021 - Atual
Orquestração e Alocação de Recursos em Arcabouço MEC-NFV para Redes 5G (Bolsa de Produtividade em Pesquisa - PQ-2)
Descrição: As redes de comunicações móveis de quinta geração (5G - Fifth Generation) estão em fase de implantação e objetivam fornecer serviços aprimorados em relação aos das atuais redes 4G, considerando três categorias: eMBB (enhanced Mobile Broadband), URLLC (Ultra Reliable Low Latency Communication) e mMTC (massive Machine Type Communications). Apesar da maior vazão proporcionada pela interface aérea 5G, a orquestração e alocação de recursos tanto no núcleo quanto na rede de acesso ainda são aspectos que carecem de soluções eficientes, escaláveis e flexíveis para garantir a melhor experiência para o usuário final e de soluções setores verticais como automotivo, e-health e agricultura de precisão, ao mesmo tempo em que garantam o uso otimizado dos recursos da rede. Neste sentido, os paradigmas de Redes Definidas por Software (SDN ? Software-Defined Networking), Virtualização de Funções de Rede (NFV- Network Functions Virtualization), Computação na Borda (Edge Computing) e Redes Centradas na Informação (ICN ? Information Centric Networking), como a implementação NDN (Named Data Networking), além das técnicas de inteligência artificial, são considerados elementos importantes tanto para a arquitetura 5G quanto para a Internet do Futuro.
Este projeto emprega técnicas de otimização, modelos baseados em cadeias de Markov e inteligência artificial na análise e proposição de novas soluções para quatro frentes de pesquisa no contexto de redes 5G e Futuras arquiteturas para a Internet: 1) Alocação dinâmica de VNFs (Virtual Network Functions) e Controladores SDN em ambiente NFV-MEC; 2) Confiabilidade e alocação de recursos em ambiente MEC híbrido baseado em máquinas virtuais e contêineres na borda da rede; 3) Alternativa baseada em NDN para caching e encaminhamento de mensagens e integração com a borda da rede utilizando aprendizado de máquina; e 4) Arquitetura em camadas que se beneficie dos paradigmas SDN, NFV, MEC e NDN para a orquestração e gerenciamento de recursos de forma inteligente e com base em técnicas de aprendizado de máquina no contexto de network slicing em ambientes dinâmicos 5G.. Situação: Em andamento; Natureza: Pesquisa. Alunos envolvidos: Mestrado acadêmico: (1) Doutorado: (2) . 2019 - 2020
P4Sec - Securing Networks in the Programmable Data Plane Era
Descrição: Recent advances in Software Defined Networking (SDN) have expanded our ability to program the network to its data plane. Through domain specific languages like P4 [3], network operators can quickly deploy new protocols on forwarding devices, customize their functionality, and develop innovative services [13, 5, 17]. This flexibility comes however with a cost: network-wide security and correctness properties (e.g., isolation, reachability, etc.) become much harder to ensure, because network behavior is now determined by a combination of the control plane-driven configuration and the data plane program that resides on devices (also called switches). Existing network verification tools [7, 14, 19, 15, 35], which rely on a fixed, invariant model of the data plane, are inadequate for programmable data planes. We propose to develop new techniques to verify and enforce security properties in data plane networks. Our proposed verification techniques extend existing verification tools by automatically generating a data plane model from a P4 program.We then adapt existing verifications tools to work with our dynamicallygenerated models to verify network configuration updates issued by an SDN controller. We also propose a new approach to ensuring that network security properties are satisfied by a network configuration that is based on data plane enforcement. We propose to develop an in-line monitor, implemented in the data plane itself, that enforces critical security properties, such as isolation and bandwidth limits, even in the presence of a faulty user data plane program or controller. Today?s networks are increasingly called on to perform security services, e.g. intrusion detection, in the network. A programmable data plane network can be used to accelerate and enhance such services by performing some of the required processing in the network data plane. The second thrust of this proposal concerns new data plane security services made possible by programmable data plane networks. We propose to develop new data plane-based security functions and techniques for composing them in a manner that balances performance characteristics of the data plane (very fast but less complex) and dedicated virtualized network function units attached to the network (slower but more complex) to implement the required security function. Programmable data planes are poised to transform the architecture of modern networks. The proposed work will advance the current state-of-the-art by extending verification and enforcement techniques to programmable data plane networks, for which neither network verification nor security policy mechanisms currently exist. Our work will also develop new data plane services that will enable a new class of security functions to be deployed in the network in order to improve the overall security of computer networks. Our project follows two branches of investigation: First, we propose to determine which network security properties can be verified in networks with programmable data planes and which properties can be dynamically enforced by a programmable data plane. Second, we investigate new security services made possible by programmable data planes.. Situação: Concluído; Natureza: Pesquisa. Integrantes: Kelvin Lopes Dias - Integrante / Stênio F. L. Fernandes - Integrante / José Augusto Suruagy Monteiro - Coordenador / Michel Sales Bonfim - Integrante / Marcelo Anderson Santos - Integrante.

2018 - 2021
Tecnologias-Chave para as Redes sem Fio do Futuro
Descrição: Até recentemente, as pesquisas em redes de computadores focavam em desafios (ex., controle de congestionamento, QoS, mobilidade), cujas soluções consideravam abordagens confinadas em tecnologias de redes de um único padrão e, em geral, sob a ótica de uma ou poucas camadas da pilha de protocolos. Esses desafios têm sido tradicionalmente resolvidos através de remendos (patches) aplicados à pilha de protocolos. Dessa forma, abordagens clean-slate no projeto de Internet do futuro para atacar o chamado engessamento (ossification) das redes baseadas na arquitetura TCP/IP têm sido o foco de muitos pesquisadores e da indústria com vistas a promover inovação, flexibilidade e programabilidade da rede. Além disso, há o surgimento e fomento de diversos novos nichos e aplicações móveis/sem fio, como computação vestível (Wearable Computing), realidade aumentada, carros conectados e autônomos, Internet tátil, com os mais variados requisitos. Tais aplicações são destinadas aos usuários finais por meio de smartphones, sensores e diversos gadgets (e.g., google glass), que compõem a Internet das Coisas (IoT ? Internet of Things). Dessa forma, considerar abordagens one-size-fits-all parece improvável diante de tamanha heterogeneidade de requisitos de aplicações e tecnologias de redes. Esta proposta considera elementos essenciais para o sucesso das redes sem fio do futuro e aplicações que delas dependerão. Neste projeto serão avaliados isoladamente e de forma integrada os seguintes pilares para as redes sem fio do futuro: 5G (Fifth Generation of Mobile Communications), SDN (Software-Defined Networking), NFV (Network Function Virtualization), ICN (Information Centric Networking), CR (Cognitive Radio)/SDR (Software Defined Radio) e EC (Edge Computing). Propostas de algoritmos, mecanismos e arquiteturas para este projeto envolvem soluções para as demandas diversas, tais como: garantir requisitos de atrasos, aumentar tempo de vida das baterias dos dispositivos, eficiência energética, flexibilidade e programabilidade da rede, virtualização de funções de rede, uso eficiente do espectro eletromagnético e comunicação centrada no conteúdo.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Mestrado acadêmico: (3) Doutorado: (3) . Cognitive Radio Applications in Mobile Device
Descrição: A cognitive radio is an intelligent radio that can be programmed and configured dynamically. Its transceiver is designed to use the best wireless channels in its vicinity. Such a radio automatically
detects available channels in wireless spectrum, then accordingly changes its transmission or reception parameters to allow more concurrent wireless communications in a given spectrum band at one location. This process is a form of dynamic spectrum management. This line of research aims to explore and develop cognitive radio applications with Radio emulation support provided by 3GPP simulation infrastructure.. Situação: Concluído; Natureza: Pesquisa. 2003 - 2004
P2P (Peer-to-Peer): Computação colaborativa - http://www.gprt.ufpe.br/gtp2p
Descrição: O Grupo de Trabalho em Computação Colaborativa (GT-P2P da RNP) foi criado em agosto de 2003 com o objetivo de avaliar os benefícios da implantação de um serviço de suporte a sistemas P2P na RNP e instituições conectadas, bem como analisar o impacto da utilização de tais sistemas no desempenho da rede. Portanto, o GT-P2P propôs a construção de um Projeto Piloto audacioso composto por: Infra-Estrutura e Análise do tráfego de aplicações P2P. O middleware XPeer pode ser considerado como a principal contribuição do projeto piloto. Apesar de existirem várias ferramentas que auxiliam o desenvolvimento de aplicações P2P como JXTA, .NET e Groove, construir esse tipo de aplicação foi e ainda é uma tarefa bastante árdua. Os desenvolvedores desse tipo de sistema se deparam constantemente com problemas difíceis de serem solucionados e que demandam tempo e requerem uma pesquisa exaustiva e um esforço concentrado de estudo antes de serem solucionados. Além do XPeer, fazem parte da infra-estrutura um componente que possibilita a comunicação direta entre dois nós, mesmo que estes estejam em baixo de firewall e nat, e um outro componente para realizar a comunicação entre aplicações e o XPeer. Foram desenvolvidas aplicações para avaliar o middleware XPeer. A segunda parte do projeto piloto consistiu em avaliar o tráfego de aplicações P2P no backbone da RNP através dos sistemas P2P existentes na Internet, principalmente nas aplicações de transferência de arquivos . Um conjunto de ferramentas (analisadores) foi desenvolvido para coleta e análise dos fluxos de dados das aplicações P2P. Com base nos arquivos processados pelos analisadores é possível descobrir a quantidade de banda consumida por cada aplicação, o horário mais utilizado por esse tipo de aplicação e o perfil dos dados transferidos. Situação: Concluído; Natureza: Pesquisa.
