2022 - Atual
Alocação Dinâmica de Recursos Virtualizados em Ambientes de Edge Computing
Descrição: O objetivo deste projeto é a realização conjunta de um projeto de P&D visando especificar e implementar soluções tecnológicas para a alocação dinâmica de recursos virtualizados em ambientes de Edge Computing voltados para aplicações de Cidades Inteligentes. Nesse novo projeto, serão, inicialmente, realizados extensivos experimentos para uma avaliação de desempenho abrangente dos algoritmos propostos em projeto anterior, seguindo uma metodologia bem definida. Além disso, os experimentos serão norteados por requisitos que devem ser satisfeitos e usarão data sets de aplicações reais. Para tal, serão especificados os requisitos e modelados cenários de aplicações para Cidades Inteligentes, que são altamente dinâmicas e, consequentemente, demandam agilidade e eficiência na alocação de recursos. Tais características tornam as aplicações para Cidades Inteligentes casos de uso adequados para avaliação da solução proposta. A partir dos resultados da avaliação, uma nova versão do algoritmo, mais eficiente e robusta será proposta, explorando novas técnicas para as decisões de dimensionamento, tanto vertical quanto horizontal.. 2021 - 2022
Framework para gerenciamento e execução de Funções de Rede Virtualizadas em infraestruturas 5G
Descrição: O objetivo deste projeto é a realização conjunta de um projeto de P&D visando especificar e implementar um Framework para gerenciamento e execução de Funções de Rede Virtualizadas em infraestruturas 5G. No projeto desenvolvemos soluções tecnológicas para a integração dos paradigmas de Edge Computing e de Virtualização de Funções de Rede (do inglês Network Functions Virtualization - NFV), sob a forma de um framework de orquestração de cadeias de serviços virtualizados. Nesse projeto o compartilhamento de recursos é feito na camada de serviços/aplicações, sem explorar diretamente as camadas subjacentes de redes. Neste novo projeto pretendemos investigar formas de adaptar e estender os mecanismos de gerenciamento e execução de NFV do ambiente do projeto para ser ciente da rede subjacente, mais especificamente, da rede 5G. Em particular, iremos explorar o conceito de fatiamento de redes 5G (slicing 5G) introduzido pela Aliança NGMN (Next Generation Mobile Network Alliance) como uma abordagem flexível de rede em software onde múltiplas redes virtuais (VNs) operam em uma infraestrutura compartilhada. Um slice de rede fim a fim é uma combinação de VNs na rede principal (Core Network) e nas Redes de Acesso Rádio (RANs). Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Doutorado: (4) . 2012 - Atual
Projeto IPTraf
Projeto certificado pelo(a) coordenador(a) Luis Felipe Magalhães de Moraes em 10/03/2021.
Descrição: O Projeto IPTraf é um projeto de coleta e monitoramento contínuo de fluxos de dados de redes. A ferramenta Coletor é construída a partir de uma infraestrutura de coleta de dados efetuada na Rede-Rio/FAPERJ, a qual exporta as informações estatísticas dos fluxos dados dos roteadores de borda para serem posteriormente disponibilizados pelo IPTraf através de uma interface web amigável com recursos que auxiliem a tomada de decisão por parte de administradores de rede. Sobre esses dados coletados, diversas ferramentas são desenvolvidas, como por exemplo as ferramentas de detecção de anomalias, que fazem parte da ferramenta Guardião.. 2009 - 2011
Aplicando técnicas de computação ubíqua e pervasiva a aplicações de tele-saúde
Projeto certificado pelo(a) coordenador(a) Alexandre Sztajnberg em 25/01/2021.
Descrição: Este projeto visa desenvolver tecnologia para aplicações de tele-saúde (remote assisted living) onde dispositivos de monitoramento, sensores, e atuadores podem operar de forma autônoma, mas cooperativa, permitindo o tele-monitoramento e tele-diagnóstico em tempo real de pacientes com doenças crônicas (principalmente idosos) em suas habitações, sem afetar suas qualidades de vida. O objetivo do projeto, no contexto do PIBIC, é o desenvolvimento do software de suporte para a aquisição e armazenamento de informações de contexto coletadas a partir de dispositivos como sensores, monitores de sinais vitais, equipamentos de rede ? que são geralmente dispositivos com poucos recursos. Além disso, também será desenvolvido software para exibição das informações coletadas e mensagens contendo informações sobre eventos de interesse do usuário.. Situação: Concluído; Natureza: Pesquisa.
