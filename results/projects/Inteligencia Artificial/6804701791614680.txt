2022 - Atual
Processos de Desenvolvimento para Sistemas Multiagentes
Descrição: Este projeto dá continuidade ao projeto anterior, cadastrado no SIPPEE, sob o título de ?engenharia de requisitos para sistemas multiagentes?. O projeto anterior produziu efetivamente um processo de engenharia de requisitos específico para sistemas multiagentes, cobrindo todas as fases da engenharia de requisitos, conforme definido no SWEBOK (Software Engeneering Body of Knowledge). O referido processo foi enriquecido com uma técnica de elicitação de requisitos específica para sistemas multiagentes, bem como um modelo de especificação de requisitos para esse tipo de sistema e uma técnica que permite inspecionar e verificar os requisitos especificados, garantindo assim a sua qualidade.
Estamos ainda trabalhando para melhorar e enriquecer este processo, acrescentando novas técnicas e melhorando o processo como um todo. Todavia neste projeto atual, pretendemos atuar no desenvolvimento de um processo de desenvolvimento completo para sistemas multiagentes, incluindo as fases de projeto, testes e implementação. Atualmente, estamos iniciando a acrescentar uma fase de projeto ao nosso processo atual, com o objetivo de torná-lo mais completo.. Situação: Em andamento; Natureza: Pesquisa. Alunos envolvidos: Graduação: (3)  / Mestrado profissional: (3) . Integrantes: Gilleanes Thorwald Araujo Guedes - Coordenador.

2016 - 2018
Extensão da Linguagem MAS-ML por Meio de um Metamodelo UML para a Modelagem de Requisitos
Descrição: Pretende-se estender a linguagem MAS-ML ? MultiAgent Systems Modeling Language, adicionando a esta linguagem as metaclasses propostas no metamodelo de Guedes (2012) para a modelagem de requisitos em sistemas multiagentes, posto que a referida linguagem não possui mecanismos para a modelagem de requisitos específicos para o projeto deste tipo de sistema, de forma a poder representar os papéis assumidos pelos agentes, seus objetivos, os planos para que estes objetivos possam ser atingidos, as percepções necessárias para que um objetivo se torne uma intenção e as ações que cada agente poderia realizar.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (1) . Integrantes: Gilleanes Thorwald Araujo Guedes - Coordenador.
2012 - 2014
Reestruturação do Projeto Aequator - Tutorial Hipermídia para a Inclusão Digital
Descrição: O projeto Aequator já foi iniciado há alguns anos, tendo inclusive gerado várias monografias de conclusão de curso e publicações de artigos completos e resumos. Todavia o projeto encontra-se ainda inconcluso, não tendo efetivamente atingido seus objetivos dentro do prazo previsto. Produziu-se a análise e projeto do sistema e iniciou-se sua implementação, porém ainda falta muito para considerá-lo concluído. O referido projeto foi suspenso por alguns anos, uma vez que seu coordenador saiu para cursar o doutorado e somente agora volta a retomá-lo.
Sendo assim, pretende-se revisar novamente o projeto, reestruturando-o e atualizando e retomar o seu desenvolvimento, de maneira a poder aplicá-lo na inclusão digital da população ainda não familiarizada com os recursos computacionais atualmente amplamente disponíveis e que muitas vezes são uma exigência básica para contratação.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2) . Integrantes: Gilleanes Thorwald Araujo Guedes - Coordenador.
