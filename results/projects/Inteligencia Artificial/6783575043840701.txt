2019 - 2019
Livro Rápido - Aplicativo para troca/empréstimo de livros
Descrição: O presente projeto tem por objetivo o desenvolvimento de uma plataforma para trocas/empréstimo de livros. Além disso, será possível realizar comentários sobre os livros que os usuários possuem. A plataforma será desenvolvida e implementada por meio de uma sistema web e o fornecimento de uma API REST. Esta API será utilizada para consumir dados através de dispositivos móveis. A plataforma poderá ser utilizada tanto na versão Web quanto na versão mobile através do aplicativo. Além disso, será implementada uma funcionalidade que permita identificar interesses em comum através de geolocalização disponível na plataforma Android.. Situação: Concluído; Natureza: Pesquisa. 2019 - 2019
Plataforma web para solicitação de reposições e elaboração de cronograma de estudo
Descrição: O Instituto Federal de Ciência e Tecnologia, através da Coordenação de Sistemas de Informação, tem desenvolvimento o Sistema Unificado de Administração pública (SUAP). Ele possui muitas funcionalidades que têm tornado o trabalho mais eficiente. Entretanto, o sistema ainda não possuí a funcionalidade que permite a alunos e professores realizarem a solicitação de reposição por meio da plataforma. Atualmente, esta solicitação é realizada presencialmente e por meio de formulário em papel. Esta prática pode comprometer a comunicação entre alunos e professores, assim como pode por em risco a própria solicitação por motivos de extravio do papel, por exemplo. Portanto, tem-se por objetivo desenvolver uma aplicação web que permite aos alunos e professores melhor comunicação quanto a solicitação de reposições. Além disso, o sistema contará com diversas funcionalidades como organização de cronograma de estudo.. Situação: Concluído; Natureza: Pesquisa. Integrantes: Romerito Campos de Andrade - Coordenador / TAYNAH AZEVEDO ALVES DA COSTA - Integrante / Denis Silva Soares - Integrante.
2019 - 2019
Coleta Inteligente - Aplicativo para auxílio à coleta de lixo
Descrição: O projeto tem como principal objetivo fazer com que o lixo das residências seja recolhido pouco tempo depois de ser posto para fora das residências, isso evitaria problemas com animais, chuvas, mal cheiro e aspecto visual. Para resolver isto, será criado um aplicativo que rastreará o movimento do caminhão de lixo que recolhe os resíduos de uma certa área. Quando o caminhão se aproximar o bastante da residência dos moradores, uma notificação será enviada para o usuário do aplicativo. O caminhão será rastreado por meio de um smartphone que possua o aplicativo instalado, o aparelho vai ficar dentro do caminhão, com GPS ativo e já cadastrado no software. Este aplicativo será desenvolvido utilizando os recursos da plataforma Android.. Situação: Concluído; Natureza: Pesquisa. Integrantes: Romerito Campos de Andrade - Coordenador / Tácio dos Santos Medeiros - Integrante.Financiador(es): Instituto Federal de Educação, Ciência e Tecnologia do Rio Grande do Norte - Bolsa.
2018 - 2018
Revisão Bibliográfica do Problema da Árvore de Steiner - representação de solução e algoritmos clássicos
Descrição: O problema da árvore de Steiner é um problema de otimização NP-Difícil. Várias abordagens têm sido propostas para sua resolução. Em geral, abordagens baseadas em meta-heurísticas têm sido muito empregadas para resolução do problema com ótimos resultados. Os projetos de algoritmos têm como fase inicial a determinação da representação da solução a ser otimizada e isso é uma etapa crucial que determina o projeto do restante do algoritmo. O presente projeto de pesquisa tem por objetivo fazer um levantamento bibliográfico sobre o problema da árvore de Steiner. O foco principal é fazer um levantamento sobre as abordagens para representação de solução. Além disso, espera-se classificar e implementar as soluções encontradas através de um framework que ser usado para construção de algoritmos clássicos, considerando partes do projeto como componente de software reutilizável.. Situação: Concluído; Natureza: Pesquisa. Integrantes: Romerito Campos de Andrade - Coordenador / PABLO GUSTAVO FERNANDES MAIA - Integrante / Bianca Araújo Fernandes Veras - Integrante / SÂMARA LUÍZA BARROSO DE ARAÚJO ALVES - Integrante / YWRY SCHELLER MEDEIROS GALVÃO - Integrante.
