Sistema Fuzzificado de Recomendação de Objetos de Aprendizagem Hipermidiáticos
Descrição: Ao recomendar algum dos OAs, é importante considerar as incertezas que o aluno traz consigo a acerca do conteúdo abordado. Sendo assim os conjuntos fuzzy representam perfeitamente estas incertezas. Sistemas de recomendação WEB baseados em decomposições matriciais: Um estudo sobre a aplicação de métodos estatísticos robustos no processo de predição
Descrição: A quantidade de informações disponíveis na WEB faz surgir a necessidade de filtragem automática de conteúdo. Neste sentido, sistemas de recomendação tornaram-se uma importante área de pesquisa na medida em que a internet foi crescendo e fazendo parte da rotina das pessoas. Exemplos de aplicações incluem sistemas que recomendam livros, filmes e notícias de acordo com características específicas de cada usuário. Contudo, apesar de inúmeros avanços, prover recomendações de conteúdo, produtos e serviços personalizados na WEB ainda é um desafio, e requer melhorias no sentido de oferecer métodos de recomendação mais efetivos. Este projeto tem por objetivo investigar a aplicação de métodos estatísticos robustos na recomendação de produtos/serviços condizentes com perfis de usuários. Usualmente, matrizes de avaliação de produtos (Usuários versus Produtos) são empregadas para capturar notas atribuídas por usuários a produtos. Fatorações dessas matrizes estão mostrando-se bastante úteis para sintetizar perfis similares de usuários e, conseqüentemente, recomendar produtos/serviços/conteúdos para usuários que partilham de opiniões similares. Neste sentido, algoritmos robustos que tratam outliers (dados espúrios) podem ser empregados para melhorar o desempenho das recomendações. Algoritmos como o RANSAC (random sample consensus) apenas recentemente estão sendo estudados no contexto de sistemas de recomendação. Sendo assim, pretende-se abordar este contexto para propor potenciais inovações na área. No desenvolvimento deste trabalho pretende-se utilizar bancos públicos de avaliações de produtos para a elaboração e validação dos experimentos.. Situação: Concluído; Natureza: Pesquisa. Integrantes: Sidnei Renato Silveira - Integrante / Wilson Gavião Pires Neto - Coordenador.Financiador(es): Centro Universitário Ritter dos Reis - Remuneração.
