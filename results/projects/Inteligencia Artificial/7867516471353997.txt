2012 - 2014
Modelagem de fenômenos convectivos e fenômenos solares por mineração de dados meteorológicos e dados de clima espacial
Projeto certificado pelo(a) coordenador(a) Stephan Stephany em 10/09/2012.
Descrição: O objetivo deste projeto é investigar o uso de técnicas de Inteligência Computacional, com ênfase em Redes Neurais Artificiais, como meio de se extrair de bases de dados meteorológicos, no caso análises e previsões de modelos numéricos, uma modelagem preditiva confiável de fenômenos convectivos severos. Um objetivo paralelo, relativo ao clima espacial, é a obtenção de padrões relevantes para o fenômeno da cintilação ionosférica, produzido pela atividade solar, objetivando sua predição. Ambas as metodologias serão testadas e validadas com auxílio de especialistas de forma que possam ser posteriormente disponibilizadas para uso como ferramenta de análise auxiliar para os especialistas da área. No caso dos meteorologistas, a predição de ocorrência de atividade convectiva seria usada conjuntamente com imagens e dados de satélites, radares e estações meteorológicas, além das análises e previsões de modelo numérico, no ambiente operacional de previsão do tempo.. Situação: Concluído; Natureza: Pesquisa. Integrantes: Glauston Roberto Teixeira de Lima - Integrante / Stephan Stephany - Coordenador.Financiador(es): Conselho Nacional de Desenvolvimento Científico e Tecnológico - Bolsa.
2008 - 2011
Desenvolvimento de um sistema computacional com técnicas de mineração de dados para análise de dados de previsão climática utilizando abordagens estatísticas e de inteligência computacional (Processo : 382918/2008-8)
Descrição: O trabalho como proposto está no escopo de modelagem computacional adaptável para fenômenos ambientais, focando a análise de dados meteorológicos. O objetivo do trabalho foi desenvolver uma metodologia de modelagem utilizando técnicas de mineração de dados baseadas em técnicas estatísticas e técnicas de inteligência computacional, procurando gerar informações para uso no escopo de predições meteorológicas, de forma a disponibilizar mais informações simuladas a partir de dados observacionais históricos, buscando agregar mais possibilidades que resultassem em uma melhoria dos processos e dos sistemas de predição.
Além disso, durante o desenvolvimento do trabalho, os dados resultantes dos processos de mineração foram utilizados no treinamento de sistemas preditores baseados em técnicas de reconhecimento de padrões, implementadas por abordagens estatísticas e/ou abordagens baseadas em inteligência computacional.. Situação: Concluído; Natureza: Pesquisa. 2008 - 2010
CB-MINING - MINERAÇÃO DE DADOS ASSOCIADOS A SISTEMAS CONVECTIVOS
Projeto certificado pelo(a) coordenador(a) Stephan Stephany em 10/09/2012.
Descrição: Descrição: A previsão da evolução de sistemas convectivos tem grande importância científica e econômica devido aos efeitos das tempestades associadas na superfície terrestre e na própria atmosfera. Atualmente, dispõe-se de uma grande variedade de dados meteorológicos, tais como dados de perfis atmosféricos derivados de estações de radiossondagem, dados de modelos meteorológicos de previsão numérica de tempo, dados de radares e satélites meteorológicos e dados de descargas elétricas atmosféricas. Estes últimos devem-se à atividade elétrica intensa associada aos sistemas convectivos. Em função da alta resolução espacial e temporal dos dados meteorológicos em geral e do consequente volume de dados gerados por modelagem ou sensores, sua análise requer o uso de ferramentas computacionais avançadas. Neste projeto exploram-se técnicas de mineração de dados, uma ferramenta chave, para buscar correlações potencialmente úteis entre os dados meteorológicos relativos ao Pantanal Matogrossense, de forma a caracterizar a atividade convectiva intensa, objetivando sua previsão. A existência de atividade convectiva severa será inferida de dados de ocorrência de descargas elétricas nuvem-solo, além de dados de radar e satélite. Este tipo de análise pode complementar os métodos convencionais de investigação de sistemas convectivos, auxiliando na previsão meteorológica de curto prazo de tempestades severas, tanto do ponto de vista dinâmico, eletrodinâmico e termodinâmico. A aplicação da mineração de dados à investigação de sistemas convectivos é de grande interesse em pesquisa em Meteorologia, especificamente no tocante à caracterização da atividade convectiva no Pantanal Matogrossense. Projeto relativo ao Edital Universal - chamada 02/2006, MCT/CNPq. Situação: Concluído; Natureza: Pesquisa.
