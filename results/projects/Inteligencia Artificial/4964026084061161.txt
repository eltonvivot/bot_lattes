AgDataBox (ADB) web platform for data integration, software, procedures and methodologies for Digital Agriculture
Descrição: ADB is a platform that provides free computing tools for farmers, researchers and service providers, focused on precision agriculture and digital agriculture. This cloud platform has an application programming interface (API), which allows access to several applications, and the following are already under construction: 1) ADB-Mobile ? mobile application for collecting and management of data for farmers; 2) ADB-Map - web application to work with maps and spatial data in precision agriculture; 3) ADB-Admin - web application to manage agricultural data; 4) ADB-IoT - Module for collecting and delivering data to IoT devices; 4) ADB-DataMining: Is an application for discovering patterns in large data sets involving methods at the intersection of machine learning, statistics, and database systems. This research is a joint project coordinated by UNIOESTE (Western Paraná State University, Cascavel) and UTFPR (Federal University of Technology ? Estudando Técnicas da Engenharia de Requisitos Utilizadas no Processo de Desenvolvimento Orientado a Aspectos
Descrição: Objetivamos neste projeto fazer um estudo sobre o Paradigma Orientado a Aspectos, utilizando o mesmo em todas as etapas do Processo de Engenharia de Requisitos. Para o estudo desse novo paradigma de desenvolvimento de software necessitamos compará-lo com outros tais como abordagens orientadas a agentes, a objetos, a objetivos, estruturadas, entre outras. Esta comparação permitirá que possamos avaliar as vantagens e desvantagens do uso do desenvolvimento orientado a aspectos. Pretendemos propor um conjunto de diretrizes para efetivamente integrar a definição de Requisitos não-funcionais, organizacionais e funcionais no paradigma Orientação a Aspectos. Um protótipo de ferramenta computacional será desenvolvido visando dar suporte à aplicação das diretrizes. A ferramenta e as diretrizes serão aplicadas a estudos de caso reais permitindo avaliar com mais propriedade os resultados obtidos.. Situação: Desativado; Natureza: Pesquisa.
