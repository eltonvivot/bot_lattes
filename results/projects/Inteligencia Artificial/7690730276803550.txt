2017 - 2018
Métricas em projeto resiliente de sistemas térmicos
Descrição: A primeira etapa na criação de qualquer sistema de engenharia é o projeto conceitual. A maioria dos sistemas de engenharia, incluindo sistemas térmicos, são projetados dentro de uma faixa fixa de capacidade nominal e pode levar o sistema a funcionar mal fora dessa faixa na presença de eventos adversos. No entanto, o projetista muitas vezes carece de informações fundamentais sobre taxa de falha de um componente durante a fase de projeto conceitual. Projeto resiliente é um campo relativamente novo de pesquisa onde um sistema complexo de engenharia qualquer é concebido de modo a manter-se operacional apesar existência de falhas. Ao longo últimas décadas, pesquisas em redes de computadores têm investigado a resiliência de redes complexas. O estudo da topologia da rede a partir desta área traz alguns paralelos interessantes sobre o projeto de sistemas complexos de engenharia, tais como sistemas térmicos. Neste estudo, pretende-se adaptar uma metodologia de análise de redes complexas aplicado para a análise de resiliência diferentes desenhos conceituais de sistemas térmicos. Tal análise fornece uma maneira formal para calcular as métricas relacionadas com a resiliência do sistema, de tal modo que seja possível prever, em fases iniciais, qual alternativa de projeto é mais capaz de funcionar como requerido (processo 2016/19255-7).. Situação: Concluído; Natureza: Pesquisa. 2014 - Atual
Desenvolvimento de sistema especialista para a geração automática de superestruturas para síntese e otimização de sistemas térmicos
Descrição: Sistemas térmicos para geração e conversão de energia podem apresentar configurações de alta complexidade, o que torna seu projeto uma atividade inerentemente árdua. Uma abordagem bastante difundida para a síntese de sistemas térmicos é o emprego de técnicas de programação matemática para se obter uma configuração ótima a partir de uma superestrutura mais geral. Uma limitação dessa abordagem é justamente a necessidade de se decidir a priori quais alternativas tecnológicas devem compor a superestrutura. A tentativa de se contornar essa limitação pode levar a duas situações indesejáveis, a saber: i) ao decidir por um determinado conjunto de alternativas tecnológicas, encontra-se a solução ótima para esse conjunto, mas corre-se o risco de excluir o conjunto que levaria à verdadeira solução ótima do problema; ii) ao decidir pela inclusão do maior número possível de alternativas na superestrutura para evitar a obtenção de soluções sub-ótimas, corre-se o risco de explosão combinatorial, tornando proibitivo o esforço computacional. Tal limitação faz com que a geração automática de superestruturas seja um campo aberto de pesquisa. Uma breve revisão da literatura mostra que abordagens procedurais, algorítmicas, têm sido propostas para abordar o problema. Na presente proposta pretende-se desenvolver um sistema especialista, uma alternativa heurística e não-procedural, para a geração automática de superestruturas para síntese e otimização de sistemas térmicos. Os resultados devem ser comparados com aqueles obtidos pelas abordagens disponíveis na literatura, subsidiando assim a discussão e a análise crítica das virtudes e limitações de cada método.. Situação: Em andamento; Natureza: Pesquisa. Alunos envolvidos: Mestrado acadêmico: (1) . 2012 - 2014
Desenvolvimento de um sistema especialista para projeto conceitual de centrais de cogeração a biomassa
Descrição: A diversificação da matriz energética nacional se mostra importante no atual momento do Brasil. É uma estratégia adequada o desafio de se aumentar a oferta de energia a custos compatíveis com um modelo de desenvolvimento no qual se exige sustentabilidade e minimização de impactos ambientais. Neste cenário, centrais de cogeração têm forte apelo, em particular em setores industriais onde resíduos do processo produtivo podem ser utilizados como combustível. O projeto de uma central deste tipo é um problema de síntese sujeito a restrições termodinâmicas, econômicas e ambientais e que inclui a alocação e o dimensionamento de componentes de modo a satisfazer as demandas elétricas e térmicas. Tais problemas são intrinsecamente complexos devido à possível existência de inúmeras alternativas que atendam um determinado conjunto de requisitos de projeto. A natureza heurística do problema sugere sua solução através de sistemas especialistas, uma das técnicas de inteligência artificial mais bem sucedidas e já aplicada em outros domínios de conhecimento, tais como projetos de sistemas hidráulicos, transformadores elétricos, rodovias, entre outros. O objetivo da presente proposta é desenvolver um sistema especialista para projeto conceitual de centrais de cogeração a biomassa, no qual sejam integradas ferramentas de metodologia de projeto adequadas para este domínio de conhecimento. O resultado final é uma ferramenta computacional inteligente, capaz de, a partir dos requisitos de projeto, projetar por si mesma conceitos para uma central de cogeração, justificando as principais decisões tomadas (Processo 2011/19099-1)..
