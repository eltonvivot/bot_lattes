2010 - 2012
eGuia
Descrição: O projeto tem como objeto, o desenvolvimento de um sistema para guiar (facilitar o acesso provendo autonomia) deficientes visuais no sistema de transporte urbano, qual será chamado e-Guia. Para tanto, permitindo que o deficiente visual possa de forma autônoma através do uso de um aparelho celular identificar com antecedência (100 metros antes) quando o transporte coletivo que o mesmo está aguardando se aproxima do ponto. O projeto e-Guia é dividido sobre três perspectiva, do usuário, da parada (ponto onde os usuários aguardam o transporte) e do transporte. Nestas perspectivas, o projeto e-Guia será divido em três módulos: Módulo do Usuário, Módulo da Parada e Módulo do Transporte, observando que para cada módulo serão desenvolvidos diversos subsistemas, os quais contemplarão todo o sistema e-Guia.. 2004 - 2006
Projeto de Infra-estrutura para pesquisa: Sistema Supervisório para Monitoramento Remoto de Leitos Hospitalares
Descrição: É proposto o desenvolvimento de um sistema para monitoramento remoto de leitos hospitalares, baseado na implementação de um sistema dedicado, composto por sensores com "inteligência" incorporada, smart sensors, um computador com programas para gerenciamento destes sensores, supervisório, e uma estrutura em rede para proporcionar a conexão e composição remota dos sinais adquiridos. Esta tríade (sensores-supervisório-conexão em rede) constitui o proposto Sistema Supervisório para Monitoramento Remoto de Leitos Hospitalares (SSMRLH). Um eventual desdobramento deste sistema possibilitará seu uso como sentinela domiciliar, em que kits dedicados de baixo custo irão monitorar sinais eletrofisiológicos de indivíduos em seus respectivos lares, locais de trabalho ou onde praticam atividades rotineiras, enviando, através da linha telefônica, dados com utilização do protocolo TCP/IP em tempo real. Estes dados tratados irão gerar um relatório corrente dos sinais do indivíduo, também em tempo real, dando a um especialista um retrato da condição do usuário, o que pode ser realizado também pelo programa supervisório, que possui algoritmos baseados em inteligência artificial para traçar o diagnóstico..
