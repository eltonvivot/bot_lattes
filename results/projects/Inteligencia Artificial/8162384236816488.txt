2017 - 2020
The Adoption of OMG Standards in the Development of SOA-Based IoT Systems
Descrição: A common feature of the Internet of Things (IoT) is the high heterogeneity, regarding network protocols, data formats, hardware and software platforms. Aiming to deal with such a degree of heterogeneity, several frameworks have applied the Model-Driven Development (MDD) to build IoT applications. On the software architecture viewpoint, the literature has shown that the Service-Oriented Architecture (SOA) is a promising style to address the interoperability of entities composing these solutions. Some features of IoT make it challenging to analyze the impact of design decisions on the SOA-based IoT applications behavior. Thus, it is a key requirement to simulate the model to verify whether the system performs as expected before its implementation. Although the literature has identified that the SOA style is suitable for addressing the interoperability, existing modelling languages do not consider SOA elements as first-class citizens when designing IoT applications. Furthermore, although existing MDD frameworks provide modeling languages comprising well-defined syntax, they lack execution semantics, thus, are not suitable for model execution and analysis. This work aims at addressing these issues by introducing IoTDraw. The framework provides a fully OMG-compliant executable modeling language for SOA-based IoT systems; thus, its specifications can be implemented by any tool implementing OMG standards.. Situação: Concluído; Natureza: Pesquisa. Integrantes: Bruno Carlos da Cunha Costa - Coordenador / Paulo de Figueiredo Pires - Integrante / Flavia Coimbra Delicato - Integrante.

ArqIoT: Um Ambiente para Desenvolvimento Orientado por Arquitetura de Aplicações em Internet das Coisas
Descrição: O objetivo desse projeto é prover um ambiente orientado por arquitetura para o desenvolvimento de aplicações para Internet das Coisas. Esse ambiente é formado por dois níveis: (i) o nível arquitetural: composto por uma linguagem de descrição de arquiteturas, SysADL, baseada em SysML, contendo suporte a descrição estrutural, comportamental e de execução da arquitetura; (ii) nível da plataforma de Internet das Coisas: composto pela plataforma EcoDiF. 2012 - 2014
Projeto Flexível de Aplicações em Nuvem
Descrição: Processos de desenvolvimento de software tradicionais, tais como o Rational Unified Process (RUP), fornecem diversas visões como contexto (Casos de Uso), dinâmica (Sequência, Atividades e Estados), implantação (Componentes), dentre outras. Contudo, tais processos não definem uma visão para o projeto e implantação de aplicações que utilizam o modelo de PaaS, fazendo com que a implantação do software seja feita de maneira ad-hoc, ou semelhante a implantação em um modelo fora do contexto de PaaS. Isto se mostra como um problema, pois sem um controle de elasticidade na aplicação a cobrança é feita conforme as decisões de escalonamento do fornecedor de serviços, além de retirar do arquiteto da aplicação a decisão de prioridade de escalonamento de componentes em um domínio. O objetivo deste trabalho é propor modelos de projeto e implantação aplicado ao paradigma de Computação em Nuvem. O modelo de projeto irá propor uma visão de Elasticidade, enquanto que o modelo de implantação definirá uma visão de deploy de componentes conforme as definições de Elasticidade presentes no modelo de projeto. Os modelos serão aplicados nas fases de projeto e implantação da aplicação, conforme o paradigma de desenvolvimento baseado em componentes. Desta forma, este projeto propõe uma perspectiva de Engenharia de Software aplicada ao domínio de Computação em Nuvem, denominada Engenharia de Nuvem (Cloud Engineering). Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Mestrado acadêmico: (1) . Integrantes: Bruno Carlos da Cunha Costa - Integrante / Paulo de Figueiredo Pires - Coordenador.Financiador(es): Centro de Pesquisa e Desenvolvimento Leopoldo Américo Miguêz de Mello - Bolsa.
2012 - 2013
Integração de dispositivos físicos a Web para instrumentação de Cidades Inteligentes
Descrição: A rápida evolução da tecnologia de hardware e software tem ocasionado uma mudança expressiva no ambiente urbano. Há uma crescente instrumentação de áreas urbanas com sensores interconectados por redes sem fio que fornecem diversos dados sobre o ambiente. O desenvolvimento de sistemas capazes de tirar proveito das novas tecnologias de informação e comunicação e que façam uso da massa de dados disponível, agregando-lhes significado e integrando-as de uma forma sensível ao contexto em que são produzidos, será capaz de prover novos níveis de inteligência urbana, viabilizando a concepção de Cidades Inteligentes (smart cities). O objetivo principal deste projeto é desenvolver uma infra-estrutura de suporte para construção de aplicações para cidades inteligentes, centrada na publicação de informações oriundas de fontes heterogêneas de dados e no consumo dessas informações a partir da Web. Dentre as fontes de informações, serão incluídas, além de bases de dados digitais tradicionais, aquelas oriundas de sensores de instrumentação de ambientes urbanos. Nesse contexto, este projeto visa desenvolver uma infraestrutura de software, baseada no paradigma da WoT, para construção de aplicações que agreguem valor contribuindo para a concretização do conceito de cidades inteligentes no Brasil. Situação: Concluído; Natureza: Pesquisa.
