2009 - 2010
ProCCSys ? Um Sistema Web de Controle de Protocolo de Câncer Colorretal
Descrição: O avanço tecnológico possibilita o armazenamento de grandes volumes de dados. Isso se observa na área de saúde, onde são reunidos dados de pacientes em formato digital. A análise manual desse volume de dados é difícil e custosa e, assim, o uso de métodos computacionais para a extração de padrões e construção de modelos que representem os dados, auxiliam os especialistas na tomada de decisão. Neste trabalho é apresentado o protótipo de um sistema computacional Web para a construção de bases de dados com informações de pacientes com câncer colorretal, o qual representa um dos cânceres de maior incidência no mundo. O sistema finalizado pode ser utilizado em hospitais e clínicas para constituir um repositório integrado de informações dessa doença.. Situação: Em andamento; Natureza: Pesquisa. 2009 - 2010
EnDia 2.1: Um Sistema para Construção de uma Base de Dados de Exames de Endoscopia Digestiva Alta
Descrição: O avanço tecnológico tem possibilitado o armazenamento e processamento9 de grandes volumes de dados. Esse fato pode ser observado em diversas áreas, por exemplo, na medicina, em que hospitais e clínicas vêm armazenando informações médicas em formato digital. Entretanto, o acúmulo dessas informações potencializa um problema cada vez mais comum: a dificuldade de analisar e extrair informações úteis de grandes quantidades de dados. Nesse contexto, o sistema Web EnDia 2.1 tem o objetivo de construir um repositório de informações de endoscopia digestiva alta para posterior aplicação de algoritmos de extração de conhecimento sobre essas informações.. Situação: Concluído; Natureza: Pesquisa. 2009 - Atual
Padrões de metadados e ferramentas de digitalização de dados de polinizadores
Descrição: O projeto visa pesquisar e desenvolver ferramentas de TI de apoio ao projeto Conservation and Management of Pollinators for Sustainable Agriculture through an Ecosystem Approach, realizado no Brasil, sob coordenação do Ministério do Meio Ambiente. Além do projeto brasileiro outros dois são realizados em paralelo na África ( QUenia, Uganda e Africa do SUl) e Ásia (Nepal, Paquistão e India) sob a coordenaçãoo global da FAO-ONU e com recursos do Global Environmental Facilities (GEF). Neste projeto pesquisaremos padrões de troca de dados existentes em biodiversidade e proporemos alguns para atender as demandas da comunidade de pesquisa em polinizadores, visando à integração de dados entre sistemas distribuídos. Com base nesses padrões serão desenvolvida ferramentas para digitalização dos dados a serem coletados em experimentos em campo nesse projeto. Elas incorporarão recursos para melhoria da qualidade dos dados digitalizados. Um portal do projeto também será desenvolvido não só para divulgar os resultados mas também pa servir de ferramenta de colaboração e interação entre os participantes de mais de uma dezena de instituições só no Brasil..
