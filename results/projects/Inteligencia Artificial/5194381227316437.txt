Projeto Edital MCT/CNPq14/2009-Universal -FaixaB-DeR$20.000,01aR$50.000,00
Descrição: Titulo: Desenvolvimento de Novos Métodos de Aprendizagem de Máquina e Suas Aplicações a Problemas de Engenharia de Software
Resumo: O uso de software por pessoas, empresas e governos tem crescido continuamente. As empresas de software necessitam produzir software com alta qualidade e no prazo estimado para assegurar a competitividade no mercado. O tamanho e a complexidade do software aumentam constantemente e as empresas se esforçam para que seus projetos sejam concluídos no prazo previsto. Conseqüentemente, as áreas de planejamento (incluindo estimativa de esforço de projetos), gerência e desenvolvimento de software se tornaram de suma importância para o sucesso de projetos de software. Outra área muito importante é a área de teste de software, que é também fundamental para garantir a qualidade do software.
Este projeto abordará dois problemas da área de Engenharia de Software. Para cada um dos dois problemas abordados, serão propostas novas técnicas de aprendizagem de máquina especificamente projetadas para eles, será feita uma comparação através de diversas simulações entre as técnicas propostas e técnicas de aprendizagem de máquina tradicionais e será desenvolvida uma ferramenta de software com as técnicas de aprendizagem de máquina consideradas mais adequadas (no estudo comparativo) para o problema investigado.. 2007 - 2009
Projeto Edital MCT/CNPq 02/2006 - Universal
Descrição: Titulo: Ferramenta para Estimativa de Esforço de Projetos de Software Baseada em Técnicas de Computação Inteligente.
Resumo: A estimativa precisa do custo e do prazo de projetos de software no início de seu ciclo de vida (ainda na fase de levantamento de requisitos) é fundamental para a indústria de software. Diversos projetos de software foram interrompidos devido a atrasos no prazo e, conseqüentemente, devido ao custo real do projeto (ainda incompleto) ter superado significativamente o custo estimado no início do mesmo. A estimativa imprecisa do custo e do prazo de projetos de software contribui fortemente para a interrupção de projetos de software. Estas interrupções geraram prejuízos entre US$ 25 bilhões e US$ 75 bilhões na economia dos Estados Unidos nos últimos cinco anos. Portanto, o desenvolvimento de técnicas e a construção de ferramentas para a estimativa precisa do custo e do prazo de desenvolvimento de projetos de software é um problema bastante relevante para a economia dos países, que são cada vez mais dependentes de software. 
O problema da estimativa de esforço de software pode ser tratado como um problema de regressão. A partir do tamanho estimado do software (em número de linhas de código) e de outras métricas de desenvolvimento de software pode-se construir um modelo de regressão usando dados de projetos concluídos. A partir daí, o modelo construído é usado para estimar (prever) o esforço em homens-hora de novos projetos de software. O COCOMO tem sido o modelo mais utilizado na prática para esta estimativa. No entanto, recentemente alguns trabalhos mostraram que técnicas de computação inteligente, em particular, técnicas de aprendizagem de máquina, tais como redes neurais artificiais (RNAs), são capazes de fornecer estimativas mais precisas que as do modelo COCOMO. Há diversas técnicas recentes de aprendizagem de máquina que tem obtido melhor desempenho que modelos mais antigos de RNAs em diversos problemas práticos. Em trabalho recente nós. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2)  / Mestrado acadêmico: (3) .
