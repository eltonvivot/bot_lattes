2015 - 2020
Linha de Processos de Software de Acesso Público
Descrição: O desenvolvimento do projeto envolve as seguintes atividades: (i) coletar, junto as empresas de desenvolvimento de software selecionadas, as suas necessidades específicas com relação aos processos aplicados em seus projetos de desenvolvimento de softwares; (ii) definição da linha de processos de software, decorrente da compilação dos resultados das necessidades reveladas pelas empresas que aceitarem colaborar com o projeto; (iii) modelagem da referida linha de processos de software com a ferramenta EPF Composer; (iv) implementação do mecanismo de derivação, a partir do EPF Composer, de um processo de desenvolvimento de software customizado, dada a seleção de um conjunto características necessárias ao mesmo; (v) integrar o processo de derivação desenvolvido com o sistema Web a ser disponibilizado ao público geral; (vi) desenvolver e testar o sistema Web que disponibilizará ao público geral o acesso a linha de processos de software desenvolvida nestes projeto e (vii) realizar entrevistas com os representantes das empresas selecionadas para realizar os testes com o sistema desenvolvido, identificando como se deu o uso desse sistema e a aplicação do processo gerado automaticamente pelo sistema..
