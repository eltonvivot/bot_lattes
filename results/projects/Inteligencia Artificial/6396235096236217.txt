2019 - 2021
Aumento de Dados Textuais para Aprendizagem de Máquina
Descrição: Para conseguir bons resultados, técnicas de aprendizagem de máquina geralmente precisam de uma grande quantidade de dados. Entretanto, a obtenção do volume de dados necessário pode ser muito custosa ou inviável.
Com isso, técnicas para treinamento de modelos de aprendizagem de máquina que exijam uma quantidade menor de dados têm sido investigadas. Em visão computacional, o aumento de dados, que consiste em gerar dados artificiais baseados num conjunto pequeno de dados existentes, é uma das técnicas mais difundidas para expandir os dados de treinamento.
Por outro lado, em processamento de linguagem natural, o aumento de dados ainda é pouco explorado. Assim, este projeto de pesquisa tem como objetivo pesquisar, analisar e propor técnicas de aumento de dados textuais, para descobrir como e quanto essas técnicas podem melhorar tarefas de processamento de linguagem natural.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2) . Integrantes: Yuri de Almeida Malheiros Barbosa - Coordenador.

2018 - 2020
Análise de sentimentos para língua portuguesa em redes sociais
Descrição: As redes sociais possuem bilhões de usuários em todo planeta. Nelas, os usuários costumam compartilhar variados tipos de conteúdos, incluindo suas opiniões sobre eventos, filmes, produtos, etc.
Com isso, tal riqueza de informações acessíveis publicamente tem despertado o interesse de pesquisadores e empresas que desejam analisar as respostas dos usuários a estímulos diversos e assim conseguir entender melhor o que eles estão expressando e sentindo.
Este projeto de pesquisa tem como objetivo criar e avaliar uma aplicação para analisar automaticamente os sentimentos em mensagens textuais em português compartilhadas em redes sociais.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (3)  / Especialização: (0)  / Mestrado acadêmico: (0)  / Mestrado profissional: (0)  / Doutorado: (0) .
