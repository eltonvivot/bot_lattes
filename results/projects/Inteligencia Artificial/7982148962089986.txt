2020 - Atual
Uma investigação sobre uso de Requirements Smells para Verificação de Qualidade de Requisitos Ágeis em Linguagem Natural
Descrição: O objetivo deste projeto é investigar o uso de ?Requirements Smells? como boas práticas para especificação de requisitos em processos ágeis. Para tanto, será analisado se o catálogo proposto é eficaz para a verificação da qualidade de requisitos baseado nos critérios de qualidade para requisitos ágeis, apresentados na literatura. Caso o catálogo não seja capaz de cobrir todos os critérios de qualidade para requisitos ágeis, será proposto melhorias para este catálogo. Em seguida artefatos para especificação de requisitos em linguagem natural serão analisados de acordo com o catalogo proposto.. Situação: Em andamento; Natureza: Pesquisa. 2018 - Atual
Documentação de Requisitos e Rastreabilidade de Artefatos para Evolução Sustentável de Sistemas Computacionais
Descrição: A diversidade das características das pessoas interessadas em um sistema é substancial a julgar os interesses envolvidos. Assim, a documentação e a rastreabilidade dos artefatos do ciclo de vida do sistema computacional exercem papel preponderante ao seu aceite e se seu uso será perene, considerando o dinamismo e a necessidade de adaptação imperativa de todo e qualquer sistema frente às demandas dos interessados. Ponderando isto, e utilizando técnicas adequadas, esse projeto intenciona oferecer alternativas eficazes e eficientes à indústria de desenvolvimento de sistemas computacionais para a criação e a evolução da documentação e da rastreabilidade dos artefatos de um sistema computacional, no âmbito da área de Engenharia de Requisitos.. 2016 - Atual
Aplicação de Ferramenta de Rastreabilidade em ambientes ágeis para desenvolvimento de software
Descrição: Metodologias Ágeis estão cada vez mais presentes na indústria. Conhecida como uma metodologia que se adapta facilmente a mudanças, que diminui riscos e proporciona a criação do produto de maneira rápida e segura, possui características como: constante contato com cliente, geração de releases ao final de cada iteração, priorização no que tem mais valor para o cliente, além de ?abraçar? facilmente as mudanças solicitadas. Sendo rastreabilidade de Requisitos importante quando pensamos em mudanças, pois facilita atividades como análise de impacto e sendo possível perceber que esta seria uma atividade bastante útil no processo ágil. No entanto, existem alguns desafios ao tentar aplicar a rastreabilidade em ambientes ágeis. Assim, este trabalho tem o objetivo de aplicar e ou ajustar ferramentas existentes para contribuir na rastreabilidade de requisitos em projetos de desenvolvimento de software que atuem em ambientes ágeis. Para isso serão levados em consideração um conjunto de funcionalidades levantadas em trabalhos anteriores para serem consideradas em ferramentas com este objetivo. Como forma de validação da ferramenta proposta um estudo de caso usando a ferramenta será desenvolvido.. Situação: Em andamento; Natureza: Pesquisa. Alunos envolvidos: Mestrado acadêmico: (3) . Integrantes: Marcia Jacyntha Nunes Rodrigues Lucena - Coordenador / Gabriela Trindade - Integrante / Erica Miranda - Integrante / ERITON DE BARROS FARIAS - Integrante.
2010 - 2011
Uma Abordagem de Mapeamento de Modelos de Requisitos para Modelos Arquiteturais através de MDD
Descrição: Engenharia de requisitos e projeto arquitetural são atividades chave para o desenvolvimento de sistemas de software. Ambas as atividades estão altamente entrelaçadas e interligadas. No entanto, muitos passos para gerar modelos arquiteturais a partir de modelos de requisitos são baseados na intuição e conhecimento do arquiteto. Desta forma, abordagens sistemáticas que integram atividades de engenharia de requisitos e de projeto arquitetural são necessárias. Assim, este projeto de pesquisa tem como objetivo desenvolver uma abordagem relacionando modelos de requisitos e modelos arquiteturais no contexto desenvolvimento orientado a modelos.. Situação: Concluído; Natureza: Pesquisa. Integrantes: Marcia Jacyntha Nunes Rodrigues Lucena - Coordenador / Lyrene Silva - Integrante / Apuena Gomes - Integrante.
