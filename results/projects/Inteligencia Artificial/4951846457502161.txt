2021 - Atual
Evolução de Arquitetura de Software em Projetos de Software de Código Aberto
Descrição: Projetos de software de código aberto bem sucedidos podem servir de referencial de
boas práticas em Engenharia de Software. O número de participantes e a forma atuante das suas respectivas comunidades na proposição, tratamento e implementação de melhorias e correção de bugs e code smells, dentre outras práticas, são alguns dos critérios que podem ser utilizados para classificar tais projetos como bem sucedidos. Uma das formas de se avaliar o impacto destas boas práticas nestes projetos é a forma como sua arquitetura evolui ao longo do tempo. Para esta finalidade, neste projeto são utilizadas abordagens de aprendizagem de máquina para avaliar esta evolução à luz das boas práticas adotadas ao longo das releases de projetos de código aberto bem sucedidos.. Descrição: O projeto tem o objetivo de caracterizar o impacto da pandemia do COVID
19 nos serviços de atenção à saúde no sistema prisional de Salvador. Para tanto será utilizado

2017 - 2019
The Influence of ISO/IEC 25010 Characteristics on Bug Fixing in Open Source Software Projects
Descrição: The influence of specific ISO/IEC 25010 characteristics on the bug fixing process of successful open source software projects is an important phenomenon to understand the rationale behind the decisions that drive these projects over releases. For example, Security can influence the choice of which software bugs to prioritize, when to fix it and in which future release the corresponding fixed code should be available for users. It is expected that contributers from the software community would prefer to vote and focus on bugs related to relevant characteristics for the project over the others. This project has the goal to analyze to which extent the relevance to fix bugs and implement new functionalities in OSS projects is influenced by a subset of the ISO/IEC 25010 characteristics.. Um Toolkit de Referência para o Desenvolvimento de Ambientes Interativos baseados em Múltiplas Visões
Descrição: O objetivo deste projeto é propor e avaliar experimentalmente um toolkit para apoiar o desenvolvimento de ambientes interativos baseados em múltiplas visões (AIMVs).
