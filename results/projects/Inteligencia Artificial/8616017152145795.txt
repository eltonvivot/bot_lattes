2006 - 2008
Modelos de scheduling utilizando estruturas hibridas CLP-MILP
Descrição: O objetivo principal deste projeto é o desenvolvimento de estruturas de modelagem baseadas nos
paradigmas da lógica inferencial que permitam fazer uma integração entre as técnicas de pesquisa
operacional baseadas em MILP e as técnicas oriundas da Inteligência Artificial/Linguagem de
programação conhecidas como CLP. Estas estruturas devem permitir a tradução de restrições
matemáticas contendo zeros e uns em restrições lógicas e vice versa, facilitando o desenvolvimento de
modelos. Em seguida, estas estruturas devem ser usadas para solucionar problemas combinatoriais, em
especial aqueles relacionados ao scheduling dos arranjos produtivos.
O produto final deste projeto será a formulação de um procedimento que permita implementar modelos
híbridos (MILP-CLP) computacionalmente eficientes e com garantias de factibilidade e/ou otimalidade da
solução.  (Edital Universal CNPq - 2006). Situação: Em andamento; Natureza: Pesquisa. Alunos envolvidos: Graduação: (1)  / Mestrado acadêmico: (1)  / Doutorado: (1) . SIGA: Sistema Informatizado para Gerenciamento de Alinhamentos
Descrição: Este projeto trata do desenvolvimento da ferramenta (SIGA - Sistema Informatizado para Gerenciamento de alinhamentos) que auxilia os operadores de refinaria na escolha de rotas, sugerindo rapidamente várias possibilidades de percursos e uma lista de tarefas necessárias para que a transferência seja feita corretamente. 1999 - 2003
Arquiteturas e Metodologias para Supervisão de Sistemas Híbridos de Produção
Descrição: In this project two models for hybrid systems are studied and developed. The first one is a based on Petri
net which integrates the continuous and dicrete parts of a process. So it is a hybrid model allowing the simulation of
both aspects of a process behavior. The models allow the visualization and simulation of the industrial plants, helping
the system operator to easily test and correct scheduling plans despite the complexity of the operations. This model
is used to simulate Liquefied Petroleum Gas (LPG) transfer and storage operations in a refinery. The second model
is a formal model to a network which is characterized by hybrid automata. The network components are modeled and
represented by independent hybrid automata and the global dynamics is obtained through the product hybrid
automata. Languages representing the desirable states of the system are obtained from the hybrid automata,
allowing a hybrid control procedure. An automatic tool as SHIFT must be used to modeling and simulation. This
formal specification technique is used to model a transfer system integrating a natural gas distribution network. Situação: Concluído; Natureza: Pesquisa. Aplicação de redes neurais na identificação de sistemas dinâmicos com perturbação limitada
Descrição: : Este projeto desenvolve uma metodologia baseada em redes neurais para estimação robusta de modelos lineares quando o erro de modelagem é limitado. Uma rede de Hopfield modificada é desenvolvida e usada na obtenção de uma região de incerteza para os parâmetros do modelo. Os parâmetros internos da rede são derivados utilizando o conceito de sub-espaço válido de soluções, que além de garantir a convergência e a estabilidade da rede, permite a descrição politópica exata da região de pertinência paramétrica, ou ainda sua aproximação por um ortotopo. Através de exemplos simulados, o desempenho da rede é comparado a de outros algoritmos robustos para cálculo destas regiões politópicas. Situação: Concluído; Natureza: Pesquisa. Desenvolvimento de um ambiente CAD baseado em conhecimento para projeto de sistemas de controle de processos petroquímicos
Descrição: : Alguns processos industriais são caracterizados por requisitos de ambiente e desempenho que não podem ser tratados através de metodologias clássicas de controle. Um exemplo aplicativo (protótipo para simulação e controle de uma coluna fracionadora de óleo pesado) é descrito, a fim de ilustrar as potencialidades do ambiente proposto..
