2023 - Atual
Optimização Discreta no interior de um Hiper-Elipsoide no espaço real n-dimensional
Descrição: Uma das maiores dificuldades enfrentadas ao tentar resolver um problema de programação não linear inteira (PNLI) é não poder usar o conceito de derivada para determinar o crescimento da função objetivo envolvida, pois o conjunto de soluções factíveis é discreto. A questão é a seguinte: dado um ponto interior factível a um PNLI, como determinar a partir do ponto dado um novo ponto interior ao PNLI que melhore a função objetivo? Um caso interessante a se considerar é quando o conjunto viável do PNIL é finito, como é, por exemplo, o PNLI com apenas uma restrição e a função objetivo linear de coeficientes positivos. Neste projeto visa desenvolver e aplicar um algoritmo na resolução de problemas de optimização não linear discreta onde deseja maximizar uma função linear no interior de um hiper -elipsoide no espaço n-dimensional com variáveis inteiras. Situação: Em andamento; Natureza: Pesquisa. Alunos envolvidos: Graduação: (1) . Integrantes: Eleazar Gerardo Madriz Lozada - Coordenador / Camilo Rocha - Integrante / Mauricio Romero Sicre - Integrante / Nelson Maculan - Integrante.

Avaliação da fluência e competências digitais dos educadores: um estudo no Brasil para formação de professores em uma sociedade digital
Descrição: A crescente inserção das Tecnologias Digitais de Informação e Comunicação (TDIC) no processo educativo tem impulsionado novas formas de ensinar e aprender em todos os níveis de ensino. Diante do surgimento de uma geração hiperconectada que se movimenta de forma fluída e híbrida nos espaços virtuais e presenciais, impõem-se aos educadores o desenvolvimento de competências digitais, visando a utilização das TDIC para se alcançar os objetivos pedagógicos desejados. 2009 - 2011
Estudo de Estruturas Algébricas Finitas na Resolução de Problemas de Programação Linear Inteira.
Descrição: Na área da Programação Matemática existe uma subárea chamada Programação Linear Inteira (PLI), que resolve problemas específicos onde as variáveis de decisões não podem ser mensuradas por números reais, senão por números inteiros, restrição que gera um alto grau de complexidade quando a questão é procurar métodos para resolver este tipo de problema.  As técnicas mais utilizadas pelos pesquisadores desta área do conhecimento envolvem heurísticas e meta heurísticas. O PLI pertence a categoria dos problemas para os quais não existe um algoritmo  em tempo polinomial que o resolva no modelo de maquina não determinística de Turing, categoria conhecida como as dos problemas NP-dificil.  Nesta área, uma das metodologias existente esta fundamentada no estudo do poliedro associado ao PLI, metodologia onde o poliedro é transformando, usando alguma Estrutura Algébrica Finita (EAF) ( grupos, semigrupos ou sistemas aditivos) para logo aplicar a teoria de polaridade de poliedros visando gerar faces do poliedro, e assim obter cortes que podam ser usado para definir o poliedro associado ao PLI. Existem muitos trabalhos na literatura onde os problemas de PLI tratados são problemas que seu poliedro está definido sobre todos os elementos das EAF's. Em geral podemos dizer que usa-se, discretamente, resultados teóricos relacionados com EAF?s visando o desenvolvimento de heurísticas, meta heurística o algoritmos para gerar faces dos poliedros associados aos problemas de PLI. Neste projeto serão estudado teoricamente as EAF?s visando definir e analisar algoritmos, heurísticas ou meta heristicas que permitam desenvolver soluções computacionais para problemas reais que possam ser modelados como um PLI.. Situação: Concluído; Natureza: Pesquisa. Integrantes: Eleazar Gerardo Madriz Lozada - Coordenador.
