2009 - 2010
BASE DE DADOS SOBRE PRODUTOS FLORESTAIS NÃO-MADEIREIROS E O DESENVOLVIMENTO SUSTENTÁVEL DE RONDÔNIA
Descrição: Este projeto de pesquisa tem por objetivo a criação de uma base de dados sobre produtos florestais não-madeireiro e desenvolvimento sustentável do estado de Rondônia. A criação desta base de dados deverá oferecer às populações da Amazônia rondoniense alternativas de geração de trabalho e renda a partir do extrativismo florestal não madeireiro, protegendo a floresta e oferecendo condições para que seus habitantes sobrevivam. O desenvolvimento do presente trabalho deverá contemplar as seguintes etapas: levantamento dos dados (tipos, localização, formas de coleta, utilização, rentabilidade, etc..) sobre produtos não-madeireiros do estado de Rondônia; modelagem e implementação da base de dados utilizando ferramentas de software livre; disponibilização da base de dados através da Internet. A base de dados deverá estar em constante atualização para que reflita a realidade local e, a qualquer tempo, forneça respostas condizentes com a realidade. Espera-se, como resultados deste projeto, a caracterização dos produtos não-madeireiros proveniente do extrativismo florestal do estado de Rondônia, através de uma base de dados disponível a partir do IFRO ? Campus Ji-Paraná com informações iniciais sobre alternativas para o desenvolvimento sustentável na região, que poderá ser continuamente alimentada por pesquisas posteriores ou interação com outras instituições interessadas.. Situação: Concluído; Natureza: Pesquisa.
