Reconfiguração Dinâmica em Sistemas de Automação e Controle
Descrição: A reconfiguração dinâmica tem se tornado um requisito chave para sistemas de automação flexíveis. Ela permite que o sistema seja reconfigurado em hardware e software em tempo de execução. Algumas linguagens de programação e a norma IEC 61499, que substitui a norma IEC 61131, já possibilitam a alteração da lógica de controle em tempo de execução. Entretanto, não há garantias de funcionamento, o que pode gerar inconsistências no sistema ou tornar o processo de reconfiguração ineficiente. O objetivo deste projeto é desenvolver um método para execução da reconfiguração dinâmica, o qual ofereça garantias quanto ao funcionamento e consistência do sistema.. Atualmente a tecnologia de simulação interativa de usinagem a controle numérico se baseia fortemente no uso de polígonos para representação dos modelos 3D, porém tal técnica pode apresentar limitações na qualidade de visualização do modelo. possível, onde a visualização de superfícies é feita usando ray tracing e sem a conversão do modelo 3D para polígonos.Esse avanço tecnológico leva a um aumento significativo da qualidade da imagem final, e com isso diminuindo a diferença visual entre o modelo simulado e a peça usinada. Este trabalho fará uma exploração das principais técnicas modernas de ray tracing e estrutura de dados para simulações de usinagem. Após a fase exploratória, o projeto dará início a fase de modelagem e design da arquitetura do software, para então fazer o desenvolvimento iterativo e incremental. 2011 - 2016
Uso de STEP-NC e IEC61499 em CNC e Integração da Manufatura
Descrição: Este projeto tem como objetivo o desenvolvimento de um Comando Numérico Computadorizado (CNC) aderente às normas ISO14649, conhecidas como STEP-NC (STandard for the Exchange of Product data for Numerical Control), e à IEC61499 conhecida como Blocos de Funções (Function Blocks ou FBs). Como premissa este terá interface aberta e será interoperável, reconfigurável e portável. A ISO14649 provê um modelo para troca e compartilhamento de dados entre projeto e manufatura além de permitir a programação de máquinas de maneira genérica, ou seja, sem códigos proprietários. A IEC61499 é uma norma emergente para uso em processos industriais distribuídos e sistemas de controle e que permite encapsular algoritmos que controlam partes do processo. Este trabalho continua pesquisas anteriores realizadas pelo proponente com objetivo de investigar o uso em conjunto dessas normas internacionais, visando à construção de uma nova geração de CNCs inteligentes e garantir a interoperabilidade entre vários dispositivos no processo industrial. Em especial, o foco deste trabalho será em utilizar a STEP-NC como um ?compositor de trabalho?, provendo todas as informações necessárias e os blocos de funções como um ?realizador de trabalho?, executando os comandos de usinagem para manufatura interoperável. Até o momento as pesquisas foram realizadas utilizando micro-controladores e motores de passo.  Para testes e validação deste novo trabalho, um protótipo de máquina será desenvolvido onde serão utilizados servo motores e servo acionamentos com Controladores Lógicos Programáveis(CLPs) incorporados. Espera-se explorar as funcionalidades de hardware e software presentes nos servo acionamentos e CPLs e expandir em muito as capacidades dos protótipos iniciais da pesquisa que eram máquinas construídas com motores de passo e micro-controladores.. 2010 - 2013
Orientação de robôs industriais por sistemas de visão
Descrição: Este trabalho busca resolver um problema real encontrado em várias empresas e que no caso será estudada a solução dentro de uma indústria em Joinville. Trata-se uma grande empresa de fundição que vem utilizando uma grande quantidade de robôs industriais na área de produção de blocos de motores para a indústria automobilística.
O problema que deverá ser estudado é o uso de sistemas de visão para encontrar a posição e orientação das peças para a pega do robô. Entre as dificuldades está o fato de as peças serem diferentes (blocos de motores de vários tipos). Além disso a posição e orientação dos blocos tem que ser conhecida para que ocorra a pega por parte do Robô. A solução convencional é o uso de dispositivos mecânicos chamados indexadores, para posicionar as peças e aí então ocorrer a pega. Estes dispositivos mecânicos não dão flexibilidade ao sistema robótico pois, são construídos para um ou alguns tipos/tamanhos de peças, de forma bem restrita. Uma forma de resolver o problema é garantir que o sistema tenha a informação do tipo, posição e orientação da peça a ser pega. Hoje isto é um problema para a indústria a indústria pois, em alguns processos, diferentes tipos de peças tem que ser manipulados pelo mesmo robô gerando problema para o sistema realizar a tarefa com uma programação estática.
A solução proposta é o uso de sistemas de visão integrado via interface à programação do robô permitindo o posicionamento correto do atuador em tempo real, para diferentes peças em diferentes posições. Assim o Objetivo Geral do trabalho é  Posicionar e orientar o efetuador final de um robô industrial por meio de um sistema de visão interferindo dinamicamente na programação. Ou seja: fazer navegar o efetuador final, com base em informações atualizadas via sistema de visão. A interface entre o sistema de visão e o robô é um grande desafio, pois se hoje a robótica e os sistemas de visão são tecnologias dominadas, a integração entre eles ainda é alvo de estudos. Requisitos para armazenamento em XML de modelo de informação de manufatura baseado nas normas ISO10303 e ISO14649
Descrição: A linguagem XML constitui um padrão aberto para troca de informações entre sistemas computacionais. Neste projeto, XML será o formato escolhido para armazenar modelos de informação de manufatura baseados nas normas ISO10303 e ISO14649.  A ISO10303 no seu protocolo 28 (ISO/TS10303-28 2002) provê a forma uso de XML como formato de documento para troca/compartilhamento de informações e conhecimento utilizando EXPRESS. EXPRESS é uma linguagem de especificação de dados usada para descrever tipos de dados e restrições em informação de representação de dados a serem exportados ou compartilhados. A norma ISO10303 provê vários métodos de implementação que podem ser empregados por dados descritos por um esquema em EXPRESS.
A norma ISO10303 é, hoje, consolidada e em uso na indústria de software de engenharia. Já as normas da série ISO14649 estão atualmente em plena fase de desenvolvimento no contexto de manufatura. Por tal razão, em breve, serão necessários modelos de informação coerentes e integração entre sistemas CAD, CAPP, CAM e CNC baseados nessas normas.  O projeto analisa o possivel uso de XML para criar uma infra estrutura de informacoes de manufatura a serem compartilhadas por sistemas CAD/CAE/CAPP/CAM/CNC. Situação: Concluído; Natureza: Pesquisa.
