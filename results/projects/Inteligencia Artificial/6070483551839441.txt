2016 - 2017
PIBIC Aplicações de Métodos Heurísticos de Otimização em Ambientes Paralelos CPU/GPU
Descrição: Este projeto procura explorar os aspectos inerentes ao
desempenho de algoritmos de otimização heurísticos orientados para
ambientes heterogêneos compostos por CPUs e GPUs. Para atingir tal
objetivo, pretende-se realizar um estudo detalhado da arquitetura GPU
e seu relacionamento com o tradicional modelo de paralelismo orientado
para CPU, buscando destacar os requisitos relevantes ao desempenho dos
componentes que representam o maior impacto na performance de
algoritmos heurísticos. Para validar essa abordagem pretende-se lidar
com problemas de otimização reais por meio de aplicações selecionadas
dentre diversas áreas do conhecimento, notadamente aquelas que
possibilitem a integração com outros projetos de pesquisa
desenvolvidos nesta IES.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (12)  / Doutorado: (3) . Integrantes: Sérgio Barros de Sousa - Integrante / Francisco das Chagas Rocha - Integrante / Eyder Franco Sousa Rios - Coordenador / Cristina Boeres - Integrante / Luiz Satoru Ochi - Integrante / Rodrigo Baluz - Integrante / Átila Rabelo Lopes - Integrante / Igor Machado Coelho - Integrante / Ricardo Farias - Integrante / Aleph Breno Melo Rodrigues da Silveira - Integrante / Matheus Aranha Silva - Integrante / Pablo Henrique da Silva - Integrante / Alexandre Rosa Carvalho - Integrante / Athyrson Machado Ribeiro - Integrante / Filipe Alves Sampaio - Integrante / Isaac Bruno Siqueira de Souza - Integrante / Nayra Beatriz Paiva de Oliveira - Integrante / Francisco Augusto dos Santos Calaça - Integrante.
