2010 - 2010
Estratégia de testes de unidade para o desenvolvimento de sistemas de informação: O caso do projeto SIGA-EDU
Descrição: O SIGA-EDU é um projeto que tem o objetivo de planejar e desenvolver um sistema integrado de gestão acadêmica com tecnologias livres e de forma colaborativa para os IFET´s. O SIGA-EDU lida com um sistema grande e complexo, com várias tecnologias distintas para o desenvolvimento web, onde há a necessiadde de desenvolver uma estratégia de testes de unidade para o projeto considerando sua arquitetura e características, assim como escolher um arcabouço adequado para realizar testes de unidade. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (3) .
