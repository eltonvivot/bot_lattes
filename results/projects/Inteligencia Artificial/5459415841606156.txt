DESENVOLVIMENTO DE UM SISTEMA DE VIDEO LOTERIA
Descrição: OBJETIVO
Desenvolvimento de Jogo interativo para máquinas de divertimentos eletrônicos (tipo video-loteria). 
PROPOSTA
Desenvolvimento de uma plataforma para jogos interativos. Para tal, o projeto possui uma plataforma composta pelos seguintes módulos:
·	Desenvolvimento de software de raspadinha eletrônica
·	Desenvolvimento do software do gerador aleatório configurável, baseado nas exigências do CODESC.
·	Desenvolvimento da Interface de Hardware.
·	Manual de utilização do software e do gerador aleatório. Situação: Em andamento; Natureza: Pesquisa.
