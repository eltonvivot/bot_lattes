2022 - Atual
A New Model for the Edge Intelligence
Descrição: This work proposes the study of a hierarchical Intelligence model for merging the best characteristics of the artificial intelligence models, that is, do not share user data and enable training from a tiny amount of labeled data. Also, the initial proposed architecture and model were developed based on current state-of-the-art articles. Nevertheless, the edge data quality may avoid unnecessary computing and allows a more generic model for discrete and continuous data. Thus, we include a learning hierarchy of Machine Learning execution processes to find patterns between inter domains to achieve the best rewards. Finally, the proposal will integrate the human-in-the-loop system to improve learning abstraction, reduce technological gaps, and enable advanced decision-making technology. The project also includes technology transference between partnership enterprises, such as Nvidia, through comprehensive technical training.. Situação: Em andamento; Natureza: Pesquisa. 2012 - 2017
Big Data Processing using Hybrid Infra-structures as Cloud Computing and Desktop Grids
Descrição: A comunidade científica tem proposto Cloud Computing como uma infraestrutura para Big Data e o gerenciamento de grandes bases de dados não estruturadas, distribuídas em sistemas de arquivos e bases de dados NoSQL, como uma solução de armazenamento. Contudo, os provedores de Cloud, em aplicações de análise de Big Data, ainda necessitam explorar as alocações de recursos de uma maneira efetiva to encontrar economia de escala e elasticidade. Devido a grande variedade de fontes de dados, as bases de dados coletadas tem diferentes ruídos, transformações, filtragem de dados e mecanismos de depuração para remover ruídos e corrigir insistências. o Trabalho para fazer análise em Big Data é ainda um desafio intensivo. Os sistemas emergentes são altamente heterogêneos com estruturas variáveis, onde os recursos podem se adicionados ou removidos se isto for necessário. Desktop grid é uma plataforma que tem recebido atenção da comunidade científica para o uso em aplicações Big Data. Este trabalho explora as infraestruturas híbridas para encontrar as melhores opções para a alocação de recursos heterogêneos em sistemas Big Data.. Situação: Concluído; Natureza: Pesquisa. A finalidade é poder implementar soluções de forma a tornar sistemas Big Data mais seguros e tolerantes a falhas para evitar a degradação de desempenho do sistema..
