Uso de múltiplos pipelines para redução dimensional do conjuntos de dados em um modelo de predição com Machine Learning aplicado a uma base de dados clínicos de pacientes chagásicos.
Descrição: O presente projeto tem o objetivo de combinar diversas técnicas de seleção de atributos (feature selection) em diferentes pipelines para redução dimensional de um conjunto de features em uma base de dados clínicos de pacientes chagásicos. A finalidade é tentar diminuir, através de um metodologia criteriosa, o número de features para encontrar o melhor modelo de predição aumentando sua estabilidade, eficácia, sua eficiência ou seu custo computacional.. 2020 - 2020
FULTBLIND Um estudo com pessoas com deficiência visual para a criação de um aplicativo que facilite a prática do futebol de 5.
Descrição: O presente projeto consiste em encontrar os requisitos necessários para a construção de um aplicativo mobile que facilite a prática do futebol de 5 por pessoas com deficiência visual. Serão feitos testes de viabilidade onde o objetivo será colher dados no que diz respeito a forma como eles interagem no aplicativo, os quais servirão de base para a sua implementação. Esse teste consistirá em uma simulação de uma partida de futebol de 5 para encontrar as principais dificuldades que os jogadores enfrentam ao praticar esse esporte. A partir daí iremos propor uma forma de como superá-las.. Situação: Concluído; Natureza: Pesquisa. Uso de Machine Learning para predição de morte súbita em pacientes com Doeças de Chagas
Descrição: O presente projeto tem o objetivo realizar a predição de morte para paciente portadores de doença Chagas baseado em features de sinais de ECG através do uso de algoritmos de aprendizagem de máquina. Este trabalho deve servir de base para desenvolvimento de futuras aplicações que possam auxiliar os médicos, além de possibilitar acesso e potencializar sua popularização em locais que ainda não está acessível devido a restrições financeiras.. 2016 - 2017
Automatizando
Descrição: Os Sistemas de Informação (SI) podem ajudar, recebendo, agrupando, transformando e disponibilizando dados e informações, além de fornecer suporte e automatizar os processos gerenciais nas empresas, facilitando e agilizando às tomadas de decisões dos gestores. Nesse sentido, este estudo irá evidenciar a utilização do uso dos SI nos empreendimentos turísticos da cidade de Canindé como ferramentas de auxílio na gestão desses estabelecimentos. O estudo pressupõe uma abordagem quantitativa, através de uma pesquisa com coleta de dados, através da aplicação de questionários e entrevistas informais. Espera-se que a realização desse trabalho possibilite detectar o nível de gestão e o conhecimento do uso dos SI pelos gestores nos empreendimentos pesquisados. Dessa forma, poderá se pensar em políticas de capacitação para melhorar a eficiência da gestão das empresas, evitando falências, desempregos e consequentemente contribuindo com a economia local..
