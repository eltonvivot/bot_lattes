2009 - 2011
Utilização de Técnicas de Identificação Caixa Cinza e de Redes Neurais Artificiais para a Análise e Síntese de controladores
Descrição: Aos mecanismos para se incorporar informação a priori na identificação de sistemas dinâmicos não-lineares é um problema ainda em aberto. Outro aspecto que pode ser considerado em aberto é a identificação para propósito de controle. Quando o objetivo é somente a sintonia de controladores, ou o teste de diferentes estratégias de controle, o modelo é usado apenas para simular o sistema dinâmico real. Entretanto, outra opção é o uso de modelos identificados para desenvolver estratégias de controle baseados em modelos. Boa parte destas estratégias se baseiam na curva estacionária do sistema, e cabe ao controlador, neste caso, a tarefa de linearizar tal curva estacionária, de tal forma que em malha fechada o sistema se comporte como um sistema linear. O que se propõem neste projeto de pesquisa é o desenvolvimento de técnicas de controle a partir de modelos identificados através de processos de seleção de estruturas. Neste caso, o que se procura é a definição de métodos para escolhas de estruturas não-lineares de controladores a partir dos modelos identificados. A motivação básica deste projeto de pesquisa é o de aprimorar técnicas até aqui desenvolvidas para incorporar informação a priori em representações NARMAX racionais, polinomiais e neurais. Outro objetivo considerado de fundamental importância é o desenvolvimento de procedimentos de validação de modelos para propósito de controle. Nesse sentido o desenvolvimento de um projeto de pesquisa voltado para identificação e controle de processos, é de grande importância para o desenvolvimento regional, pois muitas melhorias implementadas em processos industriais só são possíveis a partir de modelos matemáticos. Melhorias nestes modelos, que poderiam representar significativos ganhos na qualidade dos produtos.. Situação: Em andamento; Natureza: Pesquisa. 2007 - 2009
Identificação Caixa-Cinza Não-Linear Aplicada na Análise e Controle de Processos Industriais
Descrição: Existe muito a se desenvolver na identificação caixa-cinza. Um aspecto importante é o desenvolvimento de mecanismos capazes de incorporar o maior número de informações {\it a priori} do processo. Por outro lado, faz-se necessário também integrar cada vez mais procedimentos de identificação com desenvolvimento de controladores. Acredita-se que a identificação orientada para o desenvolvimento de controladores não-lineares pode representar um importante passo no sentido de melhorar o desempenho dos processos industriais. Por outro lado, quando se objetiva aplicar as ferramentas em processos industriais, faz-se necessário desenvolver ferramentas de identificação em malha fechada. Nos últimos anos tem se observado grande desenvolvimento de ferramentas caixa-cinza para modelos NARMAX polinomiais, porém estes avanços ainda são bastantes incipientes quando se trata de redes neurais. Acredita-se que muito dos resultados obtidos para modelos NARMAX polinomiais podem ser estendidos para redes neurais. Portanto considera-se de grande importância os aspectos apresentados nos objetivos deste projeto.. Situação: Em andamento; Natureza: Pesquisa. 2005 - Atual
Incorporação de Informação Auxiliar no Desenvolvimento de Controlador e na Identificação de Sistemas Não Lineares Baseados no Estudo de Caso em uma Planta de Neutralização de pH.
Descrição: A obtenção de modelos matemáticos não lineares a partir de dados de entrada e saída observados em sistemas dinâmicos é uma ciência
que adquiriu importância nas últimas décadas. Recentemente, a possibilidade de incorporar informações auxiliares neste processo
de identificação vem recebendo atenção. Das diversas razões para o uso de informações auxiliares pode-se destacar: (i) uma
alternativa para seleção de estrutura de modelos não-lineares, (ii) possibilidade de se obter modelos não-lineares com estrutura
mais simples, (iii) obter modelos capazes de representar o sistema em regiões não visitadas pelos dados de identificação. Alguns
aspectos ainda estão em aberto e precisam ser investigados, dentre estes pode-se citar o uso de informações auxiliares para obtenção
de modelos para operação em malha fechada e/ou para o desenvolvimentos de controladores. Por outro lado, o sistema de
neutralização de pH vem sendo usado como paradigma para identificação e controle de processos não lineares. O seu uso se
justifica pela curva estática fortemente não linear e, devido a isto, pela dificuldade de controle. Neste projeto de pesquisa
propõe-se utilizar uma planta piloto de neutralização de pH como meio de desenvolver ferramentas para estimação da curva estática
de sistemas não lineares e para o desenvolvimento de controladores. Ao final desta pesquisa pretende-se (i) ter desenvolvidos procedimentos para incorporar informação auxiliar na modelagem de sistema reais e que estes mecanismos possibilitem
obter modelos cujo comportamento seja global mesmo quando os dados dinâmicos de identificação estão visitando apenas um restrita
faixa de operação e (ii) propor mecanismo de controle não linear baseados em modelos identificados.. Situação: Em andamento; Natureza: Pesquisa.
