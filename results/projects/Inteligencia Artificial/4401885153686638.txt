LARA - Laboratório Remoto em AVA
Descrição: O LARA é um projeto da UESB que tem com objetivo desenvolver uma arquitetura pedagógica que integra recursos tecnológicos e metodologia de ensino para melhorar o processo de aprendizagem de disciplinas do curso de Ciência da Computação, como: programação de computadores, algoritmos, sistemas inteligentes, robótica, etc... 2014 - Atual
Métodologia Pedagógica para utilização do LARA
Descrição: O objetivo deste projeto é desenvolver uma metodologia para uso educacional do AVA de acordo com a teoria sócio-interacionista para disciplina de linguagem de programação utilizando laboratório remoto de robótica. O LARA - Laboratório Remoto de Robótica é um conjunto de ações que visam o desenvolvimento de um ambiente com hardware, software para o ensino de programação, principalmente nos cursos de Computação.Noenatnto, é preciso definir uma estrutura pedagógica que contribua para com construção de conceitos científicos pertinentes ao curso de Ciência da Computação. Situação: Em andamento; Natureza: Pesquisa. Alunos envolvidos: Graduação: (4) . Integrantes: Alzira Ferreira da Silva - Coordenador / Maisa Soares dos Santos Lopes - Integrante / Roque Mendes Prado Trindade - Integrante.Financiador(es): Universidade Estadual do Sudoeste da Bahia - Outra.
