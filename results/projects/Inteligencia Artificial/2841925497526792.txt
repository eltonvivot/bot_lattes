UNIFESP - Projeto de Treinamento Orientado em Ressuscitação Pediátrica (TORPed)
Descrição: Projeto de Treinamento Orientado em Ressuscitação Pediátrica (TORPed). O objetivo desse projeto é divulgar os conhecimentos especializados para a prática adequada de ressuscitação pediátrica. Objetivos Específicos: Treinamento dos Suportes de Vida Pediátricos; desenvolver competência com simulações de casos. Método: O método de treinamento é configurado através dos seguintes passos: Disponibilização da apostila e realização do pré-teste, uma semana antes do dia do treinamento; Dia do Treinamento e avaliação prática, durante 1 dia; Realização do pós-teste até uma semana após o dia do treinamento; Acesso à ficha de desempenho global, depois do pós-teste. Página: http://telemedicina.unifesp.br/set/projetos/torped.. Situação: Em andamento; Natureza: Pesquisa. USP - Desenvolvimento e Implementação de Arquitetura Distribuída para o Sistema Lepidus Usando Serviços Web
Descrição: O sistema de apoio à decisão médica Lepidus foi desenvolvido para auxiliar profissionais qualificados da área de saúde a diagnosticar doenças a partir de sinais e sintomas clínicos. Apesar do sistema oferecer excelente resposta (cerca de 84% de acerto), a tecnologia empregada para seu desenvolvimento dificulta experimentações científicas e seu uso em situações reais. Ainda mais, não permite que o acesso ao sistema seja realizado por aplicativos automatizados para que se possa realizar estatísticas ou mesmo interoperar com outros sistemas de informação em saúde (SIS). Somente acesso manual foi considerado. Este projeto tem por objetivo desenvolver e implementar uma arquitetura distribuída que ofereça um excelente nível de interoperabilidade para o sistema Lepidus através do uso de tecnologias de serviços web (WebServices). Também se pretende implementar um sistema de gerenciamento de usuários e de publicação de interfaces de seus serviços para que parceiros autorizados possam usufruir da funcionalidade do sistema Lepidus independentemente da linguagem e ambiente de desenvolvimento, sistema operacional, desempenho do computador e localização geográfica. A avaliação de interfaces adequadas a diferentes grupos de usuário está prevista. Este projeto insere-se na linha de pesquisa apoiada pela FAPESP (processo no. 2001/08681-0)..
