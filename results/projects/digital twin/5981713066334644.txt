2023 - Atual
Desenvolvimento de um conjunto inovador de bobinas de controle de ELMs para o Tokamak TCABR
Descrição: Este projeto de pesquisa tem como objetivo desenvolver um conjunto inovador composto por 108 bobinas para controle de ELMs para o TCABR, o qual está sendo modernizado para poder continuar fornecendo resultados de interesse ao desenvolvimento da fusão nuclear para fins de produção de energia. O projeto físico conceitual dessas bobinas será realizado por meio de simulações do acoplamento dos campos com o plasma utilizando um código desenvolvido no Princeton Plasma Physics Laboratory baseado em um modelo MHD não linear resistivo de dois fluidos chamado M3D-C1. No entanto, dadas as condições extremas em que essas bobinas deverão operar, o projeto eletro-termo-mecânico e a própria construção desse sistema são de grande complexidade. Portanto, o objetivo dessa proposta é também desenvolver as tecnologias críticas necessárias para que a implementação completa da modernização do TCABR com a instalação desse sistema de bobinas possa ser planejada em bases sólidas.. 2019 - Atual
Desenvolvimento de Metodologia Expressa para Análise de Tensão em Armaduras de Pressão de Dutos Flexíveis
Descrição: O objetivo do projeto é desenvolver uma metodologia expressa para o cálculo das tensões em pontos críticos dos perfis das armaduras de pressão de tubos flexíveis genéricos, uma vez conhecidos os esforços que solicitam a seção transversal deste tubo e implementar essa metodologia em uma ferramenta computacional. Os resultados fornecidos por essa ferramenta devem ser verificados
experimentalmente e certificados por um agente independente. Neste contexto, o projeto estabelece como objetivos específicos:
? o desenvolvimento de modelos de elementos finitos parametrizados para representar o comportamento estrutural da armadura de pressão de tubos flexíveis;
? o desenvolvimento de metodologia expressa de análise de tensão para representar o comportamento estrutural da armadura de pressão;
? o desenvolvimento de ferramenta computacional para implementar a metodologia para implantação em ferramenta de análise de fadiga;
? avançar no desenvolvimento de modelos estruturais com validação, focando o comportamento da armadura de pressão, componente pouco estudado analiticamente, e que vem apresentando indícios de falhas por fadiga em operação.. Desenvolvimento de um Modelo Analítico-Numérico para a Avaliação da Instabilidade de Armaduras de Dutos Flexíveis sob Carregamentos Combinados
Descrição: O projeto visa desenvolver modelos para a previsão de instabilidade das armaduras helicoidais de tubos flexíveis quando submetidos a carregamentos combinados, envolvendo os seguintes tipos de esforços atuantes na seção transversal: compressão, momento fletor, torção, pressão interna e pressão externa..
