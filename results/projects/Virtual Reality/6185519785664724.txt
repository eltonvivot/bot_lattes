2020 - Atual
Compreendendo o Papel dos Eventos Colaborativos de Curta Duração na Economia Digital
Descrição: Enquanto já existem estudos sobre a motivação, preparação e colaboração em eventos colaborativos de curta duração, o conhecimento sobre as atividades que ocorrem após os eventos ainda é limitado. Mais do que isto, até onde sabemos, não existem estudos endereçando o impacto destes eventos na economia digital. Este projeto visa justamente endereçar este lacuna, assim, a pergunta de pesquisa deste projeto é: ?Qual o impacto dos eventos colaborativos de curta duração no desenvolvimento de novos negócios?? Esta pergunta será respondida sob diferentes perspectivas. A primeira e mais importante é a continuidade dos negócios iniciados durante os eventos. Como mencionado, resultados iniciais (Gama, 2017b) sugerem que a maior dos projetos são descontinuados, porém estes resultados são de escopo limitado, já que os dados da pesquisa são referentes a eventos de caráter cívico, que não necessariamente visavam a geração de negócios voltados para a economia digital. Pretende-se estudar as razões para esta descontinuidade, bem como identificar mecanismos para lidar com estas razões, além de tentar compreender se isso é um padrão que se repete em hackathons voltados para geração de negócios. A segunda perspectiva é sob a ótica das redes de contatos estabelecidas durante os eventos, visto que é possível que negócios inicialmente criados durante os eventos não tenham continuidade, porém as conexões estabelecidas nos eventos possam levar a outros negócios igualmente relevantes. A terceira perspectiva visa avaliar a continuidade na utilização de abordagens (métodos, modelos, ferramentas, conceitos, etc) que foram inicialmente utilizadas durante os eventos. 2018 - 2019
Monitoring Coastal Areas with Mapathons and Hackathons
Descrição: Geographic information in coastal areas is essential for the development of the so-called ?blue economy?, i.e. the sustainable use of marine and ocean resources for economic growth and improved livelihoods in coastal areas. In developing countries, such as South Africa, such information is often lacking, with huge implications for the blue economy.  This project aims to address this situation by conducting hackathons and mapathons for carefully selected areas of the South African coast. Such intensive efforts have been shown to be enormously useful for bringing out creativity and co-ordinating the human capital of a large number of participants from interdisciplinary backgrounds. Mapathon outputs need to be high quality to be reusable, but often, when applications are later built on top of map data, there is a mismatch between data collected and application requirements. 
We propose to address these challenges by creating a circular process that starts from an intensive mapping effort to create new data for unmapped coastal areas. We will then develop apps that use this data in a hackathon, and feed back the difficulties and successes to a second mapathon that will focus on improving the usability of the data for app development. We will develop metrics to analyse the quality of the data collected and use this to quantify how the maps improve from their initial state to the first iteration of the mapathon, and from the first mapathon to the second mapathon.  
The goal of this project is to create and refine this process of scalably collecting high-quality map data, and  brainstorm ways to  improve the coastal economy by improving map data and geospatial applications.. IoT-Flows: Lightweight Policy Enforcement of Information Flows in IoT Infrastructures
Descrição: Dispositivos utilizados para Internet das Coisas (IoT) em geral são equipamentos com recursos limitados para barateamento de custos. São assim, vulneráveis a diversos riscos de segurança. Técnicas tradicionais de detecção de intrusão (IDS  Intrusion Detection Systems) e/ou de anomalias do tráfego podem ajudar a identificar potenciais comprometimentos de segurança. Este projeto tem como objetivo realizar uma defesa cross-layer com análise dos aplicativos IoT e software dos dispositivos, extração de políticas de fluxos a partir da análise do tráfego normal, e uso destas políticas para garantir os fluxos desejados e detectar violações às mesmas nas camadas de rede e do dispositivo. Este projeto foi aprovado na Chamada Conjunta RNP-NSF para Projetos de Pesquisa e Desenvolvimento em Segurança Cibernética, com a participação dos pesquisadores Darko Marinov da University of Illinois at Urbana-Champaign (UIUC, US), Atul Prakash (UMich, US) e Ann Arbor (UMich, US).. 2015 - 2020
Melhores Práticas e Serviços para o Desenvolvimento do Ecossistema de Dados Abertos da Cidade de Recife
Descrição: Nos últimos anos, governos de diversos países têm demonstrado grande interesse em disponibilizar seus dados em formato aberto com o intuito de oferecer maior transparência aos cidadãos. A ideia principal desta iniciativa consiste em oferecer dados de forma livre e pública na Web, a fim de que os cidadãos possam ter fácil acesso aos dados produzidos pelo governo. Abrir dados públicos fortalece instituições, garante cidadania e controle social, combate a corrupção, promove a transparência, a fiscalização e a geração de ideias criativas em políticas públicas pela própria sociedade. Dados Abertos Governamentais, ou seja, dados produzidos pelo governo e disponibilizados em formato aberto, podem ser facilmente encontrados nos chamados portais de Dados Abertos, os quais oferecem uma interface para catalogação e acesso aos dados. A fim de alcançar o seu pleno potencial econômico, uma iniciativa de Dados Abertos deve fazer parte de um ecossistema composto, principalmente, por provedores e consumidores de dados que colaboram para promover a iniciativa. O ecossistema de Dados Abertos facilita a geração de novos negócios por meio da criação de sistemas e aplicativos inovadores voltados para a melhoria da produtividade das empresas existentes ou instituições governamentais, bem como para a melhoria do bem estar dos cidadãos. A efetiva criação e consolidação de um ecossistema de Dados Abertos depende, fundamentalmente, de uma boa comunicação entre os diferentes atores participantes do ecossistema. Dessa forma, no centro do ecossistema encontram-se uma plataforma de TI e um conjunto de melhores práticas, os quais têm como função facilitar a interação entre os diferentes atores, além de viabilizar a criação em larga escala de produtos e serviços inovadores, por meio da utilização de normas, padrões e ferramentas comuns. Este projeto tem como objetivo geral contribuir para o desenvolvimento do ecossistema de Dados Abertos da cidade de Recife por meio da especificação de melhores práticas e implementação de uma plataforma de Dados Abertos como serviço que facilitem a interação entre os diferentes atores participantes do ecossistema. É importante destacar que os resultados obtidos com este projeto podem ser aplicados em outras instâncias governamentais, tanto municipais quanto estaduais. Tanto as melhores práticas quanto a plataforma de Dados Abertos como serviço podem ser usadas para o desenvolvimento e consolidação do ecossistema de Dados Abertos de outras cidades do estado de Pernambuco, contribuindo, dessa forma, para o desenvolvimento do ecossistema de Dados Abertos do estado. 2015 - Atual
Técnicas de Reestruturação de Software para Melhoria de Eficiência Energética em Aplicações para Cidades Inteligentes
Descrição: O uso de TICs em cidades inteligentes impõe uma grande demanda de energia e conectividade que estão diretamente ligados à crescente necessidade de produzir e consumir grandes massas de dados. A coleta de dados nesse cenário tipicamente se dá através de sensores físicos e por crowdsensing, i.e., através de cidadãos munidos de dispositivos como smartphones. Este cenário de constante produção de dados envolve diferentes demandas energéticas até então inexistentes: de um lado, aplicativos de smartphones recolhem dados (e.g, geolocalização, pressão, aceleração) enquanto estão nos bolsos de seus proprietários, além de estimularem interações explícitas do usuário para coletar e informar dados, necessitando de maior processamento do dispositivo móvel e uso da rede. Do outro, a grande produção de massas de dados gera a necessidade de efetuar a análise e correlação destes, visando a extração de informação relevante. Este projeto de pesquisa tem por objetivo desenvolver e estender métodos, técnicas e ferramentas para melhorar a eficiência energética de aplicações que dão suporte a cidades inteligentes. Mais especificamente, pretende-se investigar técnicas para melhorar a eficiência energética de dois tipos de aplicação: aplicações móveis, muitas vezes rodando em dispositivos dependentes de baterias, como smartphones, e aplicações para servidores com alto poder computacional em execução em data centers, responsáveis pelo processamento de grandes massas de dados. Para os dois cenários, planeja-se lançar mão de técnicas de reestruturação de software. Situação: Em andamento; Natureza: Pesquisa. Alunos envolvidos: Mestrado acadêmico: (5) Doutorado: (5) .
