2018 - Atual
Sistema De Irrigação Automatizado Com Placa Arduino
Descrição: Este projeto tem como objetivo criar um sistema de irrigação automatizado de baixo custo utilizando a placa Arduino. Tendo em vista que o município de Capitão Poço é um dos maiores produtores de frutas cítricas e pimenta da região, a colheita e plantação é oriunda tanto do agronegócio como também da agricultura familiar. Pequenos agricultores não possuem recursos suficientes para custear e manter sistemas complexos de irrigação inteligente, assim também, estes sistemas geram um grande gasto para a área do agronegócio. Como uma alternativa de baixo custo, visando principalmente o agricultor de pequeno porte, este projeto busca implementar um sistema de irrigação inteligente utilizando a placa computacional Arduino com sensores específicos de temperatura, humidade, entre outros.. Situação: Em andamento; Natureza: Pesquisa. Alunos envolvidos: Graduação: (4)  / Especialização: (0)  / Mestrado acadêmico: (3)  / Mestrado profissional: (0)  / Doutorado: (1) .
