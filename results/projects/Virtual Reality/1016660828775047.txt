Desempenho Biomecânico de Parafusos Pediculares em Cirurgia de Artrodese de Coluna Lombar
Descrição: Objetivo geral
O objetivo do presente trabalho é avaliar, do ponto de vista mecânico, se os modelos de parafusos existentes 
no mercado conseguem manter a estabilidade vertebral após a secção pedicular junto ao corpo vertebral. 
Objetivos específicos
Os  objetivos  deste  trabalho  podem  ser  detalhados  segundos  três  aspectos  ou  áreas  de  interesse: 
Estabilidade, Design, Novo Implante.
?. Demonstrar quanto da estabilidade vertebral é perdida ao se seccionar o pedículo vertebral;
?. Identificar qual o melhor projeto (design) de parafuso, aplicado à técnica proposta no trabalho em questão;
?. Identificar a necessidade ou não de um novo implante para que esta técnica seja oferecida a pacientes com

2014 - 2019
Projeto, Fabricação e Testes de um novo VANT Hibrido com Aplicação no Controle de Fitopatologias
Descrição: O projeto apresenta o desenvolvimento de uma aeronave do tipo VANT com caracteristicas inovadores. Nele são exploradas as caracteristicas de uma aeronave hibrida: Asa Fixa e Asa Rotativa e  todo o ciclo de desenvolvimento. O projeto começa com a coleta de requisitos da missão com os stakeheolders. Modelos são desenvolvidos e Simulados para se obter informações importantes sobre aeronave. Dessa forma, novos requisitos de desempenho podem ser obtidos. Com esses novos requisitos um projeto detalhado da aeronave é executado. Após essa etapa a aeronave é Manufaturada/montada localmente. Por fim a aeronave é testada realizando as missões inicialmente estabelecidas no inicio do projeto. A teoria de controle e suas novas ferramentas para controle não linear também são exploradas bem como aplicação dessa teoria para o desenvolvimento de estratégias e algoritmos de controle não linear.Além disso, estratégias para o projeto de sistemas mecatrônicos são avaliados.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (2)  / Mestrado acadêmico: (1) . Integrantes: Alex Sandro de Araújo Silva - Coordenador / Antonio Ronaldo Gomes Garcia - Integrante / Elmer Rolando Llanos Villareal - Integrante.
