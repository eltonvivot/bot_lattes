2011 - 2013
Interação 3D: Dispositivos, Métodos e Aplicações (Ed. Universal CNPq)
Projeto certificado pelo(a) coordenador(a) Alberto Barbosa Raposo em 21/05/2013.
Descrição: Interação 3D (3DUI - 3D User Interaction) é definida como o uso de uma interface humano-computador onde a linguagem usada pelo usuário para transmitir comandos e informações para o computador, e/ou a linguagem usada pelo computador para transmitir informações ao usuário são baseadas no espaço físico e suas dimensões. No contexto de 3DUI, estamos vivenciando um momento repleto de novidades em relação a dispositivos de interação multi-toque, reconhecimento de gestos, projeção 3D, etc. Por outro lado, o uso de interação 3D é ainda nascente, sem massa crítica, e muito longe de uma padronização, havendo relativamente pouco conhecimento sobre os melhores métodos e técnicas para se implementar e avaliar essa interação, bem como pouco conhecimento sobre como as aplicações atuais poderão se beneficiar da interação 3D. Este projeto pretende estudar este tema por três caminhos complementares: o desenvolvimento de dispositivos de 3DUI, o estudo de métodos para implementar e avaliar tais interações, e o desenvolvimento e adaptação de aplicações que façam melhor uso da 3DUI..
