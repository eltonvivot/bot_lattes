DESENVOLVIMENTO DE JOGOS SÉRIOS PARA AUXÍLIO À REABILITAÇÃO PÓS ACIDENTE VASCULAR CEREBRAL (AVC)
Descrição: Esse projeto foi aprovado na CHAMADA PÚBLICA 11/2020 - PROGRAMA PESQUISA PARA O SUS: GESTÃO COMPARTILHADA EM SAÚDE - PPSUS EDIÇÃO 2020/2021 (FUNDAÇÃO ARAUCÁRIA-PR / SESA-PR / DECIT / SCTIE/MS / CNPq). Sua execução envolve a parceria entre a PUCPR, UFPR e o Complexo Hospitalar do Trabalhador (CHT), especificamente com Hospital de Reabilitação (HR). O objetivo geral desse projeto é desenvolver um sistema com jogos sérios do tipo ?exergames? para auxílio à reabilitação motora pós-AVC que permita o monitoramento do desempenho e qualidade de movimentos do paciente para acompanhamento e tomada de decisões terapêuticas. Para atingir esse objetivo geral, foram estabelecidos os seguintes objetivos específicos: Desenvolver protótipo de jogo sério do tipo ?exergame? tendo por base um sistema de rastreamento corporal (?body tracking?) ; Desenvolver protótipo de sistema vestível (?wereable system?) com sensores para coleta de dados de movimento relevantes para a reabilitação pós-AVC; Avaliar a confiabilidade dos dados de movimento fornecidos pelo sistema de ?body tracking? e pelo sistema vestível; Avaliar a aplicabilidade do prótipo de jogo sério em um serviço de reabilitação pós AVC...
