2002 - 2010
O projeto Cnossos - Micro Kernel P2P distribuído para Grid computing
Descrição: O projeto Cnossos visa o desenvolvimento de um Micro Kernel P2P distribuído para Grid computing onde clientes requisitam mesmo recurso podem se conectar a máquinas totalmente diferentes obtendo uma grande quantidade de serviço. Ele define as funcionalidades que serão utilizadas pelos peers para troca ou/e obtenção de serviços dentro da rede. http://cnossos.nce.ufrj.br. Situação: Concluído; Natureza: Pesquisa.
