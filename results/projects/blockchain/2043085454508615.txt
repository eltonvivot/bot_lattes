2006 - 2007
Projeto Apyon Studio - FASE II
Descrição: Descrição do Projeto:
O Projeto APYON STUDIO - Fase II consistiu no desenvolvimento de um software comercial para tratar regras de negócio e integração com várias ferramentas CASE. 
Dentre os principais objetivos desse projeto destacam-se:
- estudo de metodologias para representação, especificação, armazenamento e implementação de regras de negócio;
- estabelecimento de padrões de regras de negócio;
- desenvolvimento de módulos para tratar regras de negócio, e
- desenvolvimento e implementação de um software comercial voltado às regras de negócio.
