2018 - 2019
Sistema Integrado de Coleta Automatizada de Programas Maliciosos: E-CACURI
Descrição: A grande velocidade com a qual novas versões de programas maliciosos (malware) tem surgido e se espalhado pela Internet, combinada com crescente emprego de técnicas de ofuscação de código, destinadas a ludibriar as ferramentas tradicionais de detecção, têm exigido um contínuo esforço de pesquisadores da área de segurança de sistemas computacionais, na busca e desenvolvimento de novas metodologias de detecção. Desafios como a obtenção de quantidades relevantes de amostras de malware, necessárias para que se possa valiar a eficiência e a eficácia das metodologias propostas e a disponibilidade de um ambiente controlado e seguro para o desenvolvimento de suas pesquisas, são algumas das maiores dificuldades enfrentadas pelos pesquisadores desta área. Este projeto tem como objetivo principal a construção do Sistema Integrado de Coleta Automatizada de Programas Maliciosos: E-CACURI, um ambiente automatizado que possibilite a construção e atualização contínua de uma base de dados contendo amostras coletadas durante ataques a um sistema alvo monitorado (honeynet), bem como a criação de um laboratório de virtualização que possibilite aos pesquisadores a manipulação de programas maliciosos em um ambiente virtual que proteja os recursos computacionais utilizados durante o desenvolvimento de pesquisas na área de segurança de sistemas computacionais. Situação: Concluído; Natureza: Pesquisa. 2015 - 2019
CroMo: Crowd Data in the Mobile Cloud
Descrição: O objetivo principal do projeto CroMo é investigar alternativas para descarregar eficientemente múltiplos dados coletados de usuários móveis para a nuvem. Para alcançar este objetivo, o projeto CroMo foca em três objetivos complementares: aquisição de dados, transmissão dos dados e processamento dos dados. Em um ambiente sem-fio, os dados podem ser originados de uma enorme variedade de dispositivos sem-fio. Assim, o primeiro objetivo deste projeto é de identificar a informação mais relevante entre todos os dados disponíveis, utilizando critérios locais. A noção de localidade pode estar relacionada com um único dispositivo sem-fio ou com um conjunto de dispositivos em uma determinada vizinhança. A ideia é de enviar para a nuvem apenas dados relevantes ou de associar maior prioridade a estes. Um sistema de sensoriamento de larga escala causa transmissões massivas para a nuvem. Assim, a transmissão confiável e tempestiva dos dados é o segundo objetivo do projeto. Nuvens móveis precisam estar disponíveis para receber e processar dados dos usuários sem-fio. O terceiro objetivo deste projeto é avaliar a disponibilidade da nuvem e propor melhorias de desempenho para o processamento dos dados. Estas melhorias estão relacionadas com a adaptação da infraestrutura de nuvem de acordo com as demandas dos usuários. Neste contexto, o projeto é original e ambicioso. Comparado a outros estudos em redes sem-fio, o CroMo foca em uma solução global desde a aquisição de dados brutos até a criação de informação na infraestrutura de nuvem móvel. DARMA: Distribuição e Armazenamento de Informação em Redes Móveis de Larga Escala
Descrição: O projeto DARMA tem por objetivo tratar de diferentes desafios relacionados à transmissão de informação em redes sem-fio, assim como seu armazenamento em centros de dados distribuídos na Internet. Assim, dentro desse escopo, foram escolhidos os seguintes desafios específicos neste projeto: a transferência de dados em aplicações em ambientes de redes veiculares; a combinação de redes veiculares com redes tolerantes a atrasos e desconexões; a utilização de redes tolerantes a atrasos e desconexões em regiões de difícil acesso; a investigação de melhorias de desempenho de redes IEEE 802.11; a iteração das redes elétricas inteligentes com veículos elétricos em cidades inteligentes; a transferência de dados em redes par-a-par corporativas; a robustez a falhas em infraestruturas de centros de dados; e a movimentação de grandes massas de dados entre centros de dados. 2013 - 2015
IMI: Infra-estrutura Móvel Inteligente baseada em Redes Celulares e Veiculares
Descrição: Este projeto propõe o desenvolvimento de infraestrutura de comunicações em ambientes veiculares para suporte de aplicações inovadoras. Tais aplicações possuem dois requisitos fundamentais: elas devem ser voltadas aos grandes eventos concentrados no Rio de Janeiro nos próximos anos e ainda serem capazes de utilizar as oportunidades de contato oferecidas pela presença de redes IEEE 802.11. Este último requisito é especialmente importante, visto que a tecnologia de acesso sem-fio à Internet é, atualmente, as redes de telefonia celular. Esse sucesso é consequência do poder de ubiquidade dessas redes que possui como contrapartida a frequente ocorrência de períodos de congestionamento. Este projeto irá construir uma infra-estrutura de pesquisa em redes veiculares na UFRJ, na qual serão implementadas duas aplicações para a comunidade acadêmica: um serviço de previsão de chegada dos ônibus pela Internet e o acesso à Internet a bordo do trem MagLev-Cobra, cuja primeira linha encontra-se em construção na Ilha do Fundão.. Situação: Concluído; Natureza: Pesquisa.
