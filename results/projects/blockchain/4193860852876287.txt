ARC: Aplicações de Redes móveis em Cidades inteligentes: dos veículos aos centros de dados
Descrição: Atualmente, a Internet está presente em inúmeros ramos de atividades. Redes de sensores, redes elétricas inteligentes, a digitalização de processos são tecnologias de suporte para as Cidades Inteligentes. Este projeto concentra-se na utilização de redes móveis formadas por equipamentos pessoais, como smartphones, e veículos equipados com sensores, para a captura e transporte de informação que permite uma variedade de aplicações de cidades inteligentes. Neste contexto há diferentes desafios em função dos cenários de utilização. Redes sem-fio possuem desafios relacionados à utilização de radiofrequências e um meio de comunicação compartilhado, às redes veiculares, à mobilidade e à conectividade, etc. Além disso, o fato de as redes estarem presentes em cada vez mais atividades traz como consequência um crescimento nunca visto da quantidade de dados a serem transmitidos e armazenados. Neste projeto, são investigadas soluções para alguns dos problemas relacionados à utilização de redes sem-fio móveis em cidades inteligentes e, por outro lado, o transporte das grandes massas de dados até os centros de dados da Internet. Dentro desse escopo, foram escolhidos alguns desafios específicos para este projeto. O primeiro deles é o desenvolvimento de protocolos para a transferência de dados em aplicações em ambientes de redes veiculares. Dado que as redes veiculares serão adotadas progressivamente, outro desafio é como fazer a combinação de redes veiculares com redes tolerantes a atrasos e desconexões (DTN - Delay and Disruption Tolerant Networks), assim como a utilização de DTNs em regiões de difícil acesso. No contexto de cidades inteligentes, as redes veiculares podem ser usadas para aplicações de sensoreamento com nós móveis. Por fim, no contexto de centros de dados, serão estudadas a robustez a falhas em infraestruturas de centros de dados e a movimentação de grandes massas de dados entre centros de dados.. Descrição: A combinação das redes de acesso sem-fio móveis com o interesse dos usuários no conteúdo e não mais na fonte desse conteúdo é um desafio complexo que vem sendo investigado na literatura. Dessa forma, o Projeto MC2 tem como objetivo principal estudar diferentes propostas para melhorar a distribuição de conteúdo em redes móveis, focando particularmente nas seguintes questões: (i) disponibilidade e eficiência da distribuição de conteúdo, levando em conta (ii) a dinâmica da rede móvel e (iii) a organização da nuvem computacional e a sua interação com os usuários. O objetivo é encontrar interações entre esses três pilares que permitam uma melhor oferta de serviços com qualidade aos usuários. Situação: Concluído; Natureza: Pesquisa. CroMo: Crowd Data in the Mobile Cloud
Descrição: O objetivo principal do projeto CroMo é investigar alternativas para descarregar eficientemente múltiplos dados coletados de usuários móveis para a nuvem. Para alcançar este objetivo, o projeto CroMo foca em três objetivos complementares: aquisição de dados, transmissão dos dados e processamento dos dados. Em um ambiente sem-fio, os dados podem ser originados de uma enorme variedade de dispositivos sem-fio. Assim, o primeiro objetivo deste projeto é de identificar a informação mais relevante entre todos os dados disponíveis, utilizando critérios locais. A noção de localidade pode estar relacionada com um único dispositivo sem-fio ou com um conjunto de dispositivos em uma determinada vizinhança. A ideia é de enviar para a nuvem apenas dados relevantes ou de associar maior prioridade a estes. Um sistema de sensoriamento de larga escala causa transmissões massivas para a nuvem. Assim, a transmissão confiável e tempestiva dos dados é o segundo objetivo do projeto. Nuvens móveis precisam estar disponíveis para receber e processar dados dos usuários sem-fio. O terceiro objetivo deste projeto é avaliar a disponibilidade da nuvem e propor melhorias de desempenho para o processamento dos dados. Estas melhorias estão relacionadas com a adaptação da infraestrutura de nuvem de acordo com as demandas dos usuários. Neste contexto, o projeto é original e ambicioso. Comparado a outros estudos em redes sem-fio, o CroMo foca em uma solução global desde a aquisição de dados brutos até a criação de informação na infraestrutura de nuvem móvel.

SLAN: Software Local Area Networks
Projeto certificado pelo(a) coordenador(a) Otto Carlos Muniz Bandeira Duarte em 23/02/2016.
Descrição: Networks have significantly evolved over the last few years. Derived from the cloud computing paradigm, new network virtualization technologies are emerging. These technologies bring agility to the management and the control of infrastructures and services while rationalizing expenditures. They also enable network providers to improve the value-added of their networks and datacenters by offering new services. In complementarity to Software-Defined Networking (SDN), which aims at decoupling the control plane from the data plane and providing network programmability, Network Functions Virtualization (NFV) focuses on implementing network functions in software and deploying them on standard servers or shared infrastructures. NFV thus represents a disruptive change in the way networks are architected and operated. The SLAN project is a research project that leans on the technical expertise of the partners to bring a new paradigm with SLANs (Software Local Area Networks) and concretization impact (inverse of virtualization) to increase performance. The project will provide methods to design the placement of the different virtual machines within the femto-datacenters that form the backbone of future Software Virtual Local Area Networks (SVLAN). Moreover, the project will study how to secure these SVLANs and will show how useful is this paradigm for e-health and mobile networks (e.g., VANET -Vehicular Ad hoc NETworks and MANET ? Mobile Ad hoc NETworks). The project will also address NFV performance and dynamicity. Indeed, it is crucial to measure the utilization of resources and the performance of the services in a software environment. As a summary, the project will investigate, at the local area network level, urbanization techniques to dynamically allocate virtualized network functions. SLAN is an appealing software LAN-oriented project with a strong potential for future networking operations. We expect SLAN results to impact standards and NFV-SDN communities.. Descrição: Os sistemas de comunicação do futuro caminham para um modelo centrado em dados formado por nuvens computacionais. SecFuNet: Segurança para Redes do Futuro (Security for Future Networks)
Descrição: Projeto de cooperação entre o Brasil e a Uniao Europeia na área de Segurança (Chamada FP7-ICT-EU-Brasil). Resumo: A Internet do futuro será baseada principalmente na virtualização de redes e na computação em nuvens. Portanto, um dos maiores desafios da Internet do futuro é prover esta arquitetura de redes virtualizadas e acesso as nuvens com um alto grau de segurança. Este projeto propõe o desenvolvimento de um arcabouço capaz de prover autenticação segura, identificação segura, a transferência segura de dados, a infraestrutura segura de redes virtualizadas e a privacidade em redes virtuais e nuvens. Este desenvolvimento explorará técnicas baseadas em micro controladores, gerência de recursos, algoritmos tolerantes a intrusão e protocolos de criptografia. Este arcabouço será obtido através da elaboração e desenvolvimento de uma arquitetura coerente para redes virtuais e acesso as nuvens. A arquitetura proposta fornecerá soluções que permitam o gerenciamento da segurança das comunicações entre todas as máquinas conectadas a nuvens públicas, através das redes virtuais. Os algoritmos robustos a intrusões são necessários para criar um ambiente seguro. A arquitetura proposta também precisa garantir a segurança na infraestrutura virtual através do isolamento destas redes e do controle de acesso para usuários e gerentes de rede. A identificação dos usuários autorizados não pode, no entanto, comprometer a privacidade dos dados. Além disso, é necessário desenvolver um esquema ergonômico de segurança que seja funcional para todos os usuários, mesmo aqueles que não possuem conhecimentos na área de computação. Finalmente, o esquema proposto deve levar em conta a heterogeneidade dos equipamentos (com ou sem fio), para preservar a interoperabilidade. Situação: Concluído; Natureza: Pesquisa. QUALVOL: Sistemas em Redes de Qualidade para Aplicações Emergentes Geradoras de Grandes Volumes de Dados
Descrição: O enorme sucesso da Internet na sociedade deve-se em parte à vasta gama de serviços e aplicaçõoes que ela atualmente oferece aos usuários. Além disto, mais fundamental ainda é a facilidade com que novos serviços ou aplicações podem ser desenvolvidos e introduzidos na Internet, o que propicia uma constante evolução. A facilidade de introduzir novos aplicativos e serviços é uma consequência natural do modelo original adotado para a arquitetura da infraestrutura da Internet, que dita que o núcleo da rede deve oferecer apenas um serviço básico de conectividade e que as funcionalidades mais complexas devem ser implementadas nas extremidades da rede. Entretanto, esta infraestrutura pode limitar fundamentalmente a eficiência dos aplicativos e serviços que podem ser desenvolvidos, pois para que um aplicativo ou serviço possa ser oferecido de forma eficiente e com qualidade, é necessário que esta infraestrutura atenda minimamente às suas demandas.
Para se oferecer um sistema em redes que ofereça qualidade, em especial as  transmissões de grandes volumes de dados, é necessário que esse possua diferentes propriedades, tais como disponibilidade, acessibilidade, escalabilidade, segurança, ubiquidade e robustez. O objetivo desse projeto é avançar o estado da arte na capacitação da Internet como substrato de comunicação para os novos paradigmas de computação e de comunicação, que por sua vez, permitem aplicaçõoes com demandas crescentes por transferências de grandes volumes de dados.. HORIZON: Um Novo Horizonte para a Internet
Descrição: Projeto binacional selecionado pelo MC/FINEP (Brasil) e ANR (Agence Nationale de la Recherche - França). Resumo: O modelo TCP/IP enfrenta limitações em ambientes sem fio, móveis, ad hoc, RFID e sensores. Essa limitação possivelmente vai aumentar em ambientes sem fio futuros devido às altas vazões de dados, mobilidade em alta velocidade ou limitações de energia (redes de sensores ou RFIDs). Isso pode ser ilustrado por medidas de consumo de energia em redes de sensores: o modelo TCP/IP precisa de muita energia. A baixa eficiência da pilha de protocolos TCP/IP está também provada em um número considerável de redes,
assim que consumo de energia, alta vazão e enlaces de longa distância são previstos. Isso
prova a necessidade de uma nova arquitetura. As propostas dentro desse projeto de pesquisa estão direcionadas a um novo sistema automático de pilotagem que inclui mecanismos inteligentes prontos para compreender o contexto e para adaptar o protocolo à situação corrente. A idéia-chave desse projeto de pesquisa é introduzir uma nova arquitetura inteligente adaptada para redes virtuais. Essa nova arquitetura inteligente deve ser de fácil implementação, escalável, segura, adequada para prover garantias de qualidade de serviço sob demanda e, também, robusta. A escolha da rede virtual a ser usada é decidida dependendo do contexto e do serviço a ser oferecido. Para alcançar o objetivo deste projeto, é usado o plano de pilotagem que permite os elementos da rede definir sua própria visão. Essa visão é usada para decidir os recursos a serem utilizados pelas diferentes redes virtuais. Para construir o plano de pilotagem, um sistema multiagente é empregado. Além disso, um processo de aprendizagem é adicionado para tornar o sistema capaz de pilotar a rede em tempo real. Situação: Concluído; Natureza: Pesquisa.
