2011 - 2012
EVGUI Evolution and Verification of Event-Based GUI Systems
Descrição: Today systems based on user interactions are not limited to the field of desktop and
web applications. They are present in a widely variety of embedded systems such as
cellular phones, TVs, medical devices, cars, etc. The verification of this kind of system
is an approach to assure software quality. A common approach to verify these systems is graphical user interface (GUI) testing. GUI testing is challenging today, due to the high
user expectations as well as increasing complexity of the systems. As graphical interfaces change often during the development process and also in the production environment, we need ways to constantly guarantee high quality systems. This proposal aims to define a foundation for GUI testing and to address the software evolution problem of GUI testing in a software development process.. Situação: Concluído; Natureza: Pesquisa.
