Solucões com Machine Learning para Segurança em Redes Definidas por Software
Descrição: As Redes Definidas por Software (do inglês Software-Defined Networking - SDN) representam um moderno conceito de arquitetura separando o plano de dados e controle da rede. Devido a esta característica, novos desafios de segurança são introduzidos neste modelo de rede em virtude das fragilidades existentes entre os planos. Este projeto de pesquisa explora a utilização de técnicas de Machine Learning - ML para propor mecanismos de segurança em ataques distribuídos de negação de serviço  para SDNs.
Neste sentido, este projeto propõe o uso de métodos estatísticos existentes combinados a técnicas de ML com o objetivo de melhorar a acurácia das estratégias de detecção sem comprometer o desempenho na mitigação de ataques DDoS em SDNs. O uso dessa abordagem resultará no desenvolvimento de técnicas mais eficientes na mitigação de ataques DDoS, minimizando os impactos causados por estes ataques e contribuindo para a melhoria no tráfego de rede SDN.. 2017 - 2020
P4Sec: Securing Networks in the Programmable Data Plane Era
Descrição: Descrição: Recent advances in Software Defined Networking (SDN) have expanded our ability to program the network to its data plane. Through domain specific languages like P4, network operators can quickly deploy new protocols on forwarding devices, customize their functionality, and develop innovative services. This flexibility comes however with a cost: network-wide security and correctness properties (e.g., isolation, reachability, etc.) become much harder to ensure, because network behavior is now determined by a combination of the control plane-driven configuration and the data plane program that resides on devices (also called switches). Existing network verification tools, which rely on a fixed, invariant model of the data plane, are inadequate for programmable data planes. We propose to develop new techniques to verify and enforce security properties in data plane networks. Our proposed verification techniques extend existing verification tools by automatically generating a data plane model from a P4 program. We then adapt existing verifications tools to work with our dynamically-generated models to verify network configuration updates issued by an SDN controller. We also propose a new approach to ensuring that network security properties are satisfied by a network configuration that is based on data plane enforcement. We propose to develop an inline monitor, implemented in the data plane itself, that enforces critical security properties, such as isolation and bandwidth limits, even in the presence of a faulty user data plane program or controller. Today?s networks are increasingly called on to perform security services, e.g. intrusion detection, in the network. A programmable data plane network can be used to accelerate and enhance such services by performing some of the required processing in the network data plane. The second thrust of this proposal concerns new data plane security services made possible by programmable data plane networks. We propose to develop new data plane-based security functions and techniques for composing them in a manner that balances performance characteristics of the data plane (very fast but low complexity) and dedicated virtualized network function units attached to the network (slower but more sophisticated) to implement the required security function... 2013 - 2016
Estudo e Implementação de Técnicas de Acesso e Alocação Oportunística das Faixas de Rádio Frequências
Descrição: O aumento na demanda por conectividade sem fio, em consequência da popularização de dispositivos que utilizam esta tecnologia, tem esbarrado em um sério entrave: as faixas não licenciadas, conhecidas como ISM (Industrial, Scientific and Medical), utilizadas para comunicação sem fio, estão cada vez mais congestionadas. Em contraste com este cenário, as bandas licenciadas - que objetivam a atender demandas especificas e são alocadas estaticamente por agências reguladoras, em nosso pais, pela ANATEL - encontram-se, em determinados períodos de tempo e regiões, ociosas, na ordem de 5% de uso, por exemplo, para as faixas de 3Ghz e 5GHz, conforme estudo realizado pela agência americana Federal Communications Commission (FCC). É neste cenário em que surgiu a discussão de técnicas que permitam um melhor aproveitamento do espectro eletromagnético, onde novos usuários (usuários secundários), ou usuários provenientes de faixas congestionadas, pudessem explorar eventuais oportunidades encontradas nas faixas do espectro que encontram-se momentaneamente ociosas. Este tipo de acesso é conhecido como Acesso Oportunístico do Espectro. Neste contexto, os usuários que já possuem o direito exclusivo de acesso ao canal proprietário (usuários primários) são considerados prioritários e eventuais comunicações realizadas por novos usuários devem, obrigatoriamente, respeitar e não interferir em suas comunicações. Basicamente, o comportamento tradicional de acesso dos usuários primários não são afetados pelos usuários secundários que desejam acesso ao canal sem fio. Esta é uma nova área de pesquisa e vem ganhando cada vez mais força por apresentar um melhor aproveitamento do espectro e, com isso, minimizar o problema de escassez do espectro eletromagnético. Usuários primários continuam a utilizar suas faixas de freqüência da forma habitual e passam a "sublocar" a porção não utilizada para que os usuários secundários passem a ter acesso ao espectro mediante ao pagamento pelo seu uso. Para que o acesso oportunista seja possível, o nó secundário utiliza mecanismos de rádio definido por software (do Inglês Software Defined Radio - SDR). Dependendo do nível de sofisticação do dispositivo SDR, vários parâmetros pode ser ajustados, tais como a frequência, esquema de modulação, potência de transmissão, entre outros. Quando o dispositivo de SDR é capaz de se adaptar à frequência de rádio de acordo com objetivos pré-definidos, tais como a disponibilidade e desempenho, este dispositivo é denominado rádio cognitivo (CR). Garantir o compartilhamento do espectro de forma a preservar as comunicações do usuário primário não é uma tarefa trivial. Inicialmente, para que este acesso oportunista seja possível, o usuário secundário deve possuir mecanismos eficientes para identificar e utilizar as oportunidades de transmissão. Em um cenário distribuído, estes mecanismos implicam na capacidade do usuário secundário em monitorar um conjunto de faixas licenciadas e encontrar um subconjunto de faixas livres para comunicação. Uma vez identificado este subconjunto, que deve ser comum entre o transmissor e receptor, é que a comunicação pode ser realizada. Realizar estas atividades em tempo hábil a atender as restrições de atraso, variação de atraso e vazão, além de manter o consumo de energia em limites aceitáveis, tem se mostrado um grande desafio. De forma semelhante, protocolos de roteamento e acesso ao meio que atendam as necessidades e características dos usuários em um ambiente de acesso dinâmico precisam ser desenvolvidos ou mesmo adaptados para este novo contexto.. 2010 - 2012
Desenvolvimento de técnias e metodologias aplicadas a caracterização, parametrização e otimização de redes sem fio e redes híbridas
Descrição: A última década foi marcada pelo uso crescente de tecnologias de redes sem fio para o acesso a última milha, à Internet de banda larga. Contudo, devido a sua natureza distribuída, as redes sem fio apresentam muitos desafios quando se deparam a crescente demanda por aplicações multimídia, as quais requerem um controle mais rigoroso dos recursos disponíveis. O aumento no número de usuários de redes sem fio, associado a uma demanda crescente por maior velocidade e restrições de temporização por parte dos aplicativos, requerem mecanismos que permitam analisar e melhorar o desempenho das camadas envolvidas no processo de transmissão.
Um dos tópicos que vêm sendo pesquisados em relação à área de modelagem de tráfego de redes é a vazão de um canal de comunicação. Por meio da medida da capacidade efetiva que um canal de comunicação, é possível  obter resultados que orientem administradores de redes na modelagem dos perfis de seus usuários.  Um melhor conhecimento do limite de banda disponível por usuário, permitirá um melhor dimensionamento dos recursos e um melhor aprovisionamento de Qualidade de Serviço (QoS) para as aplicações. Este projeto de pesquisa tem como objetivo contribuir no processo de caracterização e metodologias para dimensionamento de tráfego. Espera-se avançar em questões relacionados ao cálculo de métricas de QoS, policiamento de tráfego, agregação de tráfego, e computação colaborativa que podem ser explorados através do entendimento das características intrínsecas ao perfil das aplicações..
