Plataforma Multibiométrica e Monitoramento Inteligente
Descrição: Este projeto tem o objetivo de desenvolver ferramentas e incorporar novas funcionalidades às soluções utilizadas pelas forças de segurança pública do Estado do Ceará, tais como: (i) motor de buscas para reconhecimento facial; (ii) analíticos para vídeo monitoramento, visando a identificação de objetos e comportamentos de interesse; (iii) novo modelo de registro civil com uso de tecnologias e inteligências que garantam uma coleta rápida e de qualidade; e (iv) plataforma de identidade digital.. 2017 - 2020
P4Sec - Securing Networks in the Programmable Data Plane Era
Descrição: Recent advances in Software Defined Networking (SDN) have expanded our ability to program the network to its data plane. Through domain specific languages like P4, network operators can quickly deploy new protocols on forwarding devices, customize their functionality, and develop innovative services. This flexibility comes however with a cost: network-wide security and correctness properties (e.g., isolation, reachability, etc.) become much harder to ensure, because network behavior is now determined by a combination of the control plane-driven configuration and the data plane program that resides on devices (also called switches). Existing network verification tools, which rely on a fixed, invariant model of the data plane, are inadequate for programmable data planes. We propose to develop new techniques to verify and enforce security properties in data plane networks. Our proposed verification techniques extend existing verification tools by automatically generating a data plane model from a P4 program.We then adapt existing verifications tools to work with our dynamicallygenerated models to verify network configuration updates issued by an SDN controller. We also propose a new approach to ensuring that network security properties are satisfied by a network configuration that is based on data plane enforcement. We propose to develop an in-line monitor, implemented in the data plane itself, that enforces critical security properties, such as isolation and bandwidth limits, even in the presence of a faulty user data plane program or controller. Today?s networks are increasingly called on to perform security services, e.g. intrusion detection, in the network. A programmable data plane network can be used to accelerate and enhance such services by performing some of the required processing in the network data plane. The second thrust of this proposal concerns new data plane security services made possible by programmable data plane networks. We propose to develop new data plane-based security functions and techniques for composing them in a manner that balances performance characteristics of the data plane (very fast but less complex) and dedicated virtualized network function units attached to the network (slower but more complex) to implement the required security function. Programmable data planes are poised to transform the architecture of modern networks. The proposed work will advance the current state-of-the-art by extending verification and enforcement techniques to programmable data plane networks, for which neither network verification nor security policy mechanisms currently exist. Our work will also develop new data plane services that will enable a new class of security functions to be deployed in the network in order to improve the overall security of computer networks. Our project follows two branches of investigation: First, we propose to determine which network security properties can be verified in networks with programmable data planes and which properties can be dynamically enforced by a programmable data plane.
