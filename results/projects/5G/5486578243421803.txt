Sistema de Teleoperação para Monitoramento de Correias Transportadoras
Projeto certificado pela empresa ASSOCIACAO INSTITUTO TECNOLOGICO VALE - ITV em 12/08/2016.
Descrição: A REDE "Teleoperação Avançada" tem por principal objetivo investigar o problema da operação remota de equipamentos de mineração visando ao aumento da segurança e da produtividade. Assim como os desafios, as tarefas e aplicações específicas das técnicas de teleoperação são bastante diversas. A REDE se propõe a abordar duas questões distintas: A teleoperação de escavadeiras e o monitoramento de correias transportadoras.
Tais frentes foram priorizadas levando-se em conta a dificuldade técnica de se desenvolver uma escavadeira verdadeiramente autônoma, dada a complexidade inerente da tarefa, e a criticidade das correias transportadoras para as operações da VALE.
No caso das escavadeiras, a teleoperação parece ser a maneira mais promissora tecnológica e economicamente de reduzir a exposição humana aos riscos associados à operação. O desafio associado às correias transportadoras é diferente e está associado à manutenção desses equipamentos vitais para o transporte do material produzido. Como a estrutura de suporte das correias frequentemente não está no nível do solo, a tarefa de inspeção, seja no porto, seja na mina, muitas vezes implica em trabalho em altura, o que além de aumentar o custo de cada inspeção, aumenta o risco da atividade. A possibilidade de se inspecionar tais estruturas remotamente e de forma automatizada reduz os custos operacionais, viabiliza inspeções mais frequentes, o que por sua vez a prevenir falhas potencialmente catastróficas e perdas operacionais.. KITT ? Kit de Infotenimento e Transporte Tecnológico.
Projeto certificado pela empresa Seva Engenharia Eletrônica em 28/02/2013.
Descrição: O objetivo deste projeto é criar um framework para integrar hardware e software em um ambiente de infotenimento veicular. Este ambiente é exigido pelos compradores de carros atualmente como um diferencial para o conforto no veículo, além de trazer funcionalidades de interesse privado, como diagnostico remoto do carro. Para descrever este novo tipo de aplicação é usado o termo ?infotenimento para veículos? ? in-vehicle infotainmen ? ou ?IVI?, que é associado a um sistema integrado de informações e entretenimentos veicular, exigido pelos compradores de carros atualmente. Os recursos e aplicações geralmente incluem:
Entretenimento para o assento traseiro (filmes, jogos, TV, rede social etc)
Navegação com GPS
Rádio/CD
Serviços baseados em localização
Conectividade   interna para dispositivos móveis
Comunicações externas via Internet
O IVI consiste de um computador de bordo, com uma interface confortavel ao motorista e aos passageiros. Propomos neste projeto o uso do sistema operacional Android. O Android oferece novas possibilidades para o desenvolvimento de aplicações, permitindo o uso de um ambiente de desenvolvimento gratuito e aberto a modificações, além de usar o Linux, com código fonte livre. Este fator é essencial para garantir a segurança e confiabilidade do sistema usado no IVI. Além disso, ele possui um conjunto de API que permitem o rápido acesso ao hardware do IVI e coleta de informações da rede de sensores do carro, sem afetar a segurança ou invasões do sistema veicular.  Atualmente os carros começaram a ser fabricados com o suporte a uma rede de sensores interna, chamada de CAN ? Controller Area Network ? a qual permite um relatório completo da situação do carro. O IVI atua como um integrador que comunica com a CAN ao mesmo tempo que permite acesso a internet e entrenimento do carro. O IVI servira como portal de acesso a estes dados, sendo que sua interface deve ser preparada de forma a não prejudicar a direção do carro. Ele também deve permitir. (CIA) - Construindo Cidades Inteligentes: da Instrumentação dos Ambientes ao desenvolvimento de Aplicações
Descrição: Cidades Inteligentes é um conceito relacionado à oferta de serviços de informação e comunicação para solucionar os problemas dos cidadãos em seus conglomerados urbanos, e de garantir uma melhor gestão pública. Uma cidade inteligente necessita de soluções para vários problemas, como: trânsito, vigilância, atendimento de emergência, monitoramento ambiental, saúde, educação e inclusão digital, bem como permite uma maior eficiência e transparência na gestão pública. Além disso, tanto a comunicação como as aplicações precisam ser eficientes em energia em situação de sobrecarga e emergência. Neste contexto, o presente projeto se propõe a construir uma infraestrutura de instrumentação, computação e comunicação para viabilização de Cidades Inteligentes, abrangendo desde a aquisição dos dados urbanos brutos por meio de tecnologias de redes de sensores e internet das coisas, a comunicação, o armazenamento, e o acesso a esses dados através de diferentes tecnologias e protocolos de redes sem fio, até a construção de aplicações que se beneficiem de toda essa infraestrutura, suportando uma melhor gestão pública e do meio ambiente e agregando valor ao cidadão.. Um arcabouço para monitoramento de minas utilizando redes de sensores sem fio Edital Fapemig 01/2009 - Universal - Processo APQ-00577-09
Descrição: Este projeto consiste no desenvolvimento de um arcabouço para aquisição de dados em minas utilizando redes de sensores sem fio. Esse arcabouço prevê um conjunto de soluções para o problema de aquisição de dados, inferência, manutenção e gerenciamento das minas utilizando as redes de sensores. Basicamente, o arcabouço define quatro etapas para o desenvolvimento de uma solução de monitoramento: (i) definição de um modelo conceitual para a caracterização das aplicações de minas; (ii) definição de benchmarks para o projeto da rede em ambientes de mineração, visando a caracterização e adequação das tecnologias de comunicação, processamento e armazenamento; (iii) a disponibilização de uma API com as soluções de infraestrutura da rede e processamento dos dados, os algoritmos disponíveis na API serão validados através de mecanismos estatísticos e/ou modelos de simulação; e (iv) um processo para implantação das soluções em ambientes reais. O desenvolvimento desse arcabouço possui diversos desafios, dentre eles podemos destacar: (i) o desenvolvimento de uma solução inovadora para a área de mineração; (ii) o desenvolvimento de um sistema de aquisição de dados e supervisão de uma mina através da utilização da tecnologia de redes de sensores; e (iii) o desenvolvimento de software embarcado através da utilização de sensores nos diferentes ambientes das minas..
