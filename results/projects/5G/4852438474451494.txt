2010 - 2012
Sistema Gerenciador de Ambientes Multimídia
Descrição: Esse projeto tem como propósito o desenvolvimento de uma solução para utilização de mídias digitais que recebe a denominação no meio acadêmico e no mercado de Media Center. O sistema proposto será responsável pelo armazenamento, gerenciamento, exibição de formatos de mídia que compreendem vídeo e áudio nos mais diversos formatos de codificação, streaming de vídeo e áudio de fontes como Internet, e transmissões analógicas e digitais nos diversos meio como cabo e radiofreqüência. Uma interface gráfica que atenda padrões de usabilidade para o consumidor final será desenvolvida possibilitando que todas as funcionalidades do sistema possam ser acessadas por dispositivos de controle como teclado e controle remoto sem fio.. Situação: Em andamento; Natureza: Pesquisa. Alunos envolvidos: Graduação: (4) . Integrantes: Andre Luis Meneses Silva - Integrante / Marco Túlio Chella - Coordenador.Financiador(es): SMS - Auxílio

2009 - 2010
Desenvolvimento de Soluções Eficientes e de Alto Desempenho no Contexto de Sistemas Operativos Web
Descrição: Esse projeto tem como objetivo o desenvolvimento de uma solução Web OS. Essa solução envolve o desenvolvimento do módulo servidor, cliente, bem como aplicações básicas de suporte a serem utilizadas no sistema. Esse trabalho também possui como objetivo o desenvolvimento de soluções simples, baratas e que demande um menor poder computacional comparado às versões existentes. Para tal, pretende-se desenvolver uma solução customizada, que utilizará softwares gratuitos, bem como utilizar plataformas alvo com poder computacional limitado. Essa plataforma alvo tem como intuito validar a solução proposta, bem como servir como base para o desenvolvimento de dispositivos clientes e servidores de menor custo.. Situação: Concluído; Natureza: Pesquisa. Alunos envolvidos: Graduação: (3) . Integrantes: Andre Luis Meneses Silva - Integrante / Ribamar, Admilson de - Coordenador.
