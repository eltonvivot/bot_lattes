2019 - 2021
SWAMP: Smart Water Management Platform
Projeto certificado pelo(a) coordenador(a) Carlos Alberto Kamienski em 04/05/2019.
Descrição: The SWAMP project develops IoT based methods and approaches for smart water management in precision irrigation domain, and pilots them in Italy, Spain, and Brazil (2). Water is vital for ensuring food security to the world?s population, and agriculture is the biggest consumer amounting for 70% of freshwater. The water wastages are caused mainly by leakages in distribution and irrigation systems, and in the field application methods. The most common technique, surface irrigation wastes a high percentage of the water by wetting areas where no plants benefit from it. Localized irrigation can use water more efficiently and effectively, avoiding both under-irrigation and over-irrigation. However, in an attempt to avoid under-irrigation, farmers feed more water than is needed resulting not only to productivity losses, but also water is wasted. Therefore, technology should be developed and deployed for sensing the level of water needed by the plantation and for flowing the water to places where and when needed. The SWAMP project addresses these issues by use of the Internet of Things (IoT), data analytics, autonomous devices and other related technologies. The challenges addressed by SWAMP project are following: 1) Reducing effort in software development for IoT-based smart applications. 2) Automating advanced platforms and integrating different technologies and components. 3) The integration of heterogeneous and advanced sensors, particularly flying sensors (drones) providing precision in the water supply for irrigation. 4) The use of a Software Platform together with technologies such as IoT, Big Data, Cloud/Fog and drones for the deployment of pilot applications for smart water management. 5) Proposing, testing and validating new business models for using IoT in smart water management settings. 6) Technological components must be flexible and adaptable enough in order to adapt to different contexts and to be replicable to different locations and contexts..

P4Sec - Securing Networks in the Programmable Data Plane Era
Descrição: Recent advances in Software Defined Networking (SDN) have expanded our ability to program the network to its data plane. Through domain specific languages like P4 [3], network operators can quickly deploy new protocols on forwarding devices, customize their functionality, and develop innovative services [13, 5, 17]. This flexibility comes however with a cost: network-wide security and correctness properties (e.g., isolation, reachability, etc.) become much harder to ensure, because network behavior is now determined by a combination of the control plane-driven configuration and the data plane program that resides on devices (also called switches). Existing network verification tools [7, 14, 19, 15, 35], which rely on a fixed, invariant model of the data plane, are inadequate for programmable data planes. We propose to develop new techniques to verify and enforce security properties in data plane networks. Our proposed verification techniques extend existing verification tools by automatically generating a data plane model from a P4 program.We then adapt existing verifications tools to work with our dynamicallygenerated models to verify network configuration updates issued by an SDN controller. We also propose a new approach to ensuring that network security properties are satisfied by a network configuration that is based on data plane enforcement. We propose to develop an in-line monitor, implemented in the data plane itself, that enforces critical security properties, such as isolation and bandwidth limits, even in the presence of a faulty user data plane program or controller. Today?s networks are increasingly called on to perform security services, e.g. intrusion detection, in the network. A programmable data plane network can be used to accelerate and enhance such services by performing some of the required processing in the network data plane. The second thrust of this proposal concerns new data plane security services made possible by programmable data plane networks. We propose to develop new data plane-based security functions and techniques for composing them in a manner that balances performance characteristics of the data plane (very fast but less complex) and dedicated virtualized network function units attached to the network (slower but more complex) to implement the required security function. Programmable data planes are poised to transform the architecture of modern networks. The proposed work will advance the current state-of-the-art by extending verification and enforcement techniques to programmable data plane networks, for which neither network verification nor security policy mechanisms currently exist. Our work will also develop new data plane services that will enable a new class of security functions to be deployed in the network in order to improve the overall security of computer networks. Our project follows two branches of investigation: First, we propose to determine which network security properties can be verified in networks with programmable data planes and which properties can be dynamically enforced by a programmable data plane. Second, we investigate new security services made possible by programmable data planes... Situação: Concluído; Natureza: Pesquisa.
